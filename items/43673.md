## Summary

Keep #43673 open. Current main has adjacent multi-agent routing, per-agent workspaces, per-agent tool policy, and group/sender tool restrictions, but it does not implement the requested first-class org/team deployment package: team workspace scaffolding CLI, hard per-session/operator-scope RBAC, or a declarative deployment manifest. The related #49183/#42981 items and open PRs #53496/#53530 are useful smaller RBAC slices, not a complete duplicate or shipped replacement for this umbrella feature.

## Best Possible Solution

Keep #43673 open as the parent org/team deployment tracker or split it into explicit maintainer-owned subissues: workspace/team scaffold CLI, declarative deployment manifest/schema/docs, and hard per-session/operator-scope tool policy. Treat #49183/#42981/#53496/#53530 as related RBAC implementation slices rather than superseding this broader request.
