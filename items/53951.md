## Summary

Current `main` does not implement Defuddle. OpenClaw still ships Readability as the active HTML extractor, and the extractor path was refactored on 2026-04-25 into a bundled plugin seam, so this PR is still a real feature proposal but now needs a rewrite/rebase against the new architecture rather than conservative cleanup closure.
