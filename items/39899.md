## Summary

Keep open. Current `main` does not contain the surrogate-specific recovery described in PR #39899. The auto-reply path still only auto-resets for compaction, role-ordering, and Gemini session-corruption cases, and the shared assistant error formatter has no surrogate-encoding branch. There is partial mitigation in transport payload sanitization, but that is not the same fix and does not prove this PR is already implemented on `main`.
