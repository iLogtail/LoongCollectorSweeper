The related #69584 was closed unmerged, so it is not a canonical shipped replacement.

Keep the PR open, rebase it onto current main, and narrow or split as needed around the still-missing work: `channels.status includeAccounts` plus default-account/configuration-check fixes, Control UI channel refresh queuing, OAuth browser-open fallback, quickstart plugin config prompts, scheduled-task script parsing, and TUI auth argv cleanup.
