## Summary

Keep open. Current `main` still has the resolver mismatch the PR describes, so this is not implemented on `main` and not safely closable. The reported bug remains plausible for `openclaw node install`, and the PR also bundles an unrelated `nodeCwd` feature that should likely be split before merge.
