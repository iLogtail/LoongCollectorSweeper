## Summary

Keep open. Current `main` does not contain this PR’s dropped-media pipeline or an equivalent fix: it still emits only a generic `⚠️ Media failed.` warning when all media are lost, silently hides partial drops, and still prunes/drop-routes payloads that contain only attachment-failure diagnostics.
