## Summary

Keep open. Current `main` still lacks multiple behaviors this PR is trying to add, and the PR discussion surfaced at least one auth-contract problem that needs maintainer review before merge.

## Risks / Open Questions

- Merging the PR's Google Vertex ADC hunk as written would conflict with the current runtime contract that rejects non-`aws-sdk` auth without an `apiKey`.
- The PR's `denyCommands` warning logic, as described in the provided patch/review, appears to validate against `resolveNodeCommandAllowlist()`, which is not the full node-command catalog and would mislabel valid dangerous commands such as `camera.snap` and `screen.record`.
- The HEIC normalization hunk in the PR also drew a valid review note: converted JPEG payloads need a post-conversion size check, matching the existing guard already used in `src/media/input-files.ts`.
