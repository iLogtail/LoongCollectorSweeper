## Summary

Keep open. Current main still matches the reported failure path: the gateway set-param schema rejects extra allowlist properties, while runtime allowlist entries can contain `source` and `commandText`, and the Control UI round-trips the snapshot back through `exec.approvals.set`. The related fix PRs in the supplied context are not merged, and current main does not contain the schema fix.
