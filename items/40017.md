## Summary

Keep open. Current main still treats the agent main session as the canonical `agent:<id>:main`/`global` session, not an arbitrary existing channel session. There is a heartbeat-only session override, but it does not implement a general configurable primary channel session for heartbeats, proactive/system events, or other main-session behavior.
