## Summary

Keep open. Current `main` only covers part of this PR: `docs/cli/acp.md` already documents `--provenance`, and `docs/automation/gmail-pubsub.md` has since been replaced by a redirect to `docs/automation/cron-jobs.md`, but the other reported doc gaps are still real on `main`. In particular, `OPENCLAW_CLI=1` exists in code and is not documented in `docs/help/environment.md`, and the macOS onboarding UI still exposes the remote gateway token field and unsupported-token warning without matching text in `docs/start/onboarding.md`. This is not strong evidence for closure; it is a stale-but-salvageable docs PR that needs rebasing/scope cleanup, not closure.
