## Summary

Keep open. Current `main` has a filterable cron jobs list, per-job actions, run history, and a few summary stats, but it does not implement the requested calendar/timeline/week views or drag-to-reschedule UX. The report is also recent enough that `stale_insufficient_info` does not apply, and the issue already has an open linked PR (#41892), so conservative cleanup should not auto-close it.
