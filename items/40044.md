## Summary

Keep #40044 open. Current main at 7e13f3f5140168e31027b39a9855ade3fadf24d5 still does not expose the current session UUID in the system-prompt Runtime line, and the PR remains a narrow additive agents prompt change rather than obsolete, duplicate, or ClawHub-scoped work.
