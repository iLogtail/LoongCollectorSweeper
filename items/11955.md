## Summary

Keep #11955 open. Current main implements adjacent memory search, optional transcript indexing, usage/cost aggregation, startupContext, and BOOT.md startup hooks, but the umbrella still has meaningful remaining core work: an agent-facing self-evaluation metrics API, automatic conversation chaining or summary handoff, and default gateway-restart preload of core context before first user input. The provided issue discussion also includes a recent steipete Codex review explicitly keeping #11955 open as the canonical umbrella while #11956 is closed as the duplicate/copy.

## Best Possible Solution

Keep #11955 open as the canonical umbrella unless a maintainer splits it. Preserve the shipped memory_search, extraPaths, opt-in sessionMemory, usage/cost/session aggregation, startupContext, and BOOT.md behavior, then create narrower product/API work for an agent-facing self-evaluation metrics API, documented conversation continuation or compact-summary handoff, and explicit gateway startup preload/prewarm of core workspace context before first user input.
