## Summary

Keep open. Current `main` still resolves Telegram `requireMention` with `activationOverride` before `topicConfig?.requireMention`, so the PR is not redundant and the reported bug remains plausible against shipped code. Docs also state topic settings override group settings, which matches the PR’s intent, and `main` does not include the proposed regression test file.
