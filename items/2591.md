## Summary

本 PR 为构建、打包与 Docker 镜像脚本添加了 riscv64 架构的显式检测与库路径映射。当前 main 分支在非 x86_64 环境下默认回退至 arm64 或 sw64，尚未原生支持 riscv64。修改范围清晰且验证充分，属于合理的基线可移植性改进，应保持开启以供维护者审查与 CI 验证。

## Best Possible Solution

保持开启。建议维护者审查架构检测逻辑是否符合项目规范，确认 CI 流水线在 riscv64 模拟或真实环境下通过。若后续需引入 RISC-V 特定指令集优化（如 RVV）或扩展 CentOS 生产镜像路径，可作为独立特性另行跟进。合并本 PR 即可解决基线构建支持问题。
