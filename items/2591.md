## Summary

本 PR 为构建脚本、发布脚本及 Dockerfile 添加了 riscv64 架构的显式支持。变更范围清晰保守，仅涉及架构检测与打包路径的基线可移植性改进，未改动核心数据通路。当前 main 分支尚未包含此支持，属于有价值的可移植性增强，建议保持开启等待维护者审查与 CI 验证。

## Best Possible Solution

保持开启。建议维护者审查架构检测逻辑（Makefile 与 scripts/*.sh）及 Dockerfile 映射是否符合项目发版规范。确认现有 CI 流水线在 amd64/arm64 上未受破坏后合并。若后续需将 riscv64 加入正式 GitHub Actions 发版矩阵或扩展 CentOS 生产镜像路径，可作为独立特性另行跟进。
