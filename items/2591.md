## Summary

本 PR 为构建脚本、Makefile 和 Dockerfile 添加了保守且明确的 riscv64 架构检测与库路径映射。变更范围清晰，仅涉及打包与构建路径的基线可移植性改进，未改动核心数据处理逻辑，且已签署 CLA。当前 main 分支尚未包含此支持，PR 状态健康，建议保持开启等待维护者审查与 CI 验证。

## Best Possible Solution

保持开启。建议维护者审查架构检测逻辑与 Dockerfile 映射是否符合项目规范，确认 CI 流水线在现有架构上未受破坏后合并。合并后即可解决基线构建与发布脚本的 riscv64 支持问题。若后续需将 riscv64 加入正式发版矩阵或扩展 CentOS 生产镜像路径，可作为独立特性另行跟进。
