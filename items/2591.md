## Summary

本 PR 为构建脚本、发布脚本及 Dockerfile 添加了 riscv64 架构的显式检测与库路径映射，属于合理的基线可移植性改进。当前 main 分支尚未包含此支持，修改范围清晰且验证充分，需维护者审查与 CI 验证后合并。

## Best Possible Solution

保持开启以供维护者审查。建议确认 CI 流水线通过且架构检测逻辑符合预期后合并。若后续需引入 RISC-V 特定指令集优化或扩展 CentOS 生产镜像路径，可作为独立特性另行跟进。
