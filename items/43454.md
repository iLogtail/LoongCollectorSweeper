## Summary

Keep open. Current main still does not expose `subagent:ended`, turn-complete, or tool-call-threshold events through the workspace/internal hook system. The adjacent Plugin SDK hooks are real and useful, and open PR #18889 partially overlaps agent/tool lifecycle boundaries, but neither fully supersedes this issue’s request for workspace hook events and subagent completion plumbing.

## Best Possible Solution

Keep the issue open for maintainer product/API judgment. A good path is either to implement a narrow internal-hook bridge for `subagent:ended` plus well-defined turn/tool-count lifecycle events, or explicitly close later with maintainer-approved guidance that these workflows should use Plugin SDK hooks such as `subagent_ended`, `before_tool_call`/`after_tool_call`, and `agent_end`.
