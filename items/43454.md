## Summary

Keep #43454 open. Current main still does not expose `subagent:ended`, `agent:turnComplete`, or `agent:toolCallThreshold` through the workspace/internal `HOOK.md` surface. Plugin hooks and the Skill Workshop plugin cover adjacent use cases, but they do not satisfy the requested workspace hook lifecycle events, and related #49019 is a different open API request rather than a canonical duplicate.

## Evidence
