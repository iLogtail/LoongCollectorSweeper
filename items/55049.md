## Summary

Keep open. Current main does distinguish real `system` transcript roles in the UI, but there is still a current-main restart-continuation path that represents an internal continuation as an `agentTurn` and dispatches it through the normal inbound-user reply pipeline. That matches the report closely enough that there is not strong evidence to close this as fixed or obsolete.
