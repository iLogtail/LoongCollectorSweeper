## Summary

Current `main` still appears to have the reported gap. Hybrid memory search passes the raw cleaned query into FTS, and `buildFtsQuery` still AND-joins every token, while keyword extraction is only used in the FTS-only/no-provider branch. The proposed guardrail test file from this PR is also absent on `main`, so the report remains actionable rather than obsolete.
