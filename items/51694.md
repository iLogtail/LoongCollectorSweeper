## Summary

Keep open. Current `main` still marks Google Chat `appPrincipal` optional in the config schema, generated config metadata, and TypeScript type, while the runtime still uses it for app-url add-on token verification. The linked fix PR #51817 appears closed and unmerged in the provided GitHub context, so it does not supersede the issue.
