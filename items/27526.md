## Summary

Keep #27526 open. Current main has partial session listing/search/labels/overrides, chat session switching, message-level pinning, checkpoint branching, and subagent/session orchestration, but it still does not implement the requested WebChat conversation directory with folders/tags/archive/session pinning/recommendations or the requested multi-AI collaboration/committee UX. The discussion also has explicit maintainer signal keeping this as the canonical tracker after duplicate #27603 was consolidated here, while related PR #52307 was closed unmerged and does not supersede the broader request.

## Best Possible Solution

Continue using #27526 as the canonical maintainer/product tracker. The best path is incremental: first define session metadata and protocol support for folders, tags, archive state, session pinning, richer search/recommendations, and directory navigation in WebChat/Control UI; then add an explicit multi-model/role collaboration UX and protocol for comparison, consensus, agenda/minutes, and project-task extraction. Narrow follow-up PRs such as REST session listing can remain separate but should not close this umbrella until the central user-facing workflow exists.
