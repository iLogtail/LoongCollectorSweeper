## Summary

Keep #16711 open. Current main still lacks the requested Control UI chat composer Expand button and large prompt modal; the composer remains an inline textarea with Enter-to-send behavior and a 150px max height. The existing related items are adjacent composer/modal work, not a canonical tracker for this long-prompt editor request, and the latest maintainer-side Codex review comment also explicitly kept this issue open.

## Best Possible Solution

Keep this open for first-class Control UI work. The likely implementation belongs around `ui/src/ui/views/chat.ts`, `ui/src/ui/chat/run-controls.ts`, and `ui/src/styles/chat/layout.css`: add an Expand control, modal draft state with Close/Cancel, Apply, and Send semantics, and modal keyboard behavior where Enter inserts a newline and Ctrl/Cmd+Enter sends. Coordinate with adjacent session/composer issues, but do not close this unless a canonical issue specifically tracks the expanded long-prompt composer modal.
