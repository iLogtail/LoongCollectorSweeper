## Summary

Current `main` does not implement `sandbox.docker.workspaceMountPropagation`, so this PR should stay open. The config type/schema/resolution path lack the field, the Docker workspace mount builder still emits only the existing `ro,z`/`z` forms, and current tests do not cover propagation modes.
