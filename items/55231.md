## Summary

Keep open. Current `main` still replays pending delivery-queue entries on gateway startup, with no age-based stale-entry discard, and transcript echo messages still flow through the queued outbound path. The reported restart batch-flush behavior remains plausible from the current code and is reinforced by existing tests.
