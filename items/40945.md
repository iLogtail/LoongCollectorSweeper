## Summary

Keep open. Current `main` still renders markdown images inline only for base64 `data:image/...` URLs and explicitly downgrades remote `https://...` images to alt text in Control UI chat. The shipped tests and latest release tag match that behavior, so this is neither implemented nor obsolete. The issue is also recent and already has a linked open PR (#41025), which points to active work rather than something to auto-close.
