## Summary

Keep open: current `main` still suppresses text tool-result output in group chats, with only media/exec-approval exceptions, and there is no config surface implementing the requested `allowVerboseToolResults`-style override.

## Evidence

- **Group tool-result text is still intentionally suppressed:** `src/auto-reply/reply/dispatch-from-config.ts:886` says group/native flows intentionally suppress tool summary text; `resolveToolDeliveryPayload` drops text-only payloads unless the tool result has media or an exec-approval object.

A separate test at `src/auto-reply/reply/dispatch-from-config.test.ts:1230` shows only Telegram forum-topic sessions are exempted.
