## Summary

Keep #72108 open. Current main has not implemented the proposed `tsconfig.plugin-sdk.dts.json` exclusion, and the PR addresses a real plugin SDK declaration-build boundary: a Vitest-only helper under `src/plugin-sdk` is still part of the broad declaration include set.

## Best Possible Solution

Keep the PR open and land #72108, or an equivalent maintainer patch, after CI/maintainer validation. The best code path is the narrow config change adding `src/**/*.test-helpers.ts` to `tsconfig.plugin-sdk.dts.json` while preserving intentionally public SDK test utilities such as `src/plugin-sdk/test-helpers.ts` and `src/plugin-sdk/browser-facade-test-helpers.ts`.
