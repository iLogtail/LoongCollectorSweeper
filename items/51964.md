## Summary

Keep open. Current `main` still returns raw sniffed `image/apng` values from `file-type`, and the gateway attachment path still prefers that sniffed MIME over the provided MIME. The reported bug remains plausible on current code, so this PR is not obsolete and should not be auto-closed.
