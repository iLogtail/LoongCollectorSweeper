## Summary

Current `main` still has the reported leak path in `ensureMediaHosted()`. Cleanup only happens in the `needsServerStart && !opts.startServer` branch; failures from `getTailnetHostname()`, unexpected `ensurePortAvailable()` errors, or `startMediaServer()` after `saveMediaSource()` would still leave the saved file behind. The current tests cover only the explicit no-start cleanup branch plus success paths, not those later failure cases.
