Keep open. #71710 is a fresh, concrete Linux cron/SIGTERM report, and current main does not clearly implement either requested resolution: a bounded SIGTERM shutdown path for `openclaw agent`, or documentation requiring `timeout -k`/equivalent supervisor escalation.
