## Summary

Keep open. Current `main` still lacks the MIME-preserving Discord upload change and still routes WebP images through JPEG optimization, so this PR is not implemented on `main` and not obsolete. There is also an unresolved review concern about bypassing optimization based on a `.webp` filename alone.
