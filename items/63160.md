## Summary

Keep open. Current `main` still routes WhatsApp `sendMedia` through `resolveOutboundSendDep`, so the PR is not already implemented. The attached discussion also shows both positive bug corroboration and a plausible regression risk around injected `deps.sendWhatsApp`, which means this still needs maintainer judgment rather than conservative cleanup closure.
