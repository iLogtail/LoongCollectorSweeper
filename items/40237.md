## Summary

Keep open. Current `main` still sends `cron` tool actions through the generic gateway client path, which opens a fresh Gateway WebSocket per call; there is no landed in-process dispatch for `cron`, and the existing tests still encode the gateway-hop behavior. The provided timeline also points to a still-open fix PR rather than a merged resolution.
