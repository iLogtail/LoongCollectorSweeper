## Summary

Keep open. Current `main` still suppresses commentary-only assistant text, so the TUI streaming gap described by the PR is plausible, but this patch changes a shared visibility helper that is also the contract for gateway previews/history and UI message extraction, where hiding commentary-only text is currently explicit and tested.
