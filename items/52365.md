## Summary

Current `main` does not include this cron-budget fallback fix. The outer cron timeout still wraps the whole detached run without propagating a deadline into isolated-agent fallback logic, `runWithModelFallback` still has no pre-attempt stop hook, and the PR discussion includes substantive follow-up concerns rather than evidence that the item is obsolete.
