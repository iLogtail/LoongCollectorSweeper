## Summary

Keep open. Current `main` still has a live subagent orphan-recovery path that injects the exact continuation text "Please continue where you left off.", retries recovery up to 3 times, and does not check whether the interrupted task or latest user message was an explicit stop/no-action instruction before resuming.
