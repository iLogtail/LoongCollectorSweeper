## Summary

Keep open. Current main still lacks the TUI Ctrl+C active-run hint requested by #38501, so this PR is not fully obsolete. The PR should not be merged as-is because its other bundled changes are already handled on main or conflict with current web_search policy behavior.

## Best Possible Solution

Keep the PR open only as evidence for the remaining #38501 behavior, but do not merge it as-is. The maintainer path should be a focused TUI-only patch in `src/tui/tui.ts` that adds the active-run hint, with a small regression test if practical, while preserving current main's web_search, Feishu, bootstrap-cache, and Telegram implementations.
