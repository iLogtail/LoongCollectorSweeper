## Summary

Keep open. Current `main` has landed a narrower CJK fix for context pruning, but it does not fully implement this PR’s broader request: soft-trim eligibility still uses raw UTF-16 length, and assistant `toolCall` arguments are still estimated with raw `JSON.stringify(...).length`.
