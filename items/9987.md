## Summary

Keep #9987 open. Current main has the backend `exec.approval.list` RPC and tests proving pending records include the command payload, but the requested user-facing `openclaw approvals pending` command, CLI table/JSON output, CLI tests, and CLI docs are still missing. The related implementation PRs #10013 and #10341 were closed unmerged, so this remains the canonical tracker.

## Best Possible Solution

Keep #9987 open until OpenClaw adds `openclaw approvals pending` in `src/cli/exec-approvals-cli.ts`, backed by existing `exec.approval.list`, with human table output showing command details, `--json` output for automation, focused CLI tests, and docs at `docs/cli/approvals.md` / CLI index. Treat `--watch` as either part of the first implementation or an explicit follow-up decision.
