## Summary

Keep open. Current `main` does not implement agent-scoped memory in `memory-lancedb`, the requested behavior is still coherent and in scope for a bundled memory plugin, and the draft PR still carries unresolved correctness concerns from review rather than strong evidence that it is obsolete.

## Evidence
