## Summary

Keep open. Current main does not implement the requested cleanup: there is no current `ensure-whatsapp.sh` / `whatsapp-health.log` path to retire, and the WhatsApp doctor/status code does not scan user crontabs for stale legacy checks. The related discussion also distinguishes this from #58129, and PR #60212 explicitly says it does not address #60204.
