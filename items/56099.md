## Summary

Keep open. Current `main` still hard-cancels compaction when `modelRegistry.getApiKeyAndHeaders` is absent, so the PR’s stated bug is not implemented on `main`; and the proposed bypass is not safe because the downstream summarization path catches generation/auth failures and can still emit a fallback summary instead of preserving history.

## Evidence
