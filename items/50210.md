## Summary

Keep open. Current `main` still uses the global configured default model for `sessions.list.defaults`, so the reported gateway/UI mismatch is not implemented on `main`. The PR also is not safely closable as obsolete: its core bug remains plausible, one review concern about agent-specific `contextTokens` is still valid against the shown patch, and the PR bundles an unrelated hooks change.
