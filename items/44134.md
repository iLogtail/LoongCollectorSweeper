## Summary

Keep open. Current `main` still rebuilds/normalizes tool schemas during each embedded run attempt, and the Google transport still sends tool declarations with each request. I found no cache or session-persistence layer in the affected path, and the latest release `v2026.4.22` still contains the same per-tool normalization code.
