## Summary

Keep open. Current `main` at `bfa6708c03f24c7caa5d0a96a298c292be0d85c1` does not implement raw-JSON change-line display in the config UI. Raw mode still shows only a generic unsaved-state badge and a plain textarea, while the pending-change diff UI is explicitly limited to form mode.
