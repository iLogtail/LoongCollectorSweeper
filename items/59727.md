## Summary

Keep open. Current main has not implemented the requested CI refactor: the main CI workflow is still monolithic, still emits many preflight outputs, still builds matrices inline, and the proposed `.github/matrices` / `_*.yml` / workflow README structure is absent.

## Best Possible Solution

Keep the issue open as a valid CI refactor request. The best path is a maintainer-reviewed, staged workflow cleanup: extract stable matrix definitions or matrix-building helpers, introduce reusable workflows only where GitHub Actions semantics preserve current job behavior, add `.github/workflows/README.md` or update `docs/ci.md`, and validate with workflow-sanity/actionlint plus a test PR that proves required checks and scoped lanes remain unchanged.
