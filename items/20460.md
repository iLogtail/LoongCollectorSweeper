## Summary

Keep open. Current main still stores config backups next to the config file as `openclaw.json.bak`, `openclaw.json.bak.1`, etc.; it does not use a dedicated `config-backup/` directory or datetime-based backup names. The linked PR #20877 matches the request but is closed unmerged, so it does not supersede the issue.
