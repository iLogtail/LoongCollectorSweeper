## Summary

Keep #13597 open. Current main still has no AWS install page or AWS navigation entry, and the existing VPS, Docker, Kubernetes, and Bedrock docs only cover partial inputs rather than the requested EC2/ECS/Fargate/Lambda production deployment guide. The issue discussion remains actionable, including Docker Compose/persistence/secrets/reverse-proxy concerns and an AWS sample repository link.

## Best Possible Solution

Keep the issue open until OpenClaw ships an AWS install page, likely `docs/install/aws.md` linked from `docs/docs.json`, `docs/install/index.md`, and relevant VPS/provider pages. The guide should make Docker Compose the primary EC2/Lightsail path, cover durable state and backups, gateway bind/reverse proxy/ALB/TLS/auth/origin settings, security groups, IAM roles/task roles, Secrets Manager, logging/monitoring, troubleshooting, performance and hardening, ECS/Fargate task/service/networking, Lambda applicability or non-support, multi-agent topology, and evaluate or link the AWS sample repo from the discussion.
