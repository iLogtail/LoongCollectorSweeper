## Summary

Keep open. Current main has the generic SecretRef env/file/exec secrets architecture, but it does not include Bitwarden/Vaultwarden docs, a bundled bw exec resolver, or Bitwarden-specific tests. The PR was rewritten to fit the merged #26155 exec-provider baseline, and the provided discussion includes maintainer guidance to keep it as a focused follow-up rather than obsolete work.

## Best Possible Solution

Continue tracking PR #23096 as a focused follow-up. The best path is to rebase/update the Bitwarden/Vaultwarden docs, `bw` protocol-v1 resolver script, tests, and any needed package inclusion against current main, while preserving the shipped generic `source: "exec"` architecture and avoiding a new vendor-specific provider type unless maintainers explicitly request it.
