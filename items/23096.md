## Summary

Keep PR #23096 open. Current main has the generic SecretRef env/file/exec architecture, but it still does not include the Bitwarden/Vaultwarden docs, bundled bw exec resolver, or Bitwarden-specific tests proposed by this PR. The PR was rewritten after #26155 to use the merged exec-provider baseline, and a MEMBER maintainer explicitly described it as a focused follow-up candidate after rebase.

## Best Possible Solution

Continue tracking PR #23096 as a focused follow-up. The best path is to rebase/update the Bitwarden/Vaultwarden secrets docs, protocol-v1 bw resolver script, tests, and npm package inclusion details against current main while preserving the generic source:"exec" SecretRef architecture and avoiding a new vendor-specific provider type unless maintainers explicitly request one.
