## Summary

Keep open. Current main still renders the Control UI `/agents` selector as a single dropdown, and the default-agent action appears to write an ignored config path rather than the current `agents.list[].default` contract. The related #61887 context tracks the same visibility regression and was also kept open, but #57067 additionally covers the default persistence bug, so closing this item would lose actionable scope.

## Best Possible Solution

Keep #57067 open until the Control UI replaces the single `/agents` dropdown with a visible agent list/sidebar or comparable low-friction navigation, and fixes Set Default to update the supported `agents.list[].default` contract with clear save/persistence behavior and regression coverage. If maintainers prefer a single tracker, #61887 can cover the visibility half, but this issue should not close unless the default persistence bug is also tracked or fixed.
