## Summary

PR #57224 is a duplicate/stale implementation attempt for the same gh-issues path cleanup already tracked by issue #57143 and earlier open PR #57146. Current main still has the underlying `/data/.clawdbot` references, but this PR is not the right canonical item: it carries the same patch shape as #57146, is marked dirty against current main, and has an unresolved review finding that the token fallback became a duplicate no-op.
