## Summary

Keep open. Current main has partial routing safeguards, but it does not implement the requested configurable block or owner reroute for plain-text WhatsApp DM auto-replies from third-party/non-owner inbound messages. The direct reply path still sends deliverable assistant text back to the inbound WhatsApp sender.
