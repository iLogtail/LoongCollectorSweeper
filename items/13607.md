## Summary

Keep #13607 open. Current main still does not implement the requested global `agents.defaults.reasoningDefault`; it only supports per-agent `agents.list[].reasoningDefault`, and the related #71817 work is explicitly partial/channel-specific rather than a canonical global-default implementation.

## Best Possible Solution

Keep #13607 open until OpenClaw implements schema-backed `agents.defaults.reasoningDefault` with documented precedence such as directive/session override > per-agent default > global default > model-capability/off fallback, updates generated schema/help/docs/status behavior, and adds regression coverage across the main reply path and affected channel adapters. Treat #71817 as useful partial per-agent/channel work, not a replacement for this global default request.
