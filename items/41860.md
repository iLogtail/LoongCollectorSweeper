## Summary

Keep #41860 open. Current main still sends ordinary Feishu text as an unescaped Feishu `md` post element, and the reported CLI path can still reach that code path for a bare URL with underscores. The linked PRs/comments are useful and directly relevant, but the fix is not implemented on main and the issue remains a valid bug tracker rather than conservative cleanup.
