## Summary

Keep #11301 open. Current main still does not create a runtime-container `/workspace -> /home/node/.openclaw/workspace` alias or bind mount for Docker/Podman gateway or CLI containers. Existing `/workspace` behavior remains scoped to agent sandbox containers, and related PR #12764 was closed unmerged rather than shipped.

## Best Possible Solution

Keep #11301 open until a maintainer either implements a safe Docker/Podman runtime compatibility alias or bind mount for `/workspace -> /home/node/.openclaw/workspace` and aligns docs/smokes, or explicitly decides `/workspace` is sandbox-only and updates Docker/runtime documentation plus issue expectations. Treat PR #12764 as prior art, not a superseding fix, because it was closed unmerged.
