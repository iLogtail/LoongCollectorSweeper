## Summary

Current main still does not implement WebChat syntax highlighting. The WebChat markdown renderer emits escaped code text with an optional language class, language label, copy button, and JSON collapse wrapper, but there is no highlighter integration, highlighted token markup, or Control UI syntax theme. The existing maintainer/Codex comment remains accurate against current main.

## Best Possible Solution

Keep #10029 open until WebChat implements real syntax highlighting in `ui/src/ui/markdown.ts` with an approved client-side highlighter or markdown-it highlight integration, adds Control UI theme CSS, and adds focused tests proving highlighted fenced code while preserving copy-button, language-label, JSON-collapse, cache, and sanitization behavior.
