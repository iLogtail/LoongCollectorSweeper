## Summary

Keep open. Current `main` still contains the reported Ubuntu installer hang risk: the Linux installer uses `apt-get` in the piped `curl | bash` path without setting `DEBIAN_FRONTEND=noninteractive` or dpkg `--force-conf*` options, and the release SHA provided (`v2026.4.22` / `00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5`) shows the same behavior.
