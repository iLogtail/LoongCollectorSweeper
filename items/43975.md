## Summary

Keep PR #43975 open. Current main still lacks the Windows UTF-8 bootstrap and parser skip that the PR proposes, and the linked #43943 reproduction remains plausible on the current Scheduled Task `gateway.cmd` path.

## Best Possible Solution

Keep PR #43975 open for maintainer review. The best path is to merge or reimplement the narrow fix in `src/daemon/schtasks.ts`: emit `chcp 65001>nul` immediately after `@echo off`, teach `readScheduledTaskCommand` to skip that bootstrap line, and keep the Chinese-path install/readback regression tests in the scheduled-task test suite.
