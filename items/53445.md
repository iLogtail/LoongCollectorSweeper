## Summary

Keep open. Current main still renders the Control UI Sessions table `Updated` column as relative time only, while PR #53445 is a focused implementation for the still-open #53088 request to show full date-time in that list.

## Best Possible Solution

Keep the PR open for maintainer review or reimplementation. The best product path is a narrow Control UI change that formats the Sessions `Updated` column as a full local date-time, preserves a sensible fallback for missing/non-positive timestamps, optionally keeps relative time as hover context, and adds focused formatter/rendering tests.
