## Summary

Keep open. Current main does not implement the requested `file_push`/`file_pull` agent actions or `file.read`/`file.write` node command surface. The remaining work is core/protocol/app-facing work across the node host, gateway policy, CLI/tool surface, and platform node command manifests, so this is not a conservative ClawHub close.
