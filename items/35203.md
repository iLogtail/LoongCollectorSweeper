## Summary

Keep #35203 open. Current main still has isolated subagents, static/manual model and agent selection, usage/cost reporting, memory search, and plugin hooks, but it does not implement the RFC's central learned capability profiling, shared blackboard, layered private/team/global memory, or hard token/cost governance. The Octo/teleconference and KinthAI discussion is useful prior art, but it is partial, external, or roadmap material rather than a complete merged implementation in this checkout.

## Best Possible Solution

Keep #35203 open as an umbrella RFC until maintainers decompose it into focused follow-ups. Encourage plugin-first prototypes using runtime.subagent, taskFlow, before_tool_call, subagent lifecycle hooks, session visibility, and memory plugin seams where sufficient; file narrow core or Plugin SDK issues where those seams cannot express pre-spawn routing, shared visibility, telemetry, or budget enforcement.
