## Summary

Keep #51394 open. Current main has adjacent browser network inspection support through the browser control server and CLI, but it does not expose the requested `browser act kind=network`/Network-domain request interception surface to the Browser agent tool, and it does not implement request interception beyond existing navigation-guard routing.

## Evidence
