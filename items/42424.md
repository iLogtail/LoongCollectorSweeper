## Summary

Keep open. The linked PR #46338 directly targeted this bug but was closed unmerged, and current main still has the same fail-open shape: Discord self filtering depends on a resolved botUserId, fetch failure leaves that undefined, and the replay/followup path does not show an equivalent self-message guard or chat /new queue cleanup.
