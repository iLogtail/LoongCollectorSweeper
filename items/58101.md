## Summary

Keep open. Current `main` at `90877e0d4274c53fd91bf5787265c4d6889dc9a7` still only strips `HEARTBEAT_OK` from the start/end of a string, so the reported mid-text heartbeat case is not implemented on `main` and is still plausibly user-visible.
