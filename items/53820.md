## Summary

Keep open. Current `main` and the latest shipped release still use unbounded `Promise.all` fan-out for Ollama embedding batches, the documented session-indexing path is still supported, and this PR appears to add the missing bounded-concurrency fix plus regression coverage after addressing earlier fail-fast review feedback.
