## Summary

Keep open. Current `main` does not implement this PR's Discord thread relay suppression: the parent relay still enqueues all system events, the ACP spawn path does not compute or pass a Discord/thread-specific suppression flag, and the current tests still expect resumed relay chatter to be visible.
