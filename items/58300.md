## Summary

Current `main` still appears to return heartbeat user prompts through `chat.history`, which is the path the Control UI loads. The repo now has shared heartbeat-history filtering utilities and tests on adjacent history surfaces, but `chat.history` has not adopted them. The PR therefore should not be conservatively closed: the bug still looks real, and the proposed prefix-match implementation raised valid safety/test concerns in review.
