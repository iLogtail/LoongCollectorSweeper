## Summary

Keep #14407 open. Current main has partial Control UI reveal controls for gateway login/overview and config secrets, but the requested cross-platform password/API-key visibility toggle is still missing from Skills and native macOS/iOS/Android credential entry. The related PR #7124 is closed and unmerged in the provided GitHub context, and the existing maintainer Codex comment also points to remaining gaps.

## Best Possible Solution

Keep #14407 open unless a maintainer explicitly narrows the platform scope. The best path is a reusable sensitive-input reveal pattern for Control UI Skills/API-key fields, SwiftUI reveal wrappers for macOS/iOS SecureFields, Android Compose password fields using visualTransformation plus trailing icon toggles, and focused tests or view coverage for each touched credential surface.
