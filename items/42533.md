## Summary

Current main does not satisfy the requested fix. The macOS PTT and Talk Mode speech paths still pass the input device's native AVAudioPCMBuffer directly into SFSpeechAudioBufferRecognitionRequest without a mono/downmix conversion, and the reported emptyOnRelease path remains reachable when no transcript callbacks arrive.
