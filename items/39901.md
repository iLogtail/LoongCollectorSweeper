## Summary

Keep open. Current main suppresses `HEARTBEAT_OK` acknowledgments on chat surfaces, but it does not implement the requested option to hide/remove heartbeat poll prompt entries from stored transcript/chat history. The code explicitly keeps heartbeat transcript entries and only filters heartbeat pairs during agent context construction, while `chat.history` only drops assistant-side control/commentary messages.
