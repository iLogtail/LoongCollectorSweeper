## Summary

Keep open. Current `main` still recreates `HEARTBEAT.md` unconditionally in `ensureAgentWorkspace()`, so the reported behavior is not implemented or obsolete. The PR addresses a real gap, although its new negative-path test should model a fully initialized workspace more faithfully, as the existing review comment noted.

- Leaving this PR open means current `main` still overrides user intent by recreating a deleted `HEARTBEAT.md`, which can turn heartbeats into `empty-heartbeat-file` skips instead of letting them run without workspace guidance.
