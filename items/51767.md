## Summary

Keep open. Current main still gates the cron retention sweep on `isCronRunSessionKey()` only, so `cron.sessionRetention` does not prune completed `agent:*:subagent:*` or ACP session-store entries. The linked KeepMyClaw comment is a useful external backup/workaround, but it does not resolve the OpenClaw core cleanup bug. PR #51861 proposed the right area of change but was closed unmerged, and no current-main replacement is visible in the reaper tests or implementation.
