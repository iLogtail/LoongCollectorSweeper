## Summary

Keep open. Current main has strong evidence for fixes to the /hooks/agent target-agent session rebinding and plugin HTTP route registry 404 class, but the BlueBubbles phone-format routing mismatch remains: direct-message peer routing still compares the normalized sender string exactly, and BlueBubbles handle normalization does not canonicalize +1... and 1... as equivalent.
