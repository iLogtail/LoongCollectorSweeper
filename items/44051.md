## Summary

Keep open. Current `main` still applies realpath containment checks to all skill sources, including `openclaw-managed`, so the reported managed-skill symlink/junction failure mode is still plausible. The docs say that containment enforcement is for workspace and extra-dir discovery, but the implementation in `src/agents/skills/workspace.ts` does not scope it that way, and the current tests only cover workspace/bundled escape cases, not managed-skill regressions.
