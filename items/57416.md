## Summary

Keep open. Current `main` explicitly rejects the PR's core behavior: workspace `.env` is still treated as untrusted, `OPENCLAW_CONFIG_PATH` remains blocked there, and the tests codify that policy even while preserving the state-dir redirection security invariant discussed on the PR.
