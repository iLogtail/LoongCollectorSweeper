## Summary

Keep open. Current main does not provide high-confidence evidence that the proxychains-specific google-vertex ADC authentication failure is fixed. The remaining work is also not safely superseded by the related open Google GenAI PR, because that PR is not merged into current main and the current auth/transport path still treats google-vertex ADC as a synthetic marker rather than a real OAuth/ADC token path.

## Evidence

- **Issue is not protected but is recent and concrete:** Issue #52648 was opened 2026-03-23 by a non-maintainer with no protected labels and gives concrete reproduction steps, environment, observed 401 CREDENTIALS_MISSING errors, and a workaround of removing proxychains.
