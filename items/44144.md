## Summary

Keep open. Current `main` has not incorporated this docs clarification, but the PR as written still needs maintainer judgment because it documents Linux/other-platform use inside a skill that remains gated to macOS plus `codexbar` on `PATH`. The underlying Python script does support `--input`, so the docs idea is useful, but the wording should frame that as manual script usage unless the skill metadata changes too.
