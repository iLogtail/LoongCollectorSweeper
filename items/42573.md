## Summary

Keep open. Current main still has both macOS LaunchAgent tracks by design: `ai.openclaw.mac` for app launch-at-login and `ai.openclaw.gateway` for the local Gateway service. More importantly, the current extra-service scanner can still report `ai.openclaw.mac` as an extra gateway-like service, so the issue’s doctor/confusing dual-service symptom is not clearly fixed on main.
