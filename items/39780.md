## Summary

Keep open. Current `main` still implements the Overview/Control UI insecure-auth guidance as plain text telling the user to change `gateway.controlUi.allowInsecureAuth` manually. The Control UI already has config diff/apply machinery elsewhere, but that flow is not wired into these hardening/recommendation callouts, so the requested behavior is not implemented on `main` yet.
