## Summary

本 Issue 请求为 Elasticsearch Flusher 增加 Data Streams 模式支持。经核查当前 main 分支代码，该插件 Bulk 请求默认使用 index 操作，尚未提供 op_type: create 配置或 data stream 索引路由适配。该需求属于现有插件的合理功能增强，且维护者已在讨论区邀请社区贡献，故保持开启以跟踪后续实现。

## Best Possible Solution

保持开启以跟踪该功能需求。建议后续贡献者在 Elasticsearch Flusher 中新增 data_stream 或 op_type 配置开关，适配 ES Data Streams 必需的 create 语义与索引命名规范。维护者可添加 help wanted 标签，或等待社区提交设计草案与 PR。
