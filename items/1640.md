## Summary

本 Issue 请求为 Elasticsearch Flusher 增加 Data Streams 模式支持。经核查，当前 main 分支的 flusher_elasticsearch 尚未实现 data stream 所需的 op_type: create 及索引路由适配。该需求属于对核心输出插件的合理增强，边界清晰，需修改主仓库代码，不适合转为外部插件。符合保守保留标准，建议保持开启以跟踪社区贡献或后续排期。

## Best Possible Solution

保持开启作为功能需求跟踪。建议后续贡献者在现有 flusher_elasticsearch 插件中增加 data_stream 或 op_type 配置开关，适配 ES Data Streams 要求的索引行为与 Bulk API 路由。维护者可添加 help wanted 标签吸引社区实现，或等待原作者调研完成后提交设计草案与 PR。
