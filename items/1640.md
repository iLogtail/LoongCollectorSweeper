## Summary

本 Issue 请求为 Elasticsearch Flusher 增加 Data Streams 模式支持。该功能需适配 Bulk API 的 op_type: create 及索引路由规则，属于对现有输出插件的合理增强。目前主仓库尚未实现该特性，且已有维护者邀请社区贡献，作者亦在调研中，符合有效功能需求特征，故保持开启以跟踪后续实现。

## Best Possible Solution

保持开启作为功能需求跟踪。建议后续贡献者在现有 flusher_elasticsearch 插件中增加 data_stream 或 op_type 配置开关，适配 ES Data Streams 要求的索引行为与 Bulk API 路由。维护者可添加 help wanted 标签吸引社区实现，或等待原作者调研完成后提交设计草案与 PR。
