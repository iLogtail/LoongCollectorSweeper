## Summary

本 Issue 请求为 Elasticsearch Flusher 增加 Data Streams 模式支持。经核查当前 main 分支代码，ES Flusher 的 Bulk 请求默认使用 index 操作，尚未内置 data streams 必需的 op_type: create 配置或索引路由适配。该需求属于现有官方插件的合理功能增强，维护者已明确邀请社区贡献，目前仍缺少实现 PR，故保持开启以跟踪后续开发。

## Best Possible Solution

保持开启以跟踪该功能需求。建议贡献者在 Elasticsearch Flusher 中新增 data_stream 或 op_type 配置开关，适配 ES Data Streams 必需的 create 语义与索引命名规范。实现时需修改 Bulk 请求组装逻辑，将操作类型由 index 切换为 create，并补充相关集成测试。维护者可添加 help wanted 标签或等待社区提交设计草案与 PR。
