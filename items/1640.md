## Summary

本 Issue 请求为 Elasticsearch Flusher 增加 Data Streams 模式支持。经核查当前 main 分支的 flusher_elasticsearch 插件源码，尚未内置 data streams 专属适配（如 op_type: create 及 _data_stream API 路由）。该需求属于明确且有效的功能增强，虽作者自 2024 年 7 月后未继续跟进，但需求本身清晰且具备社区贡献价值，不符合保守关闭条件。

## Best Possible Solution

保持开启作为功能需求跟踪。建议后续贡献者在现有 flusher_elasticsearch 插件中增加 data_stream 配置开关，适配 ES Data Streams 要求的 op_type: create 索引行为与端点路由。维护者可添加 help wanted 标签吸引社区实现，或等待原作者调研完成后提交设计草案与 PR。
