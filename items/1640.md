## Summary

本 Issue 请求为 Elasticsearch Flusher 增加 Data Streams 模式支持。经核查，当前 main 分支的 flusher_elasticsearch 插件尚未实现 Data Streams 必需的 op_type: create 及索引路由适配。该需求属于对核心输出插件的合理增强，边界清晰，且已有项目成员邀请贡献，不符合任何保守关闭条件，建议保持开启以跟踪后续实现。

## Best Possible Solution

保持开启作为功能需求跟踪。建议后续贡献者在现有 flusher_elasticsearch 插件中增加 data_stream 或 op_type 配置开关，适配 ES Data Streams 要求的索引行为与 Bulk API 路由。维护者可添加 help wanted 标签吸引社区实现，或等待原作者调研完成后提交设计草案与 PR。
