## Summary

Keep open. Current `main` does not implement Anthropic Batch API support for cron `agentTurn` jobs, and the report is recent, concrete, and still actionable. The repo’s architecture also points any provider-specific implementation toward the Anthropic provider plugin rather than ClawHub, so there is no strong basis to close as `implemented_on_main`, `cannot_reproduce`, `clawhub`, `incoherent`, or `stale_insufficient_info`.

## Evidence

There is no `batch` option or batch-result state.
