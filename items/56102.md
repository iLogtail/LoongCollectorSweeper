## Summary

Keep open. Current `main` still depends on `@agentclientprotocol/sdk` for ACP initialize validation, and the pinned SDK schema still requires a numeric `protocolVersion`. A client sending `"2025-11-25"` would still be rejected as invalid params, so the reported VS Code/Cursor compatibility bug does not appear fixed on `main`.
