## Summary

Current `main` does not implement this PR's allowlist-based respawn skip. The respawn policy still skips only help/version invocations, so short-lived commands like `status` still take the respawn path. This PR remains a live, plausible performance improvement rather than something already shipped or obsolete.

## Evidence
