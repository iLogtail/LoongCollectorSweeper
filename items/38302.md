## Summary

Keep open. Current `main` does not implement per-account Slack native slash-command prefixes, and the Slack native-command path is still provider-wide rather than account-aware.

## Evidence

- **Slack slash config has no per-account prefix field:** `SlackSlashCommandConfig` only defines `enabled`, `name`, `sessionPrefix`, and `ephemeral`; there is no `nativePrefix` or equivalent account-level native command rename field.
  - file: [src/config/types.slack.ts:81](https://github.com/openclaw/openclaw/blob/53aac30f5104052db56da2b26bcee0f4e5140325/src/config/types.slack.ts#L81)
  - command: `nl -ba src/config/types.slack.ts | sed -n '70,95p'`
  - sha: [53aac30f5104](https://github.com/openclaw/openclaw/commit/53aac30f5104052db56da2b26bcee0f4e5140325)
- **Slack slash normalization only returns four fields:** `resolveSlackSlashCommandConfig()` normalizes only `enabled`, `name`, `sessionPrefix`, and `ephemeral`, confirming no runtime support for a native prefix option.
  - file: [extensions/slack/src/monitor/commands.ts:19](https://github.com/openclaw/openclaw/blob/53aac30f5104052db56da2b26bcee0f4e5140325/extensions/slack/src/monitor/commands.ts#L19)
  - command: `nl -ba extensions/slack/src/monitor/commands.ts | sed -n '1,80p'`
  - sha: [53aac30f5104](https://github.com/openclaw/openclaw/commit/53aac30f5104052db56da2b26bcee0f4e5140325)
- **Native command resolution is provider-wide, not account-aware:** The channel adapter hook `resolveNativeCommandName` receives only `commandKey` and `defaultName`, and `commands-registry.ts` resolves native names by provider only. There is no account id or slash-config input that could drive per-account prefixes.
  - file: [src/channels/plugins/types.adapters.ts:460](https://github.com/openclaw/openclaw/blob/53aac30f5104052db56da2b26bcee0f4e5140325/src/channels/plugins/types.adapters.ts#L460)

- **Slack native command registration uses global Slack names:** Slack registers native commands from `listNativeCommandSpecsForConfig(cfg, { provider: "slack" })` and looks them up with `findCommandByNativeName(command.name, "slack")`, with no account-specific prefix stripping or account-specific command list.
  - file: [extensions/slack/src/monitor/slash.ts:689](https://github.com/openclaw/openclaw/blob/53aac30f5104052db56da2b26bcee0f4e5140325/extensions/slack/src/monitor/slash.ts#L689)
