## Summary

Keep open. Current main does not implement per-account Slack native slash command prefixes or the later account-level nativeNames mapping proposal. The related PRs #38326 and #38391 were closed unmerged, and the current Slack config, schema, native command registry, and Slack registration path still only support provider/default native names.
