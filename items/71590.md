## Summary

Keep #71590 open. Current main still implements only the existing exponential temporal decay path, and the proposed Weibull `model`/`weibullShape` config is absent from runtime code, strict config schema, generated help/docs, and tests. The PR is a recent draft with concrete implementation work for an open memory-ranking request, but it still has actionable review feedback about inverted public JSDoc and needs maintainer/product review rather than cleanup closure.
