## Summary

Keep PR #70647 open. Current main already has the production empty-turn fallback, but it does not contain the PR's specific regression coverage for non-strict-agentic NVIDIA/Nemotron turns with unsigned thinking-only content or fully empty content. The existing tests cover adjacent paths only, so the test-only PR is still useful rather than implemented or superseded.

## Best Possible Solution

Keep the PR open and either merge the focused regression coverage after addressing the small issue-reference nit, or add equivalent tests on main that explicitly cover non-strict NVIDIA/Nemotron stopReason="stop" turns with unsigned thinking-only content and with content: []. Closing as implemented would lose useful coverage for a real provider shape that current tests do not pin.
