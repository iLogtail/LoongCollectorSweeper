## Summary

Keep open. Current `main` still intentionally rejects arbitrary local QQ Bot media paths outside the guarded OpenClaw media roots, so the requested auto-mirroring from arbitrary workspace files is not implemented. The report is recent, concrete, and the existing discussion raises a real security tradeoff, so there is not strong evidence for an auto-close.

## Evidence

- **Outbound delivery still hard-fails outside QQ Bot media storage:** `resolveOutboundMediaPath` accepts local files only if `resolveQQBotPayloadLocalFilePath(...)` or an explicit extra root approves them; otherwise it returns `...path must be inside QQ Bot media storage`.
  - file: [extensions/qqbot/src/engine/messaging/outbound.ts:307](https://github.com/openclaw/openclaw/blob/fe930b987e33f4e1acd93b5ef1ea76c8b7260047/extensions/qqbot/src/engine/messaging/outbound.ts#L307)
  - command: `nl -ba extensions/qqbot/src/engine/messaging/outbound.ts | sed -n '302,333p'`
  - sha: [fe930b987e33](https://github.com/openclaw/openclaw/commit/fe930b987e33f4e1acd93b5ef1ea76c8b7260047)
- **Current allowlist is the shared media root, not arbitrary workspace files:** `resolveQQBotPayloadLocalFilePath` canonicalizes the path and only allows files under `getOpenClawMediaDir()` or `getQQBotMediaPath()`. It does not copy or mirror arbitrary existing workspace files into QQ Bot storage.
  - file: [extensions/qqbot/src/engine/utils/platform.ts:327](https://github.com/openclaw/openclaw/blob/fe930b987e33f4e1acd93b5ef1ea76c8b7260047/extensions/qqbot/src/engine/utils/platform.ts#L327)

- **Tests intentionally preserve the restriction:** The QQ Bot platform tests explicitly assert that files outside QQ Bot storage are blocked, while only files inside `~/.openclaw/media/**` or legacy QQ Bot workspace remaps are allowed.
  - file: [extensions/qqbot/src/engine/utils/platform.test.ts:69](https://github.com/openclaw/openclaw/blob/fe930b987e33f4e1acd93b5ef1ea76c8b7260047/extensions/qqbot/src/engine/utils/platform.test.ts#L69)

  - file: [docs/channels/qqbot.md:189](https://github.com/openclaw/openclaw/blob/fe930b987e33f4e1acd93b5ef1ea76c8b7260047/docs/channels/qqbot.md#L189)
