## Summary

Current main does not honor `OPENCLAW_WRAPPER` or expose a `--wrapper` install option. The request is actionable in core daemon/service install code, and the related PR #69425 is still open rather than merged, so this issue should remain open until that PR or an equivalent implementation lands.

## Best Possible Solution

Keep #69400 open as the product issue until #69425 or an equivalent main-branch change lands. Maintainers should review whether the shipped fix should include only `OPENCLAW_WRAPPER` ProgramArguments support or also the requested `--wrapper` persistence and doctor status signal.
