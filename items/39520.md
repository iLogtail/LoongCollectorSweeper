## Summary

Keep open. Current main still does not implement this PR's proposed `channels.feishu.localRoots` / account `localRoots` / `"any"` Feishu config surface. Main has the narrower #27884 `mediaLocalRoots` propagation fix, but this PR remains a focused, reviewed Feishu media configuration change tied to #39506 rather than obsolete or safely closeable.

## Best Possible Solution

Keep this PR open or replace it with an explicit maintainer-authored follow-up. The remaining work belongs in the Feishu plugin: add a documented top-level and account-level `localRoots` schema/config surface if maintainers want it, resolve it from merged account config in `sendMediaFeishu()`, satisfy the core `loadWebMedia` `"any"` contract with an explicit `readFile` override, and add regression tests for array override, empty-array deny behavior, account override precedence, and `"any"` security semantics.
