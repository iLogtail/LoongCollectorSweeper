## Summary

Keep open. Current `main` still hardcodes the 5 MB media-store limit and reuses it for sandbox inbound media staging, while the config/types/docs only expose per-capability `tools.media.{image,audio,video}.maxBytes` settings. The reported gap is still present on `main` and in the latest release, so this is not conservative-close material.
