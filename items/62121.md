## Summary

Keep open. Current main has phase-aware and channel-policy mitigations, but the specific remaining report is untagged DeepSeek preamble in isolated cron delivery. The code still treats unphased assistant text as visible output, and cron has no provider-agnostic final-output boundary or configurable marker sanitizer. A maintainer comment in the supplied issue context also explicitly leaves this open for a cleaner design pass.
