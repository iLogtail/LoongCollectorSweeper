## Summary

本 PR 修复了匿名扩展插件 ID 生成冲突导致资源泄露的问题，以及非法 input 类型错误被吞没的问题。修改涉及 pluginmanager 核心配置加载与 Runner 停止逻辑，并补充了单元测试。当前 main 分支尚未合入此修复，PR 状态为 open 且可合并，属于有效的缺陷修复候选。

## Best Possible Solution

建议维护者审查本 PR。需重点确认 genPluginID 返回格式从纯数字改为 主插件ID_扩展序号 是否破坏现有监控、日志解析或插件元数据约定。若格式变更可接受或已对齐规范，且 CI 通过，建议合并以修复资源泄露缺陷。若格式需保持纯数字，请作者调整实现方案（例如使用独立的全局递增计数器）。
