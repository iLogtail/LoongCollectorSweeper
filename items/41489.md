## Summary

Keep open. Current `main` does not implement AssemblyAI audio transcription, so this is not `implemented_on_main`, and the request is coherent and still within OpenClaw’s bundled-provider/plugin scope. The main reason it is not merge-ready is architecture drift: current media-understanding providers are plugin-owned under `extensions/`, while this PR targets the old core `src/media-understanding/providers/*` layout.
