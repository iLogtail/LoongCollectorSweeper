## Summary

Keep open. Current main still contains the reported trigger chain: host gateway exec probes the login shell for PATH, that probe runs `$SHELL -l -c 'env -0'`, and `openclaw gateway start` still routes to a service restart on installed systemd services. The linked #62086 discussion describes a separate no-profile SIGTERM path and does not supersede this issue.
