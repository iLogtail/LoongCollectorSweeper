## Summary

Keep #49864 open. Current main handles the simple topic-config-over-group-config case, but the discussed activation override path is still ordered ahead of explicit topic `requireMention`, and the related fixing PR #49980 remains open/unmerged in the provided GitHub context.

## Best Possible Solution

Keep the issue open and either land a maintainer-reviewed version of #49980 or add an equivalent fix in `extensions/telegram/src/bot-message-context.ts` so explicit topic `requireMention` is resolved before persisted group activation, with regression coverage for topic config overriding `groupActivation: "mention"` and `groupActivation: "always"`.
