## Summary

Keep open. Current `main` still implements the raw config editor as a plain `<textarea>`, with tests and styles built around that textarea path, and there is no CodeMirror/Monaco-style editor dependency present. The requested structured JSON editor with folding/highlighting/validation is not implemented on `main` at `b98a6a46fb54145a33e753101c02f0d7d7e0bc3b`.
