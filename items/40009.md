## Summary

Keep open. Current main still renders the Control UI Raw config surface as a native textarea, with textarea-specific styling and tests, and no CodeMirror/Monaco/Ace/jsoneditor dependency or implementation found. The issue discussion recommends CodeMirror v6 but does not point to a shipped fix or canonical superseding item.

## Best Possible Solution

Keep the issue open and implement a first-class Raw config code editor in `ui/src/ui/views/config.ts`, likely using CodeMirror v6 with JSON/JSON5 support. Preserve `rawAvailable`, sensitive-value reveal/redaction, `onRawChange`, Reset to saved, `config.set`, `config.apply`, `baseHash`, and safe raw round-trip semantics, then update `ui/src/styles/config.css` and `ui/src/ui/views/config.browser.test.ts` to assert the new editor contract, folding/highlighting/formatting, and validation behavior.
