## Summary

Keep open. Current main still makes a single `client.wiki.spaceNode.list` call for `feishu_wiki` `nodes`, does not pass `page_size` or `page_token`, and does not loop on `has_more`. Related PR #37641 directly addressed the issue, but it is closed and unmerged, so it does not supersede the open bug.
