## Summary

Keep open. Current main still returns gateway-provided existing images without the prompt image sanitization path when no prompt image refs are detected, and the streaming lifecycle error path still only finalizes the SSE stream rather than writing error content for lifecycle-only errors. The provided PR context remains relevant: it fixes linked issue #59913 and later PR commits addressed the bot review concerns, so this is not a conservative cleanup close.
