## Summary

Current `main` still drops `--force-document` on the WhatsApp `message send` gateway path. The CLI accepts the flag, the action runner reads it, but the core gateway `send` RPC schema/handler do not carry it, and the WhatsApp send implementation still only sends documents when media is already classified as a document. This is not implemented on `main`, and it is not safe to close conservatively.
