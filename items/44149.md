## Summary

Keep open. Current `main` has an internal Feishu message-resource download helper, but it is only used for inbound media ingestion and is not exposed as an agent tool. The registered Feishu tools on `main` are still limited to doc/chat/wiki/drive/perm/bitable surfaces, and `feishu_drive` does not include any download-message-attachment action.
