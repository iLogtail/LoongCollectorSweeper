## Summary

Keep #38309 open. The request is concrete, contributor-authored, has only the non-protected `stale` label, and current `main` at 81a41fe5bee350c6a035bfcc458eb8be380afe2c still treats `sandbox.docker.env` as plain string-only config. The linked PR #38310 is useful context but was closed unmerged, and the older related #33073 covered other `apiKey` SecretRef surfaces rather than this Docker env surface.
