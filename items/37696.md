## Summary

Keep open. The reported strict-mode behavior is still present on current main: an unclosed reasoning tag leaves inThinking true, so strict mode does not append the trailing assistant text. Existing tests also still assert the old drop behavior, and channel/plugin streaming paths still call strict stripping.
