## Summary

Keep open. Current main does not implement the requested sandbox-common apt retry/fix-missing hardening: the documented setup path still builds Dockerfile.sandbox-common, whose apt update/upgrade/install commands have no Acquire::Retries or --fix-missing. The fallback base image path also uses plain apt-get commands, and the related #52724 context does not show a landed or superseding fix.
