## Summary

Keep open. Current `main` still has the reported mismatch: `openclaw doctor` suggests `openclaw sessions cleanup --store "..." --dry-run` for missing transcripts, but `sessions cleanup` only counts missing-transcript pruning when `--fix-missing` is explicitly enabled.

## Evidence
