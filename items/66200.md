## Summary

Keep open. On current `main`, BlueBubbles still sends inbound text to `BodyForAgent`, and the reply pipeline prefers `BodyForAgent` over the envelope `Body`. This PR only wraps the deprecated envelope field, so it does not fix the model-visible path described in the security summary. It also retains the empty-body divergence already called out in review.
