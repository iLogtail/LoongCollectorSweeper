## Summary

Keep open. Current `main` still carries the reported Doubao and BytePlus pricing bug: both providers use placeholder `input`/`output` values and zero `cacheRead`/`cacheWrite`, and those costs flow directly into usage-cost calculation. The PR is also stale against current tree layout because the relevant code now lives under `extensions/` instead of the old `src/agents/` paths.
