## Summary

Keep #51299 open. The report is concrete, recent, unprotected by no maintainer-only labels, and current main does not appear to implement or document a working FDA/Automation path for the legacy `imsg` iMessage channel when the gateway runs as a bare LaunchAgent process. BlueBubbles is documented as the recommended iMessage path, but that is a separate integration rather than a resolution for this LaunchAgent/TCC failure mode.
