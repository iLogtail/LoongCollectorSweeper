## Summary

Keep open. Current `main` still strips MiniMax XML tool-call text instead of promoting it into structured tool calls, and the MiniMax provider still uses the default hybrid replay policy that sanitizes tool-call IDs for `anthropic-messages`. That matches the report history that the bug is still unresolved on `main`, not already fixed.
