## Summary

Keep open. Current `main` does not implement `metadata.openclaw.activates.plugins` or any comparable per-turn, skill-driven plugin tool visibility. The existing mechanisms are static load-time skill gating and config-driven tool allow/deny policy, so the request is still an unshipped core feature rather than something already fixed or cleanly deferrable to ClawHub.
