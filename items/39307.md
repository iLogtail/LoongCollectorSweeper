## Summary

Keep #39307 open. Current main has partial compaction surfacing, but the requested default, hard-to-miss user signal plus assistant-side reset disclosure/re-grounding is still not implemented.

## Best Possible Solution

Keep this open for maintainer/product work. A complete fix should define the default compaction/reset visibility contract across supported chat surfaces, make the signal durable or hard to miss where feasible, add safe assistant-side disclosure and re-grounding behavior after compaction, and update docs/tests around agents.defaults.compaction.notifyUser and post-compaction recovery semantics.
