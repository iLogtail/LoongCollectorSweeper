## Summary

Current main has related citation/source metadata and a provenance-rich memory-wiki layer, but the requested active durable-memory provenance model is still not implemented. Memory-core promotion still records a single memory path/span, retrieval exposes only path/span/source/citation fields, and there is no enforced source-ref model for every durable memory, multiple supporting sources, or provenance retention across merges/edits.
