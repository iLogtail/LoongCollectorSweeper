## Summary

Keep open. Current `main` still rejects `auth-profiles.json` entries with `"type": "aws-sdk"` as `invalid_type`, while Bedrock docs still present `auth: "aws-sdk"` as the supported auth mode. The linked fix PR (#69753) was never merged; it was self-closed for inactivity, not because `main` already implemented the change.
