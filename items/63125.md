## Summary

Keep open. Current `main` contains several post-2026.4.8 follow-up fixes in the same areas (CLI gateway timeouts/teardown, Discord interaction expiry handling, Discord voice recovery, gateway channel-health/startup), but the evidence is still piecemeal. I do not have high-confidence proof that all four reported regressions in #63125 are resolved end-to-end on `main` or in `v2026.4.23`, so conservative cleanup should not close it.
