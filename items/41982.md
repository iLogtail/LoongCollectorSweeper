## Summary

Keep #41982 open. Current main has partial Telegram forum-topic icon support for create/edit message actions, but the requested persistent config field and gateway-start reapply behavior are still missing. The related PR #41989 directly addressed this issue, but the provided GitHub context says it closed unmerged, and current main does not contain its proposed config/startup behavior.
