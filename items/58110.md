## Summary

Current `main` already has substantial multi-agent infrastructure and a chat-oriented Control UI, but the specific request for a contact-style multi-agent chat UX with explicit group-chat collaboration is not clearly implemented end to end. The idea is also still in scope for core frontend ergonomics rather than being a ClawHub/plugin-only request, so conservative cleanup should leave this open.
