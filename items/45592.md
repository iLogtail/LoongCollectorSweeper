## Summary

Keep open. Anthropic officially made the 1M context window generally available for Claude Opus 4.6 and Sonnet 4.6 on March 13, 2026, with requests over 200K working automatically and no beta header required, but current `main` still implements the older beta-header path in code, tests, and docs. That means this is not implemented on `main`, and the PR should not be auto-closed as stale cleanup. ([claude.com](https://claude.com/blog/1m-context-ga))
