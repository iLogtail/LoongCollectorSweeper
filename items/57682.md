## Summary

Keep open. Current `main` has `windowsHide` safeguards, but the same Windows restart-helper `netstat | findstr` polling script is still present on `main` and in the latest release tag, multiple users confirmed the bug after the linked PR appeared in the timeline, and I could not perform a Windows reproduction from this checkout to prove the issue is gone.
