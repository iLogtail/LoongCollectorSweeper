## Summary

Keep open. Current `main` still generates the Windows Startup-folder fallback as a `.cmd` launcher that runs `start "" /min cmd.exe /d /c ...`, and the tests on `main` still assert that exact behavior. I did not find evidence in code, docs, or shipped changelog entries that the proposed silent `wscript`/`.vbs` launcher has been implemented.

## Evidence
