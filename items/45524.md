## Summary

Keep open. Current `main` still does not emit the requested startup dangerous-flag warning for explicit `agents.defaults.sandbox.mode="off"`. The PR's extra auth-related warnings are partly redundant with newer security-audit findings already on `main`, but the title's core sandbox warning is not implemented there.
