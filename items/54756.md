## Summary

Keep open. Current `main` already prefers `dist/index.js` for the update-command gateway refresh path, but the auto-update root-fallback path in `src/infra/update-startup.ts` still checks `dist/entry.js` before `dist/index.js`, so the PR's requested behavior is not fully implemented on `main` yet.
