## Summary

Keep #52662 open. Current main still treats the sandbox browser as a Docker-managed Chromium/noVNC container, has no `externalCdpEndpoint` or `externalNoVncEndpoint` sandbox-browser config, and rejects browser sandboxes for built-in non-Docker backends. Existing host remote-CDP support is related, but it does not implement the requested sandbox-browser external endpoint path.
