## Summary

Current `main` does not already implement this PR’s Telegram forum-topic cron-delivery fixes, and the PR discussion identifies a real unresolved correctness gap in the proposed patch. Conservative maintainer cleanup should keep this PR open rather than close it as implemented or obsolete.

## Evidence
