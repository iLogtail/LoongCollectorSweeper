## Summary

Keep open. Current `main` does not implement per-binding workspace overrides: route bindings still have no `workspace` field, routing does not carry a binding workspace through `ResolvedAgentRoute`, and reply execution still resolves workspace only from the agent-level config/default. Current docs also still direct users to create a second agent when they need separate workspaces.
