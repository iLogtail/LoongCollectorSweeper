## Summary

Keep open. Current `main` still exposes the requester session key to the target A2A run, still performs the peer ping-pong/announce follow-up path for normal sessions, and the only recent shipped fix in this area is the narrower ACP parent-child echo-loop case. The issue discussion also includes a fresh April 24, 2026 report that it still reproduces on a new install.
