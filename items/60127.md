## Summary

Keep open. The request is clear and recent, the author is not protected but the issue asks for a meaningful core security/product capability: shared-server tenant isolation with RBAC/user accounts. Current main has multi-agent routing and per-agent workspace separation, but project docs and gateway code explicitly do not claim hostile multi-tenant isolation or per-user tenant roles, so it is not implemented and should not be closed as ClawHub-scoped.
