## Summary

Keep open. Current main supports a single heartbeat-level model override and structured HEARTBEAT.md due-task parsing, but the parsed task schema only has name/interval/prompt and the runner sends all due tasks in one heartbeat model call using the merged heartbeat.model. The requested per-task model hint/escalation remains missing.
