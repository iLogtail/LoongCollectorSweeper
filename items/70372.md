## Summary

Keep #70372 open. Current main has phase-aware suppression for tagged commentary, but the specific Anthropic/Bedrock unphased thinking-plus-text path described by this PR is not implemented on main, and the PR still needs a provider-scoped fix/regression coverage before it is safe to land or close as superseded.

## Best Possible Solution

Keep the PR open, but require a narrower implementation before landing: either tag Anthropic/Bedrock text blocks with explicit phase metadata in the transport layer, or derive the fallback only for known unphased Anthropic/Bedrock streams after seeing native thinking events. Add a regression that emits `thinking_start`/`thinking_delta`/`thinking_end` followed by unphased `text_delta`/`text_end`, verifies reasoning stays off user-visible channel replies, and verifies OpenAI phase-pending text is not misclassified.
