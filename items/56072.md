## Summary

Keep open. Current `main` still rolls stale daily sessions by archiving the prior transcript and then firing `session_end`, with no rollover path into the compaction memory-flush logic. The bundled `session-memory` hook is still limited to explicit `/new` and `/reset`, and session reset config still only supports `daily` and `idle`, so the reported gap remains on `7130e56f872aea96577ac9fcbda0c7555a9ecc69`.
