## Summary

Keep open. Current main has useful adjacent pieces (cron execution records, background task tracking, failure notifications, and a reminder-specific guard), but it does not implement the requested native commitment/deferred-task primitive. There is no `openclaw commit add/list/done` surface, no general commitment ledger, and the existing guard only warns on reminder-style phrasing when no cron was created.
