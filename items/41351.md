## Summary

Keep open. Current `main` does not contain the PR’s executable-bit fix or the release-time guard: both bundled skill shell scripts are still mode `100644`, the skill docs still tell users to invoke them directly, and OpenClaw’s executable resolution requires Unix execute permission for path-based commands. The latest release commit `00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5` still shipped these scripts non-executable, so this is not implemented on `main` and not obsolete.
