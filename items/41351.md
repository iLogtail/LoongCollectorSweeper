## Summary

Keep open. Current main still ships the bundled Whisper and video-frame shell scripts as non-executable files, the skills instruct direct script execution, and main has no release-time executable-bit guard. The PR targets a real remaining repo bug, though it should be cleaned/rebased because its head includes unrelated WhatsApp test changes and earlier review comments noted the guard must cover all existing skill shell scripts.

## Best Possible Solution

Keep the PR open but require a scoped rebase/fixup: restore executable bits for all bundled shell scripts that are invoked directly, add the release-time checker with regression tests, remove unrelated WhatsApp changes, and rerun the relevant release-check/test lane before merge.
