## Summary

Keep PR #62648 open. Current main still formats schema-validator AJV additionalProperties failures with the generic AJV message, and plugin/channel config validation surfaces that generic message to users. The linked dreaming config path is now accepted by the memory-core manifest schema, but the PR's generic property-name improvement remains unimplemented and is a focused, useful follow-up rather than obsolete.
