## Summary

Keep #65691 open. Current main still lacks Slack message search support: there is no Slack `searchSlackMessages` helper, no `searchMessages` action runtime branch, no Slack generic `search` dispatch branch, and the CLI/docs still treat `message search` as Discord-only. The PR is an active rebuilt implementation attempt, not a shipped fix or a duplicate of the closed-unmerged #58143.

## Best Possible Solution

Keep this PR or a successor open and finish the bundled Slack plugin implementation: add a tested `search.messages` action/runtime path, expose `search` only for runnable account/token configurations, preserve or replace Discord `--guild-id` validation with a clear provider-specific error, resolve Slack channel scoping, and update the message CLI/docs before merge.
