## Summary

Keep open. Current main has absorbed some of this batch, but the PR is not obsolete: the Discord Unknown Channel permanent-error rule is still missing, chat.history still lacks a delivery-mirror filter, and the plugin duplicate-warning change conflicts with current tested registry behavior. The PR also has unresolved review comments that would need a rework before merge.

## Best Possible Solution

Keep this PR open only if it is reworked into current-main-shaped fixes: add the Discord Unknown Channel/10003 permanent-error classification with a focused delivery-queue test, implement any webchat delivery-mirror filtering before limit pagination with a regression test, preserve the current completion-route merge behavior, and align any plugin duplicate-warning change with the registry's installed-plugin precedence policy. If the author does not update it, maintainers should prefer smaller targeted PRs for the remaining bugs rather than merging the batch.
