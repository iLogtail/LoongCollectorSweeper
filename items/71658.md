## Summary

Keep #71658 open. Current main still has the reported strict-agentic planning-only retry gaps: the actionable prompt gate rejects the reported imperative prompt, and the assistant planning detector still misses present-continuous narration such as “I'm running it now.” The prior ClawSweeper comment’s keep-open conclusion still matches current main at c74fb781943f3bad7d6fec5256729c861d285cab.

## Best Possible Solution

Keep this issue open and patch `src/agents/pi-embedded-runner/run/incomplete-turn.ts`. The conservative fix should add regression tests for the exact non-allowlisted imperative prompt and for an accepted ack prompt followed by zero-tool present-continuous narration, then broaden actionable prompt and planning-only narration detection without retrying casual chatter or completed-result summaries.
