## Summary

本 PR 修复了 Prometheus 抓取器在目标返回 404 或错误页面时，因首字符校验过宽而将非指标内容误判为指标的问题。通过收紧 IsValidMetric 校验逻辑以符合 Prometheus 官方数据模型规范，并在抓取失败时清理事件组与样本计数，提升了采集准确性。目前 PR 处于开放待合并状态，维护者已进行代码审查且作者已回应规范依据疑问，建议保持开启等待最终审查与合并。

## Best Possible Solution
