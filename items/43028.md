## Summary

Keep #43028 open. Current main has adjacent cron hardening: execution timeouts, recurring error backoff, failure alerts, isolated-run session retention, and per-job run-log pruning. However, the requested recurring-job circuit breaker/auto-pause remains missing, `openclaw cron rm` still only removes the job record and has no cleanup flag/path, and there is no core `.learnings/changelog.md` cron changelog feature. The existing comment about 2026.3.8 cron execution bugs is also relevant background, but current main’s improvements do not fully satisfy the feature request.
