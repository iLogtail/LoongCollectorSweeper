## Summary

Keep open. Current main has adjacent prompt guidance through Execution Bias, Tool Call Style, Safety, and the GPT-5 overlay, but it does not implement the requested all-model systematic Agent Behavior Principles or an explicit surgical/scope-control rule. The related #71937 context is closed design-only and treats #69291 as a related issue, not a shipped replacement.

## Best Possible Solution

Keep the issue open for maintainer product judgment. If accepted, implement the behavior in the current prompt architecture under src/agents/system-prompt.ts and related provider overlay docs/tests, likely by extending Execution Bias or adding a clearly named core section, with coverage for all relevant prompt modes and model families. The obsolete proposed src/core/prompts/system-prompt/components path should not be followed literally unless the prompt architecture is refactored first.
