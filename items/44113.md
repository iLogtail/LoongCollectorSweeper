## Summary

Keep open. Current `main` does not implement the proposed rescue-watchdog onboarding flow: the CLI has no `--rescue-watchdog` flag, onboarding options/types do not carry a rescue setting, cron still only supports `systemEvent` and `agentTurn` payloads, and docs still direct users to set up a separate manual `--profile rescue` gateway instead.

## Evidence
