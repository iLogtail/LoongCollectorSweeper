## Summary

本 Issue 请求为 flusher_kafka_v2 增加本地磁盘缓存（DiskBuffer）能力以提升断网或 Kafka 故障时的灾备与防丢失能力。维护者已确认需求价值并明确邀请社区贡献。当前 main 分支该插件仍主要依赖内存队列与 checkpoint 机制，尚未内置磁盘持久化缓存。属于合理且需插件/队列层架构改动的增强请求，应保持开启以跟踪设计讨论或社区 PR。

## Best Possible Solution

保持开启以跟踪该增强需求。建议贡献者或维护者后续明确磁盘缓存的设计方案（如复用或扩展现有 Queue/Buffer 架构、配置参数、刷盘策略、与 Checkpoint 的协同及异常恢复逻辑）。欢迎社区提交设计草案或直接贡献 PR。在实现前，用户可考虑调整内存队列大小、优化 Checkpoint 策略或结合外部可靠传输方案作为临时替代。
