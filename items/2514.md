## Summary

本 Issue 请求为 flusher_kafka_v2 增加本地磁盘缓存（DiskBuffer）能力以提升灾备与防丢失能力。维护者已确认需求价值并明确邀请社区贡献。该功能涉及底层队列与缓冲架构设计，当前 main 分支尚未内置该能力，属于合理且需进一步设计或社区 PR 的增强请求，应保持开启以跟踪后续进展。

## Best Possible Solution

保持开启以跟踪该增强需求。建议后续由贡献者或维护者提交设计方案（如复用或扩展现有 Queue 架构、增加磁盘刷盘策略、配置参数及与 Checkpoint 的协同机制），或直接提交 PR 实现。在功能落地前，用户可通过调大内存队列、优化 Checkpoint 间隔或结合外部可靠传输组件作为临时缓解措施。
