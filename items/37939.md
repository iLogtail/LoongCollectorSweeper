## Summary

Current `main` already addresses this Telegram stale-socket report. Telegram polling now records transport activity on successful long-poll responses, the gateway exposes a configurable stale threshold, and coverage exists for both the Telegram idle case and the config override. The same implementation is present in release `v2026.4.22` (`00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5`).
