## Summary

Keep open. Current `main` still enforces the old behavior this PR is changing: bundled plugin runtime deps are only treated as satisfied from the plugin install root/package install root, not from the OpenClaw package root `node_modules`, and the current tests explicitly lock that behavior in. The PR is therefore not already implemented on `main`, and the review discussion indicates a plausible real packaged-deployment win with only minor review nits so far.
