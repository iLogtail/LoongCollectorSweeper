## Summary

Current main still leaves the reported UX gap: the Cron History button only selects/loads run history into the existing lower-page panel, with no scroll, focus, modal, drawer, or other explicit feedback. The issue is recent, contributor-authored, and only labeled enhancement, so it should remain open for a UI fix.
