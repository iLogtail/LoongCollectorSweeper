## Summary

Keep #43260 open. Current main supports model overrides for cron jobs and plugin hooks, but it does not implement the requested `model` field in `SKILL.md` frontmatter or any skill-aware temporary model switch. The related #11826 cron gap appears addressed separately, and #61391/openclaw-intent-router are useful adjacent routing prior art, but they do not supersede per-skill frontmatter routing.
