## Summary

Keep open. Current main already has a refreshed Control UI login gate and clearer auth-error messaging, but the issue's remaining OAuth-login piece is not implemented: the login page only exposes gateway URL plus token/password fields, and core gateway auth modes still do not include a native Control UI OAuth mode.
