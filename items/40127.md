## Summary

Keep open. Current main already has a polished Control UI login gate, responsive layout, and better structured auth error messages, but the specific OAuth login flow requested in issue #40127 is not implemented on main. The dashboard login UI still only supports gateway URL plus shared token/password entry, and the documented dashboard auth paths are token/password/Tailscale/trusted-proxy rather than OAuth.
