## Summary

Keep open. Current `main` does not implement macOS gateway sleep prevention, the docs still describe `caffeinate` as a manual workaround, and the PR itself has an acknowledged unresolved correctness issue around `spawn()` error handling. This is not strong evidence for any close path.
