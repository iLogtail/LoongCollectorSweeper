## Summary

Current main does not show a conservative basis to close #43020. The Mattermost inbound draft-preview path still disables core block streaming, only resets local partial-text bookkeeping at assistant message boundaries, and does not rotate/finalize the preview post for each assistant turn. The related fix PRs #43041 and #46330 were closed unmerged, so they do not supersede the open issue.
