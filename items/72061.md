## Summary

Keep open. Current main still has the reported reload-diff behavior: `diffConfigPaths` treats a missing field and an empty object as different, the reload planner treats `plugins.*` paths as gateway-restart paths, and the Skills UI path writes skill toggles through `skills.update`/`writeConfigFile`. PR #72137 appears to be the right proposed fix, but it is open/unmerged in the provided context, so the issue is not implemented on current `main`.

## Best Possible Solution

Keep #72061 open until PR #72137, or an equivalent main-branch fix, lands. The best fix is to make `diffConfigPaths` collapse equivalent empty/no-value config branches such as `undefined`, `{}`, and `[]` while still reporting missing-to-populated containers, then add focused regression coverage in `src/gateway/config-reload.test.ts` for the full skill-toggle phantom-diff scenario and the micro edge cases.
