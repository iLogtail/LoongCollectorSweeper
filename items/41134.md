## Summary

Keep open. Current `main` still does not expose `browser.ssrfPolicy.allowRfc2544BenchmarkRange` in the browser config surface, schema, or docs. The underlying SSRF engine already supports the flag, so the request remains valid, but this PR is stale against the browser-extension refactor and would need to be rebased onto current paths.
