## Summary

Keep #11146 open. Current main still lacks the requested dashboard-level keyboard navigation and dashboard New Agent wizard; it only has the existing Cmd/Ctrl+K command palette and lower-level agent creation/file APIs.

## Best Possible Solution

Keep #11146 open or split it into two narrower maintainer-tracked tasks. The keyboard task should add a dashboard-level shortcut dispatcher with focus guards for chat/navigation actions. The agent task should add an Agents dashboard creation wizard that uses the existing agents.create, agents.files.*, and session APIs, supports workspace/bootstrap customization such as SOUL.md and USER.md, and includes focused Control UI tests plus user-facing docs.
