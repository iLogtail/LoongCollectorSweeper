## Summary

Keep #11146 open. Current main still lacks the requested dashboard-level shortcut set and a dashboard New Agent creation wizard; the repo only has Cmd/Ctrl+K command palette support plus CLI/RPC-level agent creation and file editing APIs. The recent maintainer/Codex comment in the issue also explicitly kept this open, and the current checked-out main still matches that assessment.

## Best Possible Solution

Keep the issue open or split it into two maintainer-tracked tasks. The keyboard task should add a dashboard-level shortcut dispatcher with focus guards and tests for chat/navigation actions. The agent task should add a Control UI New Agent wizard that reuses agents.create, agents.files.*, config/session APIs, supports workspace/bootstrap customization such as SOUL.md and USER.md, and includes focused UI tests plus user-facing docs.
