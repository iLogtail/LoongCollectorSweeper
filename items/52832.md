## Summary

Keep open. Current `main` does not implement the Liquid Glass bottom bar described in PR #52832, and the review thread surfaced a plausible bottom-content regression in the proposed `.overlay` approach. The PR also picked up an unrelated bundled-provider registry change that no longer matches the current manifest-first registry design, so there is not strong evidence to auto-close it as already implemented or obsolete.
