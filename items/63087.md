## Summary

Keep open. Current `main` still sends remote memory-embedding HTTP through the strict SSRF-guarded fetch path, and that path explicitly ignores ambient proxy env vars unless the caller opts into trusted env-proxy mode. The memory index remote embedding path does not opt into that mode or pass a proxy-aware fetch implementation.

## Evidence

  - file: [src/infra/net/fetch-guard.ts:357](https://github.com/openclaw/openclaw/blob/cf858258c71701ef62dddf88aa777873a4962046/src/infra/net/fetch-guard.ts#L357)
