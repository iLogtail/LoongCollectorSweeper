## Summary

Keep open. Current main still overwrites a configured `models.providers.github-copilot.baseUrl` with the token-derived Copilot endpoint during runtime auth, so the request in PR #44136 is not implemented on `main`. The original patch target is stale after the runtime-auth refactor, but the underlying behavior still needs a fix.

  - file: [src/agents/pi-embedded-runner/model.ts:366](https://github.com/openclaw/openclaw/blob/50e36983bb2d827adc617282c090c0a0359c6eee/src/agents/pi-embedded-runner/model.ts#L366)
