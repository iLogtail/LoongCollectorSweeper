## Summary

Keep open. Current main does not implement the requested BlueBubbles numeric allowlist normalization, and the linked user bug remains plausible. The PR also has a real cross-channel regression risk: its schema-level coercion would affect every channel using the shared AllowFromListSchema, including QQBot numeric IDs, so this needs revision or maintainer judgment rather than conservative auto-close.
