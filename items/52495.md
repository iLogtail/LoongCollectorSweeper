## Summary

Keep open. The PR’s one-line doc change is not implemented on current `main`, and the proposed wording is inaccurate: `memory_search`/`memory_get` are provided by the active memory plugin, not unconditionally as built-in tools available without plugins. The existing review discussion correctly identified that problem, so this needs maintainer judgment or a reworked doc change rather than auto-closure.
