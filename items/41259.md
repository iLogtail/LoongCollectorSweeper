## Summary

Keep open. Current `main` still swallows `mkdirSync(..., { recursive: true })` failures in the extracted memory host SDK, so the reported bug is not implemented on `main`. The original PR path is stale after the memory host refactor, but the underlying behavior remains live and is still on the memory database open path.
