## Summary

Keep open. Current `main` still contains the stale `lastAssistant` fallback path this PR is trying to fix, but the proposed `agent-runner.ts` early-return approach would conflict with current timeout and post-compaction contracts, so this item is not safely closable as already-implemented or obsolete.
