## Summary

Keep open. Current main does not implement job-level model/provider overrides for `sessionTarget: "main"` + `payload.kind: "systemEvent"`; model selection remains limited to `agentTurn`/isolated cron jobs, and the related PR #42708 is closed unmerged per the supplied GitHub context.

## Evidence

  - file: [src/cron/types.ts:112](https://github.com/openclaw/openclaw/blob/e640c0a95fcbd06a1311f18fc635518d011d45da/src/cron/types.ts#L112)
