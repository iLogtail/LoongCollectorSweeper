## Summary

Keep #71324 open. Current main only has the earlier display projection for canonical internal runtime blocks, heartbeat/internal-only rows, and silent assistant rows; it does not include this PR's legacy quote-wrapper, background-task status, gateway-restart status, or mixed-row preservation logic. The related #71312 was closed unmerged and superseded by this PR, while #71229 fixed the forward write path and only part of the legacy projection surface.

## Best Possible Solution

Keep #71324 open as the active vehicle for the remaining legacy WebChat/history projection gap. The best path is to land a constrained version of this fix, with regression tests for quote-wrapped internal context, background-task and gateway-restart status rows, mixed rows preserving real user text, and rawTranscriptSeq preservation, after addressing the existing review comments.
