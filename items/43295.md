## Summary

Keep #43295 open. Current main does not implement the requested forced recall-before-response mechanism, and the related implementation PR #43825 is closed unmerged. Existing memory/plugin paths provide advisory prompt injection, but they do not provide the requested core runtime guard with enforce/block behavior across the affected agent execution surfaces.
