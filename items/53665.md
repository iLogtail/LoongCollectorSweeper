## Summary

Keep open. Current `main` still performs the full `map/filter/toSorted/slice` fallback vector search, and current tests cover sqlite-vec KNN but not the bounded top-k fallback behavior proposed here. The PR’s original target path has since moved into `extensions/memory-core`, so this looks like a rebase/porting problem, not an obsolete change.
