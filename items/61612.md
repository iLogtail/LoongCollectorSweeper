## Summary

Current main has related CLI startup work, but it does not implement the PR’s requested built-in/plugin subcommand help fast path. In particular, plugin command help still uses the full plugin command registry path, and memory-core still imports runtime modules from its main entrypoint. The PR also has unresolved review findings about correctness regressions, so it is not safe to close as implemented or superseded.
