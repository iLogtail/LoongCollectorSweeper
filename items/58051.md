## Summary

Keep open. Current `origin/main` still allows a primary transient-cooldown probe to consume `cooldownProbeUsedProviders`, so the same-provider fallback bug this PR fixes is not implemented on main. The PR discussion also added the missing rationale comments and a follow-up dedup regression test, so the branch still contains useful work rather than an obsolete change.
