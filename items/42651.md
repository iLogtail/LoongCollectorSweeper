## Summary

Keep open. Current main has meaningful partial memory surfaces (`memory-core` CLI/tools and `memory-wiki` ingest/apply/search), and the issue discussion usefully links Engram Core as an external implementation. But #42651 asks for first-class `remember`/`recall`/`review`/`edit`/`forget` commands and ingestion helpers; current main still lacks that first-class operator surface, especially explicit remember/forget/review/edit flows. The related #49108 covers ingestion quality improvements, not the full CLI/skill MVP.
