## Summary

本 PR 修复了 RFC3164 syslog 解析中 TAG 字段未限制长度导致的数组越界缺陷，修改位于 vendored 第三方库目录。属于有效缺陷修复，需维护者确认依赖管理策略并补充边界测试后合并。

## Best Possible Solution

建议维护者评审本 PR。因修改涉及 external/ 下的 vendored 代码，需确认项目当前的第三方依赖管理策略：若仍维护本地副本，可合并此修复，但强烈建议补充针对 32 字符边界、超长 TAG 及异常分隔符的单元测试；若已迁移至 Go Modules 或计划同步上游，建议将修复提交至 jeromer/syslogparser 上游仓库，并在本仓库通过升级依赖版本解决。在明确依赖策略与测试覆盖前保持开启。
