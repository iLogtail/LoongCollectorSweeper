## Summary

本 PR 修复了 RFC3164 syslog 解析中 TAG 字段未限制长度可能导致的数组越界问题，符合 RFC 规范且源于生产环境反馈。修改位于 external vendored 目录，需维护者确认依赖管理策略后评审合并。

## Best Possible Solution

建议维护者评审本 PR。需确认：1. 项目当前是否仍直接使用 external/ 下的 vendored 代码，或已迁移至 Go Modules；2. 若仍使用该解析器，可合并此修复或考虑同步上游/切换更活跃的 syslog 解析库；3. 建议补充针对 32 字符边界、超长 TAG 及特殊分隔符场景的单元测试后合并。
