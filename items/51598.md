## Summary

Keep #51598 open. Current main supports only full system prompt replacement via `agents.defaults.systemPromptOverride` and `agents.list[].systemPromptOverride`; it does not implement the requested append-style `agents.defaults.systemPrompt` / per-agent `systemPrompt` config that preserves OpenClaw's assembled prompt. Existing plugin hooks can add system context, but that is an extension hook, not the durable core config requested here. Related #54504 is narrower subagent-instruction-file work and does not supersede this issue.
