## Summary

Current `main` already includes mitigations for stale post-compaction context, but I did not find strong proof that the reported auto-compaction/topic-switch bug is fully fixed. The code and tests show prompt reinjection after compaction, not a direct guarantee that a new user question will override an old compaction summary without repetition.
