## Summary

Keep open. Current `main` already emits a separate terminal stop chunk on the normal success path, so part of the PR is superseded, but the streaming error path still sends `delta.content` and `finish_reason: "stop"` in the same SSE choice and the tests still assert that behavior. That means the spec-alignment request is not fully implemented on `main` yet.
