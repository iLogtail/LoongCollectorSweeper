## Summary

Keep open. Current `main` still models exec approvals as resolved-path allowlists, not command-content deny rules, and the docs still describe shell wrappers as bypassable once the shell binary itself is allowlisted.

## Evidence

  - file: [docs/tools/exec-approvals.md:76](https://github.com/openclaw/openclaw/blob/e94c0bf51542fe518e49ad4e2ee29882891b5603/docs/tools/exec-approvals.md#L76)
