## Summary

Keep open. Current `main` still ships and tests the Kimi Coding `User-Agent: claude-code/0.1.0` override that commenters identified as the workaround trigger, while the only in-repo handling for this failure is UI sanitization of the invalid stream-order error rather than a transport fix.
