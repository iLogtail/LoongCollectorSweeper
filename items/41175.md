## Summary

Current `main` already addresses this Discord attachment wedge path: attachment fetches now have a 120s total timeout, failed downloads fall back instead of blocking the turn, remote fallback URLs are not re-injected as native images, and the prompt image loader clamps images to the 6MB image budget with image optimization. The same implementation is already present in release `v2026.4.22`.
