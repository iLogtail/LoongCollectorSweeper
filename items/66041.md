## Summary

Keep open. Current `main` still appears to have the Telegram ack-reaction mismatch described by the PR: unmentioned group messages can proceed when `requireMention=false`, but the shared `group-mentions` ack gate still returns false for that same configuration. The proposed PR is coherent, narrowly scoped, and adds the missing regression coverage.
