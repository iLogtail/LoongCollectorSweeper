## Summary

Keep open. Current main still does not implement the requested Volta-specific normalization: gateway install resolves the selected Node path through `resolveStableNodePath`, but that helper only rewrites Homebrew Cellar paths and returns non-Cellar paths unchanged, so `/Users/<user>/.volta/tools/image/node/<version>/bin/node` would remain version-pinned instead of becoming `~/.volta/bin/node`. The related open PR #52252 remains the narrower implementation track, but the issue itself should not be closed until that work lands or is otherwise superseded.
