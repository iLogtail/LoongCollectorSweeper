## Summary

Current `main` does not contain the PR's help-text clarification. The config help source and checked-in generated schema still use the older generic wording, while separate docs already list the tool profiles. This PR is therefore not implemented on `main`, and there is not a strong maintainer-cleanup basis to close it.
