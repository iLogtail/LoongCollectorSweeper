## Summary

Keep open. The RFC is fresh, clear, and not protected by maintainer association or protected labels. Current main has adjacent trust mechanisms: generic prompt guidance to prefer tool evidence, input provenance metadata, and memory-wiki structured claims with confidence/evidence. Those do not implement the requested cross-agent behavior of confidence tiers/source attribution on every factual claim or autonomous completion reporting for unverified claims. This also should not be closed as clawhub-only yet because the request cuts across core agent output and completion-report contracts, and I found no existing linked plugin/extension or complete current plugin surface that makes the core item obsolete.
