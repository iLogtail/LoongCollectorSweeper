## Summary

Keep open. Current `main` still does not implement `appendContext` or full `prompt` replacement for `before_prompt_build`/legacy prompt hooks, and the user-facing docs/tests on `main` still describe only prepend/system-prompt mutations. The PR remains actionable cleanup rather than a conservative close candidate.
