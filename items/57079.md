## Summary

Current `main` now detects and repairs gateway service entrypoint drift, preserves stable service entrypoints across package updates, and documents the separate browser-plugin allowlist failure mode that also produced `browser.request` errors in this thread. The fix appears to be present on `main` via commit `68315792677f9628d1370d6869366442f1d3ecad` and is not in the latest tagged release yet.
