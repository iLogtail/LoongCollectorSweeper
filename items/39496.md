## Summary

Keep PR #39496 open. Current main at edf40ab6c9440604fe6a74aeaa8d8bc81b5ee16b still lacks several central pieces from the PR: Feishu ops tools/skills, event_id/create_time dedupe, recall-abort buffer handling, semantic streaming/card changes, the typing emoji switch, and production daemon working-directory resolution. ## Best Possible Solution

Keep PR #39496 open for maintainer triage, but ask for a refreshed split against current main: one focused Feishu streaming/card delivery fix, one event_id/create_time dedupe fix with bounded and account-safe semantics, one recall/fast-abort fix, one daemon working-directory fix, and a separate Feishu ops tools/skills decision that either lands with account-scoped tests and secret-safety cleanup or moves to ClawHub/plugin packaging. Each slice should address the existing review findings and add targeted tests.
