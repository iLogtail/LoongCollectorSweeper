## Summary

Keep PR #72319 open. Current main still has the reported Control UI leak path: tool start args are emitted raw, text tool results are only truncated, and exec command output can be rebuilt from raw result details before being sent to Control UI subscribers. The PR is an active, focused security fix for open security issue #72283 and is not obsolete or implemented on main.

## Best Possible Solution

Keep the PR open for maintainer review, or land/rework an equivalent core fix in `src/agents/pi-embedded-subscribe.tools.ts`, `src/agents/pi-embedded-subscribe.handlers.tools.ts`, and `src/logging/redact.ts`. The fix should redact tool args, text content blocks, exec aggregated output, and related structured summaries before any Control UI websocket/session event emission, while preserving user-configured `logging.redactPatterns` and adding focused regression coverage.
