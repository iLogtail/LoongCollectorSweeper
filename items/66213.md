## Summary

Keep open. Current `main` still does not include the proposed post-publish macOS asset verification workflow or the `appcast.xml` CODEOWNERS protection, and the existing release docs/workflow still describe this gap as a manual release-manager follow-through rather than an implemented safeguard.

## Evidence
