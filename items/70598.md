## Summary

Keep PR #70598 open. Current main already implements the core exec heartbeat prompt fix, but the PR still contains useful follow-up work that current main does not have: preserving deferred non-exec and duplicate reminder events after an exec-event heartbeat run. The later PR discussion about duplicate deferred reminders remains relevant to current main.

## Best Possible Solution

Keep the PR open, but ask for a rebase against current main. Retain the shipped #71213 exec prompt behavior, then land only the remaining queue-consumption fix and regression tests that preserve non-exec and duplicate deferred events after exec-event heartbeat runs.
