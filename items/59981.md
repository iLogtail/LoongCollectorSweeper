## Summary

Keep open. Current main still appears to contain the reported Mattermost DM threading bug: the docs and reply mode resolver say direct messages are non-threaded, but the outbound reply-root helper still has no chat-kind guard and can fall back to payload.replyToId. PR #60115 is a relevant proposed fix, but it is still open/unmerged in the provided GitHub context and current main does not include that fix.
