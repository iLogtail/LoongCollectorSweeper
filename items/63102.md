## Summary

Keep open. Current `main` still logs duplicate plugin tool names at resolution time instead of deduplicating them, and that resolution path is used when building agent tools. The bundled QQ Bot plugin still registers `qqbot_channel_api` and `qqbot_remind`, while the docs also still advertise installing the external `@tencent-connect/openclaw-qqbot` plugin, so the reported overlap remains plausible and not clearly fixed on `main`.
