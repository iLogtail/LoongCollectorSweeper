## Summary

Keep open. Current `main` does not contain the PR’s fix. The Anthropic beta wrapper logic moved out of `src/agents/pi-embedded-runner/extra-params.ts` into the Anthropic provider plugin, and the provider plugin still only applies `createAnthropicBetaHeadersWrapper(...)` when `resolveAnthropicBetas(...)` returns a non-empty list. Because the OAuth/default beta injection happens inside that wrapper, the reported `sk-ant-oat*` case without explicit Anthropic extra params is still not handled on `main`.
