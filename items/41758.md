## Summary

Current `main` does not implement this PR's automatic zombie-Chrome cleanup. Browser launch still fails fast on an occupied CDP port, and the current fallback is a manual `reset-profile` flow rather than pre-launch detection/cleanup. The proposed PR also still has unresolved correctness and safety issues in its only commit, so there is not strong evidence to close it as already implemented or obsolete.

## Evidence
