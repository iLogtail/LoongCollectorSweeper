## Summary

Keep open. Current `main` contains a recent fix for cached/snapshot memory capability state, but the wiki CLI path still appears able to run without an active memory capability registration, which matches the reported `bridgePublicArtifactCount: 0` false-negative. The issue is recent, repeatedly confirmed in discussion, and linked to still-open follow-up PRs/issues rather than a landed fix.

## Evidence
