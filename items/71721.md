## Summary

Keep open. Current main has guarded OpenClaw network-display/pairing helpers, but the issue’s reported stack matches the bundled Bonjour dependency path, and current main does not yet classify or suppress `uv_interface_addresses` failures from `@homebridge/ciao` as non-fatal.
