## Summary

Keep open. Current main does not implement native Slack token rotation: Slack config and runtime still model static bot/app/user/signing tokens, and there is no Slack oauth.v2 refresh flow, tokenRotation schema, refresh loop, or init command. This also should not be closed as ClawHub-only work because Slack is an existing bundled major channel and the requested behavior belongs in the Slack plugin/auth lifecycle, not just a standalone optional skill.
