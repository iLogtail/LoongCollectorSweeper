## Summary

Keep open. Current main still implements the reported fallback: when no per-agent heartbeat entries exist, heartbeat enablement falls back to the default agent instead of distinguishing between no heartbeat config and agents.defaults.heartbeat. The regression test still asserts the reported broken defaults-only behavior.
