## Summary

Keep open. Current `main` still rejects `sandbox.docker.binds` sources outside the runtime allowlist by default; the behavior the reporter described is still implemented. There is now an explicit break-glass override for external bind sources, but the sandbox docs still include arbitrary host-path examples while also describing allowed-root enforcement, so the original docs/runtime mismatch is not cleanly resolved.
