## Summary

Keep open. Current main fixed the explicit `openclaw gateway restart` path substantially, but the update refresh path still routes through `gateway install --force`, and LaunchAgent install still performs `launchctl bootout`/`unload` before `bootstrap`. That means the specific update/refresh failure mode in #40089 is not proven fixed on current `main`.
