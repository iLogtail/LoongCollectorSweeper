## Summary

Keep open. Current main still clears self-declared operator scopes for device-less shared-token/password WebSocket connects outside the narrow direct-local backend exception, and the auth regression tests explicitly assert the reported missing-scope behavior. The linked fix PRs are still open, so there is no merged current-main implementation that resolves #51396.
