## Summary

Keep open. Current `main` does not implement a Control UI workspace file-explorer toggle, and the existing file-related UI/API surfaces are limited to agent bootstrap/core files rather than a general workspace tree. The issue is recent, concrete, and there is not enough evidence to conservatively auto-close it as ClawHub-only.
