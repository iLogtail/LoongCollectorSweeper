## Summary

Keep open. Current `main` still does not implement the PR’s MS Teams implicit-mention fallback: the handler parses `conversation.id;messageid=` for thread routing, but mention gating still only checks `replyToId`, and sent-message caching still records only outbound reply IDs. That means the reported thread-reply case remains unaddressed on `main`.

## Evidence
