## Summary

Current `main` still exposes only manual `openclaw skills update` / `openclaw plugins update` flows, while the only built-in auto-update path is `update.auto` for the OpenClaw package itself. I found no plugin- or skill-level auto-update config or startup updater on `main`, so the feature request is not implemented and should remain open.

## Evidence
