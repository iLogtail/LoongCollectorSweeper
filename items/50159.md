## Summary

Keep open. Current `main` still does not validate value-mode `openclaw config set` paths against the runtime schema or require the path to already exist, so the headline fix is not implemented on `main`. The discussion also points to an open superseding PR (#56615), not a landed replacement, and #50159 itself carries unresolved review blockers and unrelated churn.
