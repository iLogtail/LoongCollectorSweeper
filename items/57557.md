## Summary

Keep open. Current `main` still synchronously reads and parses `subagents/runs.json` on read paths, and gateway `sessions.list` still reaches those subagent-registry helpers in hot code. The linked closed PR does not change the fact that the present codebase still shows the reported pattern, so this is not safely closable as implemented or obsolete.
