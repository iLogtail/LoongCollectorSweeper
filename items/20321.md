## Summary

Do not close #20321. Current main still does not implement `groupPolicy: "members"`: the shared type/schema and Telegram runtime only accept `open`, `disabled`, and `allowlist`. The related PRs in the provided GitHub context claimed to implement the feature, but they are closed and unmerged; the linked `openclaw-chat-only-trigger` plugin is a useful workaround, not the requested Telegram all-members allowlist policy.
