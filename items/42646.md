## Summary

Keep open. Current main has memory search/indexing, file-backed durable notes, a LanceDB memory plugin, and memory-wiki, but it does not implement the requested SQLite canonical durable record model with typed records, revision history, explicit statuses, validity windows, or normalized entity/link tables.
