## Summary

Keep #43410 open. Current main still uses SKILL.md frontmatter descriptions as prompt-facing skill-selection text, and several reported bundled skills still have terse descriptions without when-to-use trigger phrases. The canvas skill still lacks frontmatter/description, and related PRs are partial and unmerged rather than canonical replacements for the umbrella issue.

## Best Possible Solution

Keep the issue open as the umbrella. The best path is a focused skills/docs cleanup that adds prompt-useful trigger phrases to bundled skill descriptions, gives canvas valid frontmatter or documents its intentional exclusion, and splits or trims oversized skills such as gh-issues. If maintainers prefer smaller changes, link the partial PRs as batches while keeping this issue open until all remaining findings have canonical coverage.
