## Summary

Keep open. Current `main` still omits positional arguments from generated zsh leaf-command completions, so the underlying bug is still present. This PR also is not a safe auto-close candidate because its `_files` heuristic for every `<path>` positional would mis-handle existing non-filesystem commands such as `openclaw config get <path>` / `unset <path>`, so maintainer judgment is still needed.
