## Summary

Keep #33329 open. Current main has per-provider discovery toggles for Bedrock, GitHub Copilot, Hugging Face, and Ollama, and tests cover the non-Bedrock toggles. The issue is only partially satisfied because public docs still document the Bedrock discovery config path while omitting the GitHub Copilot, Hugging Face, and Ollama disable paths, and the requested global disable-all model discovery switch is still not present.

## Best Possible Solution

Keep the issue open until maintainers document the shipped GitHub Copilot, Hugging Face, and Ollama discovery toggles in the public provider/config docs, then either add a global disable-all model discovery control or split that later request into a focused follow-up with the intended config path and interaction with `models.mode`.
