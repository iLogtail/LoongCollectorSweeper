## Summary

Keep PR #64436 open. Current main still has the internal model-pricing helpers and internal callers, but it does not expose `runtime.usage`, `api.runtime.usage`, or an equivalent supported plugin SDK path. The PR remains a focused additive plugin runtime API proposal for context engine plugins such as ContextClaw, not a ClawHub-only plugin request or an obsolete duplicate.

## Best Possible Solution

Keep the PR open for maintainer review. The best path is to land a narrow additive `api.runtime.usage` capability, likely based on this PR, after confirming the public types are stable, documenting the namespace in `docs/plugins/sdk-runtime.md`, keeping the SDK API baseline aligned, and adding a focused runtime-shape test for `api.runtime.usage`.
