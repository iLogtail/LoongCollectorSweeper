## Summary

Keep open. Current `main` does not surface any global tool-profile row in `openclaw status` or `openclaw status --all`, and the status overview code was recently refactored into shared builders. The PR’s intent is still unshipped, but it likely needs rebasing onto the new shared status overview path and a maintainer decision on whether showing only the global baseline is acceptable given agent/provider overrides.
