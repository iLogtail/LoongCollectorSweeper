## Summary

Keep open. Current `main` still prefers the latest assistant text over tool-result stdout/stderr, and current tests explicitly assert that tool output should be omitted once an assistant completion exists. This PR is not implemented on `main`, and the discussion shows unresolved tradeoffs rather than obsolescence or incoherence.
