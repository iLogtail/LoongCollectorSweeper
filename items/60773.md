## Summary

Keep open. The intended docs improvement is still not implemented on current main: the Control UI docs still show raw `gatewayUrl=ws://...` and `gatewayUrl=wss://...` query examples. The PR is not mergeable as submitted because it deletes nearly the whole page, but the underlying narrow docs fix remains useful and unsuperseded.

## Best Possible Solution

Keep the item open for maintainer or author cleanup. The best path is a minimal docs-only change that restores the full `docs/web/control-ui.md` content, changes only the two dev-server `gatewayUrl` examples to percent-encoded `ws%3A%2F%2F...` / `wss%3A%2F%2F...` values, and adds one short note explaining that full WebSocket URLs in query parameters should be URL-encoded while auth tokens should stay in the `#token=` fragment.
