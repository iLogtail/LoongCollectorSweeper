## Summary

Keep open. Current `main` still has the unguarded `triggerInternalHook` implementation, the added re-entrancy tests from PR #55211 are not present, and the latest shipped release `v2026.4.22` contains the same hook files as current `main`. This PR still addresses a plausible unresolved bug rather than something already implemented or obsolete.
