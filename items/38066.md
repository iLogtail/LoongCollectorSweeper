## Summary

Keep #38066 open. Current main still does not expose a Telegram stream throttle config key or pass any configured throttle into `createTelegramDraftStream`; the draft stream still falls back to `DEFAULT_THROTTLE_MS = 1000`. Related PR #38255 appears to track the requested work, but it is open/unmerged, and current main has not implemented the issue yet.
