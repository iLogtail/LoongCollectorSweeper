## Summary

Current `main` does not include the PR's account-scoped gateway reload behavior. The reload planner still collapses matching channel config changes into full-channel restarts, and the hot-reload handler still restarts entire channels rather than specific accounts. This PR remains a real, unmerged behavior change.
