## Summary

Keep open. Current `main` already supports `/reset` for `session-memory`, but it still hard-codes writes to `<workspace>/memory` and does not expose an `outputDir` knob, so the feature request is not implemented. The PR also has an unresolved safety issue from review: its proposed early `mkdir` on a user-configured path can create directories outside the workspace through an in-workspace symlink before `writeFileWithinRoot` enforces the boundary.
