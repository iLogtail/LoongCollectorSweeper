## Summary

Current `main` already routes Synology Chat inbound messages with the account id, evaluates bindings on a per-channel/per-account basis, and has regression coverage for the exact multi-account same-user scenario from this report. The same implementation is present in the latest release commit `00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5` (`v2026.4.22`).
