## Summary

Keep open. Current main appears to address the specific legacy web-search migration path for old Brave config, but the broader requested behavior is still not implemented: plugin JSON-schema config errors are still collected as fatal validation issues, `loadConfig()` throws `INVALID_CONFIG`, and existing tests assert `res.ok === false` for invalid plugin config rather than warning and disabling only that plugin.
