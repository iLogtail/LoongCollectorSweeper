## Summary

Keep open. Current `main` does not appear to implement this PRŌĆÖs Telegram announce-target fix: `resolveAnnounceTargetFromKey` still falls back to `group:<chatId>` for Telegram group sessions, and the bundled Telegram channel plugin does not provide a session-target resolver that would convert that to a bare numeric chat ID.
