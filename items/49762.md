## Summary

Keep open. Current `main` does not implement response-duration text in the per-reply usage footer, and the proposed PR introduces contract questions that match the review feedback: it would change established footer formatting and would need explicit handling for `/usage tokens` versus `/usage full` before a conservative maintainer cleanup should close it.
