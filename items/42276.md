## Summary

Keep #42276 open. Current main has backend/session support for `reasoningLevel: "stream"` and emits `stream: "thinking"` agent events, but the TUI still does not expose `stream` in command completions/help and does not handle `thinking` agent events for live rewritten reasoning updates. The linked PR #42312 is closed unmerged in the provided GitHub context, so it does not supersede or implement the remaining request.
