## Summary

Current `main` still appears to exclude DuckDuckGo from the setup wizard. The setup filter requires `onboardingScopes` to include `"text-inference"`, but the DuckDuckGo web-search provider exported through both runtime and public-artifact setup surfaces still does not set that field.
