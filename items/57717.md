## Summary

Keep open. Current `main` does not implement Slack Block Kit table sending: Slack still renders outbound tables through the plain mrkdwn chunk path, global table-mode resolution still coerces `block` to `code`, and the user-facing docs still document only `code`/`bullets`/`off`. The PR discussion shows the author iterated on review feedback with targeted follow-up commits, so this still looks like live, useful work rather than something already superseded on `main`.
