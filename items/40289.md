## Summary

Keep open. The exact PR diff is stale because Discord code moved from `src/discord/**` into `extensions/discord/**`, but the underlying bug is still present on current `main`: Discord component interactions still dispatch only the button/select label or callback payload, with no stored or forwarded card text context.
