## Summary

Keep open. Current `main` has the underlying sandbox/session/tool primitives, but it does not have the proposed top-level `safetyPosture` config or the requested preset layer (`development` / `balanced` / `strict`, `full` / `limited` / `public`). This PR is not clearly implemented on `main`, and the existing manual controls are not a strong enough match to close as already done.
