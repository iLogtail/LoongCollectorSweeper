## Summary

Keep open. Current `main` still has no `migrate` or restore/import CLI surface, and the docs still tell users to migrate by manually copying the state directory/workspace. The PR discussion shows an explicit maintainer security objection to restore/import, so this is unresolved product/security judgment rather than an obsolete or already-implemented PR.
