## Summary

本 PR 为 LogFileReader 与 Go 侧协议转换器新增纳秒级时间戳支持，通过全局配置项 EnableTimestampNanosecond 控制精度输出。属于目标明确的核心功能增强，当前 main 分支尚未包含此实现，PR 状态开放且可合并，无重复或冲突迹象，应保留供维护者审查。

## Best Possible Solution

维护者需审查新增配置项在全局配置结构体与解析器中的定义是否完整；验证 C++ 侧 GetCurrentTimeInNanoSeconds 的跨平台时钟源兼容性与性能开销；确认 Go 侧将秒与纳秒合并为 uint64 的序列化格式是否符合下游消费端预期。CI 验证通过且代码审查无误后可合并，并同步更新配置说明文档。
