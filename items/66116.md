## Summary

Keep open. Current `main` still implements Gemini web search by calling Gemini `:generateContent` with `tools: [{ google_search: {} }]`, which is the exact request shape the report identifies as likely charging the wrong quota bucket. I found no code, docs, tests, or history evidence on `main` that proves OpenClaw now routes this through a different Google API path or validates the quota behavior, so there is not strong enough evidence to close conservatively.
