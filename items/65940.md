## Summary

Keep open. On the repo's referenced `origin/main` SHA `fe930b987e33f4e1acd93b5ef1ea76c8b7260047`, DuckDuckGo still lacks `onboardingScopes`, and the setup flow only shows providers whose metadata explicitly includes `"text-inference"`. This PR remains materially relevant; the related-work note pointing to open PR #70170 reinforces that the gap is still being worked, not already shipped.
