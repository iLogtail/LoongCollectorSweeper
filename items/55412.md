## Summary

Keep open. Current `main` no longer surfaces the old generic `Agent failed before reply` text for `GatewayDrainingError`, but it still does not auto-retry. The code explicitly classifies gateway-drain/lane-clear restart lifecycle errors, marks the reply operation failed, and returns a user-visible restart notice instead of waiting and retrying the run.
