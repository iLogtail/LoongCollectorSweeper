## Summary

Keep open. Current main clearly interprets `OPENCLAW_RAW_STREAM=0` or an unset variable as disabled, but the docs still only explain how to enable raw-stream logging, and I did not find a launchd-specific regression test or a post-report fix proving the reported "still appends after restart" behavior is resolved on `main`. That is not strong enough evidence for `implemented_on_main` or `cannot_reproduce`.
