Keep open. Current main still contains the exact clipping conditions reported for the Usage tab daily bar tooltip, and the related fix PR #59244 was closed unmerged rather than shipped or superseded by another implementation.

Add a small regression check if feasible, or attach manual before/after screenshots for the Usage tab daily chart.
