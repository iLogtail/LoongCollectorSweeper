## Summary

Keep open. Current main has heartbeat re-arm and requests-in-flight retry logic, but the issue specifically reports the failure after the #52270 re-arm fix, and the follow-up comment says it still happened on 2026.4.9. The only linked newer related item is macOS/App Nap-specific, while this report is Linux, so it is not a safe duplicate/superseded close.
