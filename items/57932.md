## Summary

Keep open. Current `main` still lacks the workspace fallback described in PR #57932, and the latest shipped release still has the pre-fix staging logic, so there is not strong evidence that this request is already implemented or obsolete.

## Evidence

- **Current staging code still has no sandbox-off workspace fallback:** On `main` at `90877e0`, `stageSandboxMedia()` only uses `sandbox?.workspaceDir ?? remoteMediaCacheDir` and returns early when both are absent. There is no `workspaceOnly` fallback to stage local inbound media into the agent workspace when sandbox mode is off.
