## Summary

Keep #39565 open. Current main still has the pre-PR TUI sanitizer behavior for punctuation-wrapped copy-sensitive tokens and dotted IDs without underscores, and the linked bug #39505 remains open. The PR is still a focused formatter/test fix rather than obsolete cleanup.

## Best Possible Solution

Keep #39565 or the linked #39505 work open. The best path is a narrow TUI formatter fix in `src/tui/tui-formatters.ts`: classify punctuation-stripped candidates, strip markdown asterisks at token edges, preserve dotted identifiers and digit-bearing hyphenated IDs, keep punctuation-only and natural-language hyphenated long tokens chunkable, and retain focused regression tests in `src/tui/tui-formatters.test.ts`.
