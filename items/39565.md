## Summary

Keep open. Current main still uses the old TUI formatter heuristic, so this PR’s remaining wrapped-token and dotted-no-underscore cases are not implemented. The linked bug #39505 is still open, and the PR is a focused candidate fix rather than obsolete cleanup.

## Best Possible Solution

Keep this PR or an equivalent patch under review until TUI sanitization preserves punctuation-wrapped copy-sensitive tokens and long dotted identifiers without reopening narrow-terminal overflow for punctuation-only or natural-language hyphenated tokens. The implementation likely belongs in `src/tui/tui-formatters.ts` with focused regression tests in `src/tui/tui-formatters.test.ts`, and #39505 should remain linked as the user-facing bug.
