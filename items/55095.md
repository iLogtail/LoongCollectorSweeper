## Summary

Keep open. Current `main` still intentionally clears operator scopes for device-less WebSocket clients, and the repo contains no evidence that the reported `missing scope: operator.write` path has been fixed or made obsolete. The report is sparse, but not incoherent, and closing it as fixed/cannot-reproduce would be too aggressive.
