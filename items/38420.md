## Summary

Keep #38420 open. Current main has addressed some audit items, including masked gateway token input and overview card work, but multiple concrete requests from the full Control UI audit remain in source. The narrower related #39780 does not supersede this full-dashboard UX tracker.

## Best Possible Solution

Continue tracking #39780 as the narrower config apply/hardening item, and implement or split the remaining full-dashboard work: Instances per-row actions and reason/chip affordances, Sessions key copy/truncation and less repetitive override controls, clearer Config action semantics, operator shortcuts/help, and broader feedback/activity UX.
