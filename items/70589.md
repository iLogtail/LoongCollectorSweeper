## Summary

Do not auto-close this PR. Current main still has the reported local `agent --local --json` stdout/stderr bug: the local delivery path serializes the JSON envelope through `runtime.log()`, while `--json` routes console logs to stderr. The PR is a focused fix for open issue #62440, and the related #62453 was closed unmerged, so it does not supersede this work.

## Best Possible Solution

Keep #70589 open for maintainer review. The best product/code path is to replace the local delivery `runtime.log(JSON.stringify(...))` call with `writeRuntimeJson(runtime, buildOutboundResultEnvelope(...))`, keep the envelope shape unchanged, and add the focused regression test asserting local JSON mode writes through `runtime.writeJson()`. Once that lands, close #62440; #62453 is closed unmerged and should not be treated as the canonical fix.
