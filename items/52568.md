## Summary

Keep open. Current main still lacks the requested Discord middle-ground behavior: bot-owned autoThread threads bypass mention gating entirely, and Discord channel config does not expose a `requireMentionInThreadReplies` or equivalent option. The related #40895 context is relevant, but it is broader/ambiguous rather than a clear canonical replacement for this focused feature request.

## Evidence

  - file: [extensions/discord/src/monitor.test.ts:502](https://github.com/openclaw/openclaw/blob/8e12c24d17218dc50023aaf5543df1b7c36520aa/extensions/discord/src/monitor.test.ts#L502)
