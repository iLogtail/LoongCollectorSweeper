## Summary

Keep open. Current source and the published `v2026.4.23` npm tarball do not show the exact reported `LEGACY_CONFIG_RULES is undefined` symbol path, and current packaging checks now verify installed binary version drift, but the report targets a Bun-transpiled subprocess path (`B:/~BUN/root/entry.js`) that I did not reproduce live. With an independent corroborating comment and no direct proof that the Bun/subprocess failure is gone on current main, there is not enough evidence for a conservative close.
