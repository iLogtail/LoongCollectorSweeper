## Summary

Current `main` already fixes the proxy-dispatcher regression for Codex runs with a stronger implementation than this PR: it bootstraps an `EnvHttpProxyAgent` before timeout tuning, then preserves that proxy-aware dispatcher when applying global Undici stream timeouts.
