## Summary

Keep open. Current main still forces Telegram DM preview streaming through the message/edit transport in the dispatch path, so the reported regression is not fixed on main. The related PRs #62207 and #62210 correctly identified the likely fix direction but were closed unmerged, and current tests still assert the forced message transport behavior.
