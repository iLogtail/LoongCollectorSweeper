## Summary

Current `main` does not implement this PR’s code-file memory indexing or declaration-aware chunking. Memory indexing still accepts Markdown plus multimodal files, `buildFileEntry()` still classifies text files only as `markdown`, and indexing still chunks text with `chunkMarkdown()` only. The PR remains a live, plausible feature change rather than cleanup material.
