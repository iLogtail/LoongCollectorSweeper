## Summary

Current main has the generic cron stagger machinery, but the memory-core dreaming configuration does not expose `dreaming.staggerMs` or pass it into the managed dreaming cron schedule. The linked implementation PR #69548 is still open/unmerged, so the issue remains a valid tracker until that PR or an equivalent fix lands.

## Best Possible Solution

Keep the issue open and use PR #69548, or an equivalent maintainer-approved patch, to add `dreaming.staggerMs` to the memory-core plugin schema, host config resolution, `ShortTermPromotionDreamingConfig`, managed cron job construction, docs, and focused tests. After that lands on main, close this issue as implemented.
