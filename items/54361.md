## Summary

Keep open. Current main still ships `skills/canvas/SKILL.md` without YAML frontmatter, and the skill loader skips skills that lack a frontmatter-derived `description`. PR #54361 is a narrow patch that directly addresses the still-live defect reported in #54310, and the provided review comment supports that it matches the existing bundled-skill frontmatter pattern.

## Best Possible Solution

Keep #54361 open for maintainer review, or supersede it with an equivalent main-branch fix that adds valid `name: canvas` and trigger-rich `description` frontmatter to `skills/canvas/SKILL.md`. A small regression test that covers canvas, or all bundled skills, would prevent this omission from recurring.
