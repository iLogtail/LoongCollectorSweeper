## Summary

Keep PR #70459 open. Current main still lacks the proposed Step 0 prerequisite-check guidance, so the PR is not obsolete or implemented elsewhere. It is also a focused docs improvement, but the draft should not be closed because the remaining work is actionable: fix the shell/PowerShell snippets flagged in review and scope the installer auto-install wording to the actual installer behavior.

## Best Possible Solution

Keep the draft PR open and ask for a small revision: replace the Bash and PowerShell one-liners with robust command-existence/version checks, scope the auto-install claim to the actual Node/Git and downloader behavior in scripts/install.sh and scripts/install.ps1, keep links root-relative per docs/AGENTS.md, and then preview the Mintlify rendering for docs/start/getting-started.md.
