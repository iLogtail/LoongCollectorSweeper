## Summary

Keep open. Current main still has the reported schema/implementation mismatch: the nodes tool schema makes `node` optional, but the `describe` branch reads `node` with `{ required: true }`, which still throws `node required` before default-node resolution can run. The linked PR #51926 tracks a plausible fix but is open/unmerged, so the issue still represents a current main bug.
