## Summary

Current `main` still does not persist timeout-classified model failures into auth-profile cooldown state, so the cross-run circuit-breaker needed for this starvation report is not wired up. The fallback loop can skip/probe cooldowned providers, but only after a provider is already marked unavailable; timeout paths explicitly avoid doing that.

## Evidence

- **Timeouts are intentionally not recorded as profile failures:** `maybeMarkAuthProfileFailure` returns early for `reason === "timeout"`, and `resolveAuthProfileFailureReason` explicitly maps timeout to `null` with a comment saying timeout failures should not persist auth-profile failure state.
  - file: [src/agents/pi-embedded-runner/run.ts:633](https://github.com/openclaw/openclaw/blob/90877e0d4274c53fd91bf5787265c4d6889dc9a7/src/agents/pi-embedded-runner/run.ts#L633)
  - command: `nl -ba src/agents/pi-embedded-runner/run.ts | sed -n '632,672p'`
  - sha: [90877e0d4274](https://github.com/openclaw/openclaw/commit/90877e0d4274c53fd91bf5787265c4d6889dc9a7)
- **Model fallback skip/probe logic only runs for providers already in cooldown:** `runWithModelFallback` only enters the skip/probe path when `profileIds.length > 0 && !isAnyProfileAvailable`, i.e. after auth state already says every profile is in cooldown. Without the timeout being recorded, a new run starts from the original primary candidate again.
  - file: [src/agents/model-fallback.ts:760](https://github.com/openclaw/openclaw/blob/90877e0d4274c53fd91bf5787265c4d6889dc9a7/src/agents/model-fallback.ts#L760)
  - command: `nl -ba src/agents/model-fallback.ts | sed -n '760,860p'`
  - sha: [90877e0d4274](https://github.com/openclaw/openclaw/commit/90877e0d4274c53fd91bf5787265c4d6889dc9a7)
- **Tests cover timeout cooldown behavior only with pre-seeded cooldown state:** The timeout fallback tests construct an auth store already containing a timeout cooldown, then verify later candidates are tried. That proves the cooldown mechanism exists, but not that real timeout failures from embedded runs populate it.
  - file: [src/agents/model-fallback.test.ts:1683](https://github.com/openclaw/openclaw/blob/90877e0d4274c53fd91bf5787265c4d6889dc9a7/src/agents/model-fallback.test.ts#L1683)
