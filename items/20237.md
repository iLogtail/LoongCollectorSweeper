## Summary

Keep #20237 open. Current main still has partial coverage only: cron list/edit/run/history, delete-after-run in the WebUI, browser Web Push, an inline high-context notice with Compact, and automatic pre-compaction memory flush. The central requested feature remains missing: a native in-app toast/popup notification layer, cron-result popups with action buttons, and snooze/reschedule controls are still absent from the Control UI and cron RPC surface.

## Best Possible Solution

Keep this open until the missing Control UI notification work is implemented or explicitly descoped. The likely product path is a native in-app toast/popup layer, cron run-completion events wired into actionable result notifications, snooze/reschedule controls backed by `cron.update` or a dedicated validated RPC, and a product decision on whether context warnings should remain inline only or also produce toast-style Flush/Save prompts alongside existing Compact and automatic memory-flush behavior.
