## Summary

Keep #20237 open. Current main has partial coverage: Control UI cron editing/run history/delete-after-run, browser Web Push, Gateway cron finished events, and an inline high-context notice with a Compact action. The central requested work is still missing: a native in-app toast/popup notification layer, actionable cron result popups, and snooze/reschedule controls. The existing maintainer comment also explicitly keeps the issue open for these gaps, and the relevant files have not changed since that review.

## Best Possible Solution

Keep #20237 open until the missing Control UI notification work is implemented or explicitly descoped. A good path is a native in-app toast/popup layer, cron finished events wired into actionable result notifications, snooze/reschedule controls backed by validated cron.update schedule patches or a dedicated RPC, and a product decision on whether context warnings should remain inline-only or also trigger toast-style Compact/Flush prompts.
