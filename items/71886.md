## Summary

Keep open. Current main still serves the Control UI CSP with `connect-src 'self' ws: wss:` and no `https://tweakcn.com`, while the shipped browser-side custom theme importer normalizes tweakcn links to `https://tweakcn.com/r/themes/{id}` and fetches that URL directly. The PR is a narrow, still-relevant fix with a matching regression assertion rather than an obsolete or superseded change.

## Best Possible Solution

Do not close this PR through cleanup. The best maintainer path is to review and land this narrow CSP allowlist change, or apply the same `https://tweakcn.com` `connect-src` update with a regression test, then run the targeted gateway CSP/HTTP test lane before merge.
