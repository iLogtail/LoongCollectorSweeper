## Summary

Keep #22438 open. Current main has partial related work, but it still lacks the requested schema-backed `agents.defaults.bootstrapTier` policy and `minimal | standard | full` tier resolver. The active successor PR #22439 still tracks that remaining work.

## Best Possible Solution

Keep #22438 open until maintainers either merge #22439 or a successor that adds a schema-backed `agents.defaults.bootstrapTier` design, or explicitly decide that the current fixed allowlist plus per-run `lightContext` is the intended product direction. A complete path should thread the tier through `src/agents/workspace.ts`, `src/agents/bootstrap-files.ts`, config schema/types/help/labels/generated schema, public config docs, bootstrap hook behavior, and focused tests.
