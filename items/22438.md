## Summary

Current `main` partially addresses the motivation behind #22438, but it does not implement the requested configurable `agents.defaults.bootstrapTier` feature. Subagent/cron bootstrap already includes persona files and a lightweight mode exists, yet the exact tiered config is absent and the linked implementation PR #22439 is still open and unmerged.
