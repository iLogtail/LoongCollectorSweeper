## Summary

Keep open. PR #43366 added the base `autoThreadName: "generated"` Discord behavior, but current `main` still has no `threadPromptGuideline` config field or prompt override path. The generated-title flow uses a hardcoded English system prompt and forwards only message/channel context into title generation, so the requested customization capability remains missing.
