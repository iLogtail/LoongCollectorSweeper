## Summary

Keep open. Current `main` does not contain the Playwright dialog-race non-fatal handling from PR #40067, and the associated regression test is also absent. A `Protocol error (Page.handleJavaScriptDialog): No dialog is showing` rejection would still fall through to the generic fatal unhandled-rejection exit path on `main`.
