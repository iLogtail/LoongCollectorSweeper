## Summary

Current `main` already self-heals auto-persisted failover model overrides at the start of the next turn, so a session retries the configured primary model once it is available again. The fix lives in `src/auto-reply/reply/model-selection.ts:173` and is covered by regression tests in `src/auto-reply/reply/model-selection.test.ts:635` and `src/auto-reply/reply/model-selection.test.ts:693`.
