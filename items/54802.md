## Summary

Keep open. Current `main` still lacks the PR's Codex Responses alignment in both affected call sites: conversation-label generation still embeds the instruction text into the user message and always sends `temperature`, and TTS summarization does the same. The transport layer still derives OpenAI Responses instructions from `systemPrompt`, so the PR's change is not already implemented on `main`.
