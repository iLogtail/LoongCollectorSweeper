## Summary

Keep open. Current main still persists `models auth login` OAuth results into the resolved local agent auth store and has no `gateway.remote`/node-host guard or gateway-sync path in the login command. The related PR #42381 is useful context but remains unmerged; #42407 was closed unmerged, so the reported behavior is not implemented on main.
