## Summary

Keep open. Current `main` still does not forward most `tools.exec` policy into `/bash` chat sessions, so the behavior gap described by PR #50392 remains. The proposed patch is directionally correct, but the review discussion identified a remaining omission (`approvalRunningNoticeMs`), so this is not a strong conservative-close candidate.
