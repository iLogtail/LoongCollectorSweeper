## Summary

Keep open. Current `main` still contains the same macOS gateway keepalive `sendPing()` continuation path that matches the reported crash stack, I found no landed fix or regression test for that path, and the menu bar icon still depends on the app surviving long enough to install the `MenuBarExtra` status item.
