## Summary

Keep open. Current `main` still drops Feishu card-action fields outside `action.value`/`tag`, so the requested pass-through for `form_value`, `input_value`, and select metadata is not implemented. The PR also targets an older card-action flow and would need a rebase against the current structured interaction path before it could be evaluated for merge.
