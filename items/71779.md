Keep PR #71779 open. Current main still has the gateway/session-row mismatch this PR targets: session defaults and rows expose model/provider-derived thinking defaults and option lists, while runtime reply paths already honor configured `agents.list[].thinkingDefault` / `agents.defaults.thinkingDefault`.
