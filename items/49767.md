## Summary

Current `main` still has the unconditional Feishu `@_all` mention path, so the reported bug is not implemented on `main`. The PR also has unresolved correctness concerns from its own review thread, especially preserving legacy single-bot behavior and handling rich-text `post` messages whose mentions are parsed from content rather than the top-level `mentions` array. Conservative maintainer cleanup should leave this PR open.

## Evidence
