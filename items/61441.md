## Summary

Keep PR #61441 open. Current main has part of the light-context fix for normal embedded agent runs, but it still does not carry the lightweight bootstrap mode/run kind through the command system-prompt estimate path or direct compaction bootstrap resolution, which are explicit parts of this PR's stated scope and Greptile review summary.

## Best Possible Solution

Keep the PR open, but ask for a rebase onto current main and a narrower patch that preserves the already-shipped normal run forwarding while adding the missing lightweight context propagation for command system-prompt estimates and compaction/runtime-context paths, with focused regression tests for those remaining seams. Drop or split the unrelated NO_REPLY cron delivery changes.
