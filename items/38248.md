## Summary

Keep open. Current main has cost estimation, usage aggregation, provider spend-limit error classification, and external proxy guidance, but no native `agents.defaults.costGuard` or request-level per-hour/day spend ceiling. The issue is concrete and still describes a meaningful core failover safety gap, not just optional plugin work.
