## Summary

Current main already sends Brave API keys via `X-Subscription-Token` for both Brave web and llm-context requests, and the linked bug is closed, but the specific regression tests proposed by PR #69704 are not present on main. This is a focused test-only PR tied to a real closed regression, so it still contains useful missing coverage and should stay open for maintainer review rather than be auto-closed.

## Best Possible Solution

Keep PR #69704 open and either land or refresh its focused regression tests in `extensions/brave/src/brave-web-search-provider.test.ts`, since current main fixes the Brave auth behavior but still lacks explicit coverage preventing a return to query-param API key auth.
