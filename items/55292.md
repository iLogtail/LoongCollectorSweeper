## Summary

Current main already contains the completion-routing pieces needed for Telegram ACP topic bindings: task-completion announce delivery resolves bound destinations, external Telegram delivery context is preserved instead of being overwritten by internal webchat/session traffic, and there is an explicit regression test for Telegram forum-topic completion delivery.
