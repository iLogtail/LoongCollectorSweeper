## Summary

Keep PR #72137 open. Current main still has the old gateway config diff behavior: missing `plugins.entries.<id>.config` compared with an injected empty object is reported as a changed path, and `plugins.*` changed paths still require a gateway restart. The linked bug #72061 was closed as superseded by this PR, not because the fix shipped on main.

## Best Possible Solution

Keep PR #72137 open for maintainer review and normal landing. The best path is to merge or revise the narrow `diffConfigPaths` fix with regression coverage that collapses missing versus empty plain-object phantom diffs, preserves explicit empty-array diffs, and treats #72061 as unresolved until this PR or an equivalent main change ships.

## Evidence
