## Summary

Keep open. Current `main` does not include this provenance feature: `EmbeddedContextFile` still has only `path` and `content`, bootstrap context assembly does not attach provenance metadata, and system-prompt context rendering does not emit `<!-- ctx:provenance ... -->` tags or related guidance. The PR remains substantive rather than already-implemented or obsolete.
