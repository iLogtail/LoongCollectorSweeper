## Summary

Current `main` does not implement this PR’s GA migration. The Anthropic plugin still adds the legacy `context-1m-2025-08-07` beta for `context1m: true`, still strips that beta for OAuth tokens, and its tests explicitly assert the pre-GA behavior. The PR also targets an old core file path, while the active behavior now lives in the bundled Anthropic plugin, so the underlying idea is still relevant but would need to be rebased onto the current architecture rather than auto-closed.
