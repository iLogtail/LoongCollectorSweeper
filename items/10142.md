## Summary

Keep open. Current main still does not implement or document the requested internal `session:end` hook event. OpenClaw has adjacent plugin lifecycle hooks like `session_end`, plus reset-related internal command hooks, but those are not the internal `session:end` surface requested for HOOK.md-style handlers and workflow orchestration.

## Best Possible Solution

Keep #10142 open until maintainers either implement the internal `session:end` contract or explicitly document that plugin `session_end` / `agent_end` is the supported replacement. A complete implementation should add the internal event context/type guard, emit `createInternalHookEvent("session", "end", ...)` from the relevant completion paths, add regression tests, and update the public hooks docs at `https://docs.openclaw.ai/automation/hooks`.
