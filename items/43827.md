## Summary

Keep open. Current `main` still has the Chat UI strings hard-coded in English, so the i18n request is not implemented. The PR also appears to need rework because its proposed `t(..., { fallback: ... })` pattern does not match the current translation helper semantics, and `ui/AGENTS.md` now requires generated locale workflow rather than hand-editing non-English locale files.
