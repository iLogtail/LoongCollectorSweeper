## Summary

Keep open. Current main still routes the local `agent --local --json` result envelope through `runtime.log()`, while JSON mode routes console/log output to stderr. The gateway JSON path already uses the structured stdout writer, and the related fix PRs #62453 and #70589 are open but not merged, so the issue remains the canonical bug report until a fix lands.

## Best Possible Solution

Keep #62440 open as the canonical bug until a focused fix lands. The best code path is the one proposed by #70589: change the local delivery JSON branch in `src/agents/command/delivery.ts` to call `writeRuntimeJson(runtime, buildOutboundResultEnvelope(...))` or an equivalent structured stdout writer, then update local-agent JSON tests to assert `writeJson`/stdout behavior while preserving gateway, envelope shape, delivery, and non-JSON behavior.
