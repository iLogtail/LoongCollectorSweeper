## Summary

Keep open. Current `main` still appends the workspace `HEARTBEAT.md` hint to heartbeat prompts whenever the prompt text mentions `HEARTBEAT.md`, and there is no guard for user-configured custom prompts. That means the reported regression is still present, and the PR is addressing a real mismatch between code and the documented/verbatim custom-prompt contract.
