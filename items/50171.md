## Summary

Keep open. Current `main` has already moved Discord approval gating into the shared native-approval path, but the gateway still derives the turn source from `request.channel` (the delivery override), so the cross-channel routing bug this PR is trying to fix is still plausibly present. The branch itself looks stale against current architecture rather than obsolete.
