## Summary

Keep open. Current `main` still matches the reported behavior: the Debug snapshot refresh path returns immediately when disconnected, and the Snapshots card still has no inline loading or disconnected-state UI beyond the button label.

## Evidence

- **Disconnected refresh still silently returns:** `loadDebug()` exits immediately when `!state.client || !state.connected`, with no error state or user-facing message set before returning.
