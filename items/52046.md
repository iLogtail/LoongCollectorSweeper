## Summary

Keep open. Current main has heartbeat backpressure and efficiency mitigations, but it does not implement the requested non-blocking heartbeat execution. Heartbeat turns still run in-process through the normal reply path and occupy the same session/global command lanes as user turns; the wake layer defers or retries when lanes are busy rather than running heartbeats independently.
