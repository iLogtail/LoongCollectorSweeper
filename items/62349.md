## Summary

Keep open. Current main still caches each subsystem file child logger indefinitely, and there is no generation/invalidation mechanism in src/logging. The PR also has an unresolved substantive review concern that its proposed generation check may not refresh the root logger before comparing generations, so this needs maintainer review or a revised fix rather than cleanup closure.
