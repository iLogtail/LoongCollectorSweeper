## Summary

Keep open. Current main has the existing `extraPaths` support and missing-path diagnostics, but the requested cleanup is not implemented: there is still no `excludePaths` or glob-exclusion config surface, the builtin indexer still walks all eligible files under extra paths, and the docs do not yet cover path hygiene or benchmark/queryset contamination pitfalls.
