## Summary

Keep open. Current main does not contain a hardcoded `wangtao` path, but the root issue described in the discussion remains: default workspace setup paths are still resolved to absolute host paths and persisted into config. The related PR #56810 explicitly targeted this bug but is closed unmerged, and I did not find an implemented equivalent on main.
