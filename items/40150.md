## Summary

Keep open. Current `main` still lacks the PR's core user-visible behaviors: webchat does not append inline run-error messages, gateway WebSocket connections do not fail fast on legacy query-auth URLs, OpenRouter does not apply Mistral `strict9` replay policy for proxied `mistralai/*` models, and embedded-run payload assembly still strips downgraded tool-call text instead of surfacing it as an explicit synthetic failure. Some adjacent work landed separately, but not this PR's substantive fixes.
