## Summary

Current `main` does not implement the PR’s core fixes. The agent payload path still does not detect or surface fake tool runs/placeholder replies, web chat error events still only update hidden state instead of appending an inline assistant message, and websocket connections still start the normal challenge flow and rely on handshake timeout rather than immediately rejecting legacy query-auth clients.
