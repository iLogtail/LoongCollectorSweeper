## Summary

Keep open. Current main still has subagent lifecycle plugin hooks, but not the requested post-subagent-completion, pre-announcement hook that can inspect completion output, observe delivery status, and mutate/suppress/enrich the completion announcement. A maintainer comment on 2026-04-26 also explicitly kept #22358 open for this exact gap.

## Best Possible Solution

Keep #22358 open until OpenClaw adds a public plugin hook at the post-completion/pre-announcement point, with access to the finished child run output and delivery result/phases plus explicit announcement mutation, suppression, and fallback semantics; alternatively, a maintainer should fold that exact scope into a canonical lifecycle/delivery design issue.
