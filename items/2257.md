## Summary

本 Issue 提议新增基于 C++ 的原生 Kafka Flusher 插件。当前 main 分支已提供 kafka_v2 实现，但社区仍有对原生 C++ 版本在特定压缩算法（如 Zstd）下性能优化的明确诉求。讨论区已提出基准测试对比建议，属于有效的核心插件扩展提议。尚无关联 PR 或替代工单，符合保守保留原则，建议保持开启以待贡献者补充性能数据与设计方案后提交 PR。

## Best Possible Solution

保持开启。建议贡献者补充与现有 kafka_v2 插件在 Zstd 压缩等场景下的 CPU/内存/吞吐基准测试数据，并提供 C++ Kafka 客户端选型（如 librdkafka）与 V2 Pipeline 集成设计草案。待维护者确认核心 API 兼容性与主仓库集成方案后，可提交 PR 进入代码审查流程。
