## Summary

Keep open. Current `main` still does not implement per-agent suspend/resume: there are no `openclaw agents disable|enable` commands, no `agents.list[].enabled` config/schema/docs surface, and routing/cron/heartbeat still treat configured agents as active. The PR discussion is useful evidence: Greptile identified real data-loss/runtime gaps, and the author says they fixed them in follow-up commit `41d788f`, but those changes are not present on `main` at `c070509b7f8fdbe479e5d829643782c272e30e83`.
