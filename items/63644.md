## Summary

Keep this PR open. Current main still does not contain the viewport-height/global-scrollbar changes, and the active agent chat textarea still computes below the 16px iOS Safari auto-zoom threshold. The later #64651/#64673 thread covers the input zoom part, but not the full chat viewport/global scrollbar fix described here.

## Best Possible Solution

Keep this PR open for maintainer review instead of auto-closing it. The best path is to coordinate it with #64651/#64673: either land a combined fix that keeps the 16px input approach for zoom while separately evaluating the `100dvh`/`.shell--chat` viewport changes, or ask the author to rebase and narrow the PR to the still-uncovered iPhone Safari chat layout scrollbar problem.
