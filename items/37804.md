## Summary

Keep open. Current main has strong partial fixes for the issue family, especially runtime fallback resolution and `models list --all` visibility for configured provider models, but the broader RFC is not fully implemented. The requested shared canonical resolution result across status/list/runtime/capability surfaces still appears incomplete, particularly in `models status`, which emits resolved labels and auth state rather than consuming the runtime resolved model descriptor with provenance/capabilities.
