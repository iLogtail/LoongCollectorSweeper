## Summary

Keep #37804 open. Current main has substantial partial alignment for config-injected and forward-compatible models, but the RFC’s central invariant is still not implemented as one shared resolved model descriptor consumed by status, list, runtime, thinking/capability gates, and docs. The issue discussion explicitly framed this as phased work, and the shipped code still has separate resolution paths.

## Best Possible Solution

Keep #37804 open as the umbrella design/product tracker until maintainers either narrow the accepted invariant or land a shared resolved model descriptor consumed by models status, models list, runtime resolution, capability/thinking policy, and docs. If the team wants smaller work, split the remaining status/provenance and capability alignment into canonical follow-up issues, then close this RFC only once those are tracked or implemented.
