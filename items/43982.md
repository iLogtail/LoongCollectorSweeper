## Summary

Keep open. Current `main` does not include the PR's gateway chat-history sanitization for inline image payloads beyond the existing top-level `image.data` case, and the added regression coverage is also absent.

## Evidence

There is no current-main handling for image blocks shaped like `content[].source.data`, top-level `content[].url` data URLs, or `image_url.url` data URLs.
