## Risks / Open Questions

- Closing now would overstate the fix: the report spans upgraded installs, explicit `tools.profile` state, installed plugin allowlists, and non-GPT-5/small-model behavior, while current fixes only cover parts of that matrix.
- The reporter specifically described upgrade-era failures; current main preserves explicit existing `tools.profile` values, so an old restrictive profile can still explain continued tool refusal on upgraded setups.
- Some newer protections only prevent loops or plan-only stalls after a tool is unavailable; they do not prove that AgentMail/GitHub/custom plugin action tools now execute correctly in the reported environments.
