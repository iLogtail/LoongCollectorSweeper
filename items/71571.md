## Summary

Keep open. Current `main` is written to resolve routed-agent defaults through agent-aware helpers, and the reporter’s cited gateway log line comes from startup logging that only prints the global configured model. That weakens the reported root-cause analysis, but there is still no high-confidence end-to-end proof here that bound channel session creation on current `main` cannot mis-resolve the model, so this is not safe to auto-close.
