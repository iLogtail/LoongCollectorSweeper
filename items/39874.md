## Summary

Current `main` still contains the configured-main route inheritance behavior that this PR removes, and the regression tests on `main` explicitly assert that CLI and missing-client-metadata callers on configured-main sessions inherit external delivery routes. That means the request is not already implemented and the PR should stay open for maintainer review rather than be closed as stale or redundant.
