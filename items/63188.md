## Summary

Keep open. Current `main` does not implement the requested per-agent `onLengthTruncation` / `maxContinuations` behavior, and the existing recovery logic only covers narrow reasoning-only or zero-visible-output cases. A truncated run that already emitted partial visible text can still pass through without an explicit truncation-specific recovery path.
