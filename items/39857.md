## Summary

Keep open. Current `main` still lets an override win over a known smaller model window, and that value is still used for status reporting and persisted session metadata.

## Evidence

- **Core resolver still returns the override before checking model limits:** `resolveContextTokensForModel()` returns `contextTokensOverride` immediately when it is positive, so a larger override is not capped by a known provider/model window.
  - file: [src/agents/context.ts:426](https://github.com/openclaw/openclaw/blob/b98a6a46fb54145a33e753101c02f0d7d7e0bc3b/src/agents/context.ts#L426)

  - file: [src/commands/status.summary.ts:164](https://github.com/openclaw/openclaw/blob/b98a6a46fb54145a33e753101c02f0d7d7e0bc3b/src/commands/status.summary.ts#L164)
