## Summary

Keep open. Current `main` does not implement this feature: `openclaw logs` still tails only the gateway log, the `logs.tail` RPC has no source selector, and docs still describe gateway-only behavior. The PR also has unresolved review findings in the discussion, including an invalid-config regression on a diagnostic command path and CLI output correctness issues, so there is not strong evidence for a conservative close.
