## Summary

Keep open. Current `main` still reproduces the underlying `bind=lan` display-URL mismatch in multiple user-facing surfaces, so this is not implemented on `main`. The PR branch is also stale as a focused fix: it misses newer/current callsites and now bundles unrelated agent-error and skill changes that need separate review.
