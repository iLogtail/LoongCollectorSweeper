## Summary

Current `main` does not implement this PR’s policy-routing change. Inline skill `command-dispatch: tool` still builds tools through `createOpenClawTools()` and then applies only the owner-only filter, while normal coding sessions go through `createOpenClawCodingTools()` with the broader tool-policy pipeline and explicit plugin allowlist. The PR remains relevant and should stay open.
