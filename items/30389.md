## Summary

Current main only partially satisfies #30389. The Control UI/WebChat composer now has a discoverable attachment button and a session model picker, but uploads are still image-only end to end and there is no visible composer skill/capability picker for the requested finance/coding/writing/image/reasoning-style routing.

## Best Possible Solution

Keep #30389 open as an umbrella WebChat UX request, or split it into narrower maintainer-owned issues. The remaining work should add non-image document/file upload through the picker, preview/validation, chat.send serialization, Gateway parsing/offload/extraction, model/provider handling, and docs; separately define a visible composer skill/capability picker that maps cleanly to skills, tools, agents, or task/model profiles instead of relying only on typed slash commands.
