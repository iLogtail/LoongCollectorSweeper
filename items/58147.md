## Summary

Keep open. Current `main` still formats Control UI/WebChat timestamps with direct browser-locale `toLocaleTimeString`/`toLocaleString` calls, while `agents.defaults.timeFormat` is only documented and wired for prompt/current-time generation, not UI session displays.
