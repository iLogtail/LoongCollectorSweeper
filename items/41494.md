## Summary

Current main has several partial defenses for tagged reasoning, final-tag enforcement, and Gemini reasoning-only retry, but it still does not cover the broad reported leak shape: Gemini reasoning/planning text delivered as ordinary channel text, including multilingual `Reasoning:`-style preambles. The broader sanitizer candidate (#60551) and the final-tag fragment fix (#48593) are both still open and unmerged, so #41494 should remain open as the active umbrella tracker.

## Best Possible Solution

Keep #41494 open as the umbrella for Gemini plain-text reasoning leakage. The best path is a narrow, tested fix in the shared assistant-visible delivery sanitizer and/or Gemini provider/stream path that covers untagged planning text, appended reasoning blocks, and multilingual reasoning headers without stripping normal prose. #48593 can remain the tag-fragment fix, and #60551 can be evaluated as a candidate broader fix only after it rebases, links #41494 explicitly, and carries false-positive regression coverage.
