## Summary

Current main does not provide enough coverage to close #41494. Gemini is wired as a tagged-reasoning provider and final-tag enforcement exists, but the remaining reported leak shapes are still plausible: outbound delivery only suppresses payloads already marked `isReasoning`, the message tool only strips XML-like reasoning tags from tool arguments, and the streaming final-tag scanner still uses a restrictive `<final>` regex. The issue discussion also has recent reports through v2026.4.15 and points to still-open follow-up work (#60551), so this should remain open for maintainer/product handling rather than be auto-closed.
