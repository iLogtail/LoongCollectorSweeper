## Summary

Keep open. Current `main` does not ship the proposed `agents.defaults.replyMode` / `tool-only` behavior: the generic reply path still auto-delivers block and final assistant text, the ACP path still projects/falls back to assistant text, and the config/docs/changelog on `main` contain no `replyMode` surface. The PR discussion also shows substantive follow-up work on review findings and downstream backport interest, so this is not a strong auto-close candidate.
