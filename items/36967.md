## Summary

Keep open. Current `main` still does not wire Feishu into the shared `/allowlist`, markdown-capability, and envelope-stripping paths this PR targets. Some parts of the PR are now obsolete on `main` because hook channel validation is already open-ended and security fixes are now generic across configured channels.

## Evidence
