## Summary

Keep open. Current `main` still lacks the workspace-cwd validation and workspace-specific doctor warning this PR proposes, so the change is not implemented on `main`. The PR appears to address a real gap, but the review discussion also surfaced a maintainability concern about string-prefix coupling, so this is not a conservative auto-close candidate.
