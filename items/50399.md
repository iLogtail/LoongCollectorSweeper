## Summary

Current `main` has absorbed some of the groundwork for this Feishu fix, but it still does not actually pass a computed `duration` from `sendMediaFeishu` into `uploadFileFeishu`, so the reported behavior is not clearly fixed on `main`. The PR is also outdated against today's `extensions/feishu/src/media.ts` rewrite and the latest maintainer review says it is not landable as-is, so this should stay open for a smaller rebased follow-up rather than be auto-closed.
