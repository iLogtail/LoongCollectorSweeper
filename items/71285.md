## Summary

Keep #71285 open. Current main still lacks a `memorySearch.ignorePaths` config surface, the builtin memory indexer recursively scans `memory/**/*.md` with no directory exclusion for `memory/dreaming/`, and both builtin/QMD watcher ignore lists omit `dreaming`. Related work fixed promotion/self-ingestion paths, but not the search-index and watcher behavior requested here.

## Best Possible Solution

Keep the issue open and implement a first-class `memorySearch.ignorePaths` path: add schema/types/help/docs, resolve it per agent, and apply the same normalized matcher to builtin `listMemoryFiles`, builtin watcher ignores, QMD collection/watch handling, and tests proving `memory/dreaming/**` is searchable by default but excluded from indexing and watcher activity when configured.
