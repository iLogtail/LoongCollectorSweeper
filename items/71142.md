## Summary

Current `main` does not implement this request. The Control UI sends image attachments through `chat.send`, and the gateway still hardcodes a `5_000_000` byte attachment limit there. The public config schema for `gateway.controlUi` exposes hosting/auth settings only and does not offer any upload-size or per-media limit field, so the requested configurability is still missing.
