## Summary

Current `main` already contains a broader fix for the isolated-heartbeat key/transcript-path bugs described here, with regression tests covering the original report and the review-thread edge cases, so this PR can be closed as superseded by implementation on `main`.

  - file: [src/infra/heartbeat-runner.ts:571](https://github.com/openclaw/openclaw/blob/a43c1f880757e830b5fb4d3c7b5532a625f0e3e4/src/infra/heartbeat-runner.ts#L571)
