## Summary

Keep open. Current `main` still does not ship Xiaomi video media understanding, so this PR is not implemented-on-main. At the same time, `main` now already covers Xiaomi image understanding indirectly through the generic image-capable config-provider path, so the PR is partially obsolete and needs rebase/update rather than conservative auto-close.
