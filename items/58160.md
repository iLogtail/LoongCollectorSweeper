## Summary

Keep open. Current `main` at `90877e0d4274c53fd91bf5787265c4d6889dc9a7` still has the late-output race described in PR #58160: backgrounded sessions are snapshotted once on exit, later `appendOutput()` calls only update the live session object, and `process log` for finished sessions reads the frozen snapshot.
