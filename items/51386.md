## Summary

Keep open. Current main still treats session transcript indexing as experimental and opt-in, so the central request to graduate `sessionMemory` and make it discoverable/default behavior is not implemented. The related closed PR #51341 proposed this default flip but was not merged, and the later related items track adjacent memory continuity work rather than superseding this graduation request.
