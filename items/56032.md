## Summary

Current main already carries the subagent announce delivery fix: spawned-session delivery context is seeded from the originating channel/to/threadId, completion announces merge missing route fields back from the stored origin, and Telegram-specific regression tests cover the previously broken path. The same implementation is present in release `v2026.4.22`.

## Evidence
