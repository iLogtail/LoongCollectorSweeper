## Summary

Keep open. Current `main` has partial recovery for stuck channels and typing cleanup, but it does not implement the core request: there is no general plugin failure circuit breaker/auto-disable path, no typing-TTL-triggered session restart, and no generic hard timeout wrapper around plugin tool execution.
