## Summary

Current main supports normal xAI Grok 4.20 reasoning and non-reasoning variants, but it deliberately does not support the Grok 4.20 multi-agent model requested in #38204. The docs and tests both preserve that as an unsupported xAI API surface, and the issue discussion includes follow-up requests for native multi-agent endpoint support, so meaningful requested work remains.
