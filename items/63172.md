## Summary

Close as out-of-scope for OpenClaw core: current main documents WeChat as an external plugin, explicitly says context tokens and account monitoring are owned by that plugin, and the repo contains no WeChat runtime code beyond docs. The report is coherent, but it targets `@tencent-weixin/openclaw-weixin` rather than `openclaw/openclaw`.

  - file: [docs/channels/index.md:45](https://github.com/openclaw/openclaw/blob/cf858258c71701ef62dddf88aa777873a4962046/docs/channels/index.md#L45)

  - file: [VISION.md:54](https://github.com/openclaw/openclaw/blob/cf858258c71701ef62dddf88aa777873a4962046/VISION.md#L54)

That matches the external plugin/runtime boundary rather than a core implementation gap.
