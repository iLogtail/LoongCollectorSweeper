## Summary

Current `main` still returns the channel-tool payloads in `structuredContent` while `content` is only a terse summary or status string, and the docs still advertise these tools for generic MCP clients. That matches the reported gap rather than proving it obsolete or fixed, so this issue is not a safe close.
