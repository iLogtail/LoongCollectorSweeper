## Summary

Keep open. Current main does not implement Feishu Task API support: the bundled Feishu plugin registers doc, chat, wiki, drive, permission, bitable, and scope-diagnostic tool surfaces, but no task tool/config/API surface. Although the repo vision pushes optional integrations toward plugins and ClawHub, this request targets an existing bundled/community-maintained Feishu plugin and mirrors related #53327, which was kept open rather than redirected. No canonical duplicate, shipped task plugin, or maintainer signal was present in the provided context.
