## Summary

Keep #42196 open. Current main still allows `openclaw cron add` without `--agent`, silently sends `agentId: undefined`, and has no source-backed warning for the default-agent fallback. The linked PR #42245 appears to be the right remaining work, but it is open/unmerged in the provided GitHub context, so the user-visible request is not implemented on main yet.
