## Summary

Keep open. Current `main` still uses a shallow `mergeSessionEntry` path, the proposed `merge-plugin-metadata` policy is not implemented, and OpenClaw’s own vision says plugin-API extension requests belong in core design discussion when plugins cannot safely do it themselves. This is not strong enough for any auto-close bucket.
