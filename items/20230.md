## Summary

Keep #20230 open. Current main has shipped active assistant display names and user-vs-assistant bubble styling, but the requested per-agent configurable bubble colors and persisted per-message/per-agent assistant attribution are still missing. The existing issue discussion also already kept this item open for that remaining scope.

## Best Possible Solution

Keep the issue open until maintainers either implement or explicitly reject per-agent chat styling and assistant attribution. A complete path likely belongs in Control UI plus the shared chat/session metadata contract: persist assistant agent identity per message or group, add validated optional styling fields or render hooks, render assistant-specific CSS variables/classes, and add focused tests/docs for mixed-agent and historical-session attribution.
