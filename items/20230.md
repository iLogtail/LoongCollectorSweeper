## Summary

Keep #20230 open. Current main resolves and displays the active assistant name and has different user-vs-assistant bubble styling, but it still lacks the requested per-agent configurable bubble colors and per-message/per-agent assistant attribution for mixed or historical chat views. The only issue comment also kept this open for the same remaining scope.

## Best Possible Solution

Keep the issue open until maintainers either implement or explicitly reject a real per-agent chat styling/attribution surface. A complete fix likely needs persisted message/group metadata for assistant agent identity, a validated config/protocol field for optional per-agent chat styling, Control UI rendering/CSS hooks, and focused tests/docs for mixed-agent and historical-session attribution.
