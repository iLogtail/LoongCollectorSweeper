## Summary

Current main does not appear to implement the requested Feishu outbound @mention normalization. The Feishu plugin still has separate text/card mention formatters and applies card-style mentions only when `mentionTargets` are supplied structurally; manually written `<at user_id="...">...</at>` tags in agent reply text flow into post/card/streaming card content unchanged.
