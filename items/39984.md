## Summary

Keep open. Current main still does not add the requested `web_fetch`/`browser` fallback text to missing-key `web_search` errors, so this is not `implemented_on_main`. But the PR's original edit target is now obsolete after the web-search provider split, so the patch needs a refresh rather than an auto-close.
