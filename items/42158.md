## Summary

Keep #42158 open. Current main still cancels the compaction when the LLM summarization path throws before any successful summary, which is the stuck-session retry loop described by the issue. The related proposed fixes (#42182 and #42520) were closed unmerged in the provided GitHub context, and I found no merged replacement in current main.
