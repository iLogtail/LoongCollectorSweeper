## Summary

Keep open. Current main still contains the exact reported fallback: when `openclaw-sandbox:bookworm-slim` is missing, `ensureDockerImage()` pulls `debian:bookworm-slim` and tags it as the default sandbox image without checking or building the repo/user Dockerfile. The repo still has a `Dockerfile.sandbox` and setup script that build a richer image with `python3`, `jq`, and `ripgrep`, so the reported custom/default image replacement path remains actionable.
