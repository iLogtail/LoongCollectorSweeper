## Summary

Keep open. Current main still passes the Bonjour service instance name to @homebridge/ciao without enforcing the 63-byte DNS label limit, so the reported long-hostname crash is not fixed on main. The related PRs in the issue discussion were closed unmerged and do not supersede the issue.
