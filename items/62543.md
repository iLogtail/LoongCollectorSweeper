## Summary

Keep open. The PR targets a real docs drift, but its one-line deletion overcorrects: AWS simplified Bedrock access for commercial serverless models, while Anthropic first-time-use, Marketplace prerequisites, and GovCloud manual access cases still exist. Current main still has the broad caveat, so the best outcome is a scoped docs rewrite rather than closing or merging this PR unchanged.

## Best Possible Solution

Revise the Bedrock notes instead of deleting the line: say commercial serverless models are generally auto-enabled with the right IAM/Marketplace permissions, then explicitly call out Anthropic first-time-use form, Marketplace subscription prerequisites, and GovCloud manual model-access requirements. Link the public Bedrock docs page from the OpenClaw docs update.
