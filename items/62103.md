## Summary

Keep open. Current main does not implement the PR's requested Vertex AI routing: google-vertex is still only a hook alias on the Google AI Studio provider, the Google plugin manifest does not declare google-vertex as an owned provider or auth choice, and no Vertex region/base URL helper exists in extensions/google. The linked issue #60736 remains open and this PR still contains potentially useful implementation work, so it should not be auto-closed as superseded.
