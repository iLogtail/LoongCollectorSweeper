## Summary

Keep open. Current `main` still watches skill refresh paths via glob patterns like `<root>/*/SKILL.md`, while the repo now depends on `chokidar` `^5.0.0`; the proposed fix is not implemented on `main`, and the existing test only verifies those glob targets rather than nested refresh behavior.
