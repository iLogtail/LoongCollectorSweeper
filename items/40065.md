## Summary

Keep open. Current `main` still appears to have the reported bug: the Web UI's default-agent model editor is labeled as editing the default, but its save path still routes through `agents.list[index].model` instead of `agents.defaults.model`.

## Evidence

- **Default-agent UI still uses the default-specific label:** The Agents overview renders the selector label as `Primary model (default)` when the selected agent is the default agent, and changing that selector calls the shared `onModelChange(agent.id, ...)` handler.
  - file: [ui/src/ui/views/agents-panels-overview.ts:142](https://github.com/openclaw/openclaw/blob/f00d65a30496335fb068f01fe529dbba484d4312/ui/src/ui/views/agents-panels-overview.ts#L142)
  - command: `nl -ba ui/src/ui/views/agents-panels-overview.ts | sed -n '1,180p'`
  - sha: [f00d65a30496](https://github.com/openclaw/openclaw/commit/f00d65a30496335fb068f01fe529dbba484d4312)
- **Model form entry resolver still targets per-agent override path:** `resolveAgentModelFormEntry()` unconditionally returns `basePath: ["agents", "list", index, "model"]`, so the form write target is the per-agent override path, not `agents.defaults.model`.
  - file: [ui/src/ui/app-render.ts:774](https://github.com/openclaw/openclaw/blob/f00d65a30496335fb068f01fe529dbba484d4312/ui/src/ui/app-render.ts#L774)
  - command: `nl -ba ui/src/ui/app-render.ts | sed -n '760,815p'`
  - sha: [f00d65a30496](https://github.com/openclaw/openclaw/commit/f00d65a30496335fb068f01fe529dbba484d4312)
- **Save handler writes through the per-agent path:** `onModelChange` calls `resolveAgentModelFormEntry(index)` and then writes or removes that `basePath`, which means default-agent edits still persist under `agents.list[index].model`.
  - file: [ui/src/ui/app-render.ts:2026](https://github.com/openclaw/openclaw/blob/f00d65a30496335fb068f01fe529dbba484d4312/ui/src/ui/app-render.ts#L2026)

  - sha: [f00d65a30496](https://github.com/openclaw/openclaw/commit/f00d65a30496335fb068f01fe529dbba484d4312)

  - file: [ui/src/ui/views/agents-panels-overview.ts:68](https://github.com/openclaw/openclaw/blob/f00d65a30496335fb068f01fe529dbba484d4312/ui/src/ui/views/agents-panels-overview.ts#L68)
