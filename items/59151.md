## Summary

Keep #59151 open. Current main partially addresses the report by treating human/PluralKit `@everyone`/`@here` payloads as mentions, but the requested role-mention behavior remains missing: Discord `mentionedRoles` is collected only as a generic mention/ignore-other-mentions signal, not checked against the bot or routed agent membership as an explicit mention.
