## Summary

Keep #70882 open. Current main does not implement the PR's schema-aware coercion before bundle MCP tool calls, and the linked #70872 report is a concrete current product bug with a mcporter workaround showing the expected object-shaped params behavior. The PR is not ready to close as implemented or obsolete; it needs maintainer review and hardening rather than cleanup closure.

## Best Possible Solution

Keep the PR or linked bug open and route it through maintainer review. The likely fix belongs in the bundle MCP materialization/runtime boundary with bounded, prototype-safe, schema-aware coercion plus regression tests for object, array, already-correct values, invalid JSON, and nullable/array-form schema types.
