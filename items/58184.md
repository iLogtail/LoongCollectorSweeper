## Summary

Keep open. Current `main` still serves `connect-src 'self' ws: wss:` for the Control UI, so the hardening is not implemented. At the same time, current docs, code, and tests still support cross-origin `gatewayUrl` targets including `ws://...`, which means the PR needs maintainer judgment on how to tighten CSP without regressing documented remote-gateway workflows.
