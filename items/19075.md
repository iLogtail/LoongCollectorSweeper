## Summary

Keep open. Current main does not implement a Brave Search baseUrl override: the Brave provider still constructs both Brave request URLs from hardcoded api.search.brave.com endpoint constants, and the Brave plugin config schema exposes only webSearch.apiKey and webSearch.mode. The provided discussion also shows prior PRs for this feature were closed and a newer PR (#45843) is still open, so the requested capability remains missing on main.
