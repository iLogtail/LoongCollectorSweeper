## Summary

Keep open. Current `main` still omits the OpenAI-compatible `max_tokens`/`max_output_tokens` fields unless `options.maxTokens` is explicitly set, so the requested fix is not implemented on `main`, and the PR also carries unrelated TUI changes with unresolved review concerns.
