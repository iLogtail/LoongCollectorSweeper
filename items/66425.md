## Summary

Close as ClawHub/plugin work. PR #66425 is a self-contained third-party AstraFlow model-provider plugin with no missing core API identified. Current main already documents external provider plugins, install/publish through ClawHub or npm, and the provider SDK covers API-key auth, provider registration, model catalogs, and onboarding helpers needed by this PR.

## Best Possible Solution

Close the core PR and ask the contributor to publish the AstraFlow provider as an external ClawHub/npm plugin using the existing provider-plugin SDK. Revisit OpenClaw core only if the external plugin path exposes a concrete missing provider/onboarding API seam.
