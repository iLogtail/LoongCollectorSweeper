## Summary

Current `main` already routes configured Anthropic Claude 4.6 models to adaptive thinking by default, and the Anthropic transport/test coverage confirm the reported `budget_tokens: 1024` fallback is no longer used for the Claude Opus 4.6 path described in #63016.
