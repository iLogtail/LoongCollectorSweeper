## Summary

Current `main` already contains multiple targeted guards for repeated unknown-tool/tool-not-found loops, including runtime stream rewriting, loop-detection logic for repeated unavailable tools, regression tests for the exact failure mode, and config-reload handling to invalidate stale skill snapshots that could keep advertising removed tools.
