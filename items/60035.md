## Summary

Keep #60035 open. Current main has useful partial proxy support, but it does not yet implement the broad first-class system/custom proxy request: there is no top-level proxy config, no per-agent proxy override, web_fetch intentionally does not honor HTTP_PROXY for untrusted URLs, browser proxying requires explicit Chrome flags instead of system/env proxy auto-use, and model-provider proxy config is documented as HTTP-only.
