## Summary

Keep open. Current `main` still contains the narrow `<final>` regex in the streaming and user-facing sanitization paths, so there is not strong evidence that issue #65867 is fully fixed on `main` or in the latest release.

That regex is used by `stripBlockTags` for streamed assistant output.
