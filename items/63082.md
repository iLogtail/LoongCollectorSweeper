## Summary

Keep open. Current `main` still hardcodes shell completion profile paths and doctor hints instead of respecting `ZDOTDIR`/`XDG_CONFIG_HOME`, so the reported bug is still present. The PR is directionally useful, and the review discussion also shows unresolved edge cases that need maintainer review rather than auto-closing.

## Evidence

- **Current profile resolution ignores shell-specific env vars:** `getShellProfilePath()` on current `main` returns `~/.zshrc`, `~/.bashrc`, `~/.config/fish/config.fish`, and the default PowerShell path directly from `HOME`/`USERPROFILE`; it does not consult `ZDOTDIR` or `XDG_CONFIG_HOME`.
