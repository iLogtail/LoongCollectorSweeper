## Summary

Keep open. Current `main` does not include the proposed save-time session path canonicalization, and the PR as submitted still has an unresolved correctness concern: it uses `resolveSessionFilePath`, which intentionally preserves some cross-root `.../agents/<id>/sessions/<file>` absolute paths instead of forcing them into the active store directory.
