## Summary

Keep open. Current `main` still routes default-agent model edits through `agents.list[index].model` instead of `agents.defaults.model`, while the UI labels that control as the default model and the docs/schema define the default model under `agents.defaults.model`. The PR addresses a real current bug and is not already implemented on `main`.
