## Summary

Keep open. Current `main` still does not implement the PR's two main behaviors: `feishu_chat` has no `list` action, and it still binds to the first enabled Feishu account instead of routing per agent/account. The proposed work also matches existing Feishu core patterns already used by other bundled tools, so this is not a ClawHub-scope item.
