## Summary

Keep open. Current `main` only overlaps part of this PR: the Control UI device-auth bypass and loopback pairing fallback were reworked on `main`, but several proposed changes in this bundle are still absent or materially different, so closing as `implemented_on_main` would be inaccurate.
