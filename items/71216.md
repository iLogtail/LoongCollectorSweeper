## Summary

Keep open. Current `main` does not implement the requested config surface, and the request is still fresh and actionable. The repo already documents adjacent behavior (`agents.defaults.sandbox.mode`, separate-profile multi-gateway setup, multi-agent routing), which makes this a legitimate design/feature request rather than an obsolete or incoherent report.
