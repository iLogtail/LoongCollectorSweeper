Current main still does not add the requested Advanced Features / Next Steps bridge to `docs/tools/creating-skills.md`; the guide remains a starter workflow plus a small metadata table and related links. The full skills reference and current runtime/tests confirm the omitted features are real supported authoring capabilities, while the exact docs fix is only represented by open, unmerged PR #59650 in the provided GitHub context.

## Best Possible Solution
