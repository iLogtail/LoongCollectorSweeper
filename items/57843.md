## Summary

Keep open. Current `main` does not implement this PR's core behavior: outbound delivery still returns only `OutboundDeliveryResult[]`, cron retry/caching still cannot distinguish partial-send `DeliveryError` cases, gateway send still turns empty delivery results into `UNAVAILABLE`, and restart-sentinel still treats hook-cancelled sends as retryable/non-acked. The PR discussion also explicitly narrowed scope on April 22, 2026 to preserve these fixes while deferring only adjacent follow-up work.
