## Summary

Keep open. Current `main` still appears to leak inline `<think>...</think>` content for assistant string payloads in the TUI, so the underlying bug is not obsolete. This PR also has an unresolved regression risk already called out in review: its string-path change would strip literal `<think>` examples from non-assistant user/command messages.
