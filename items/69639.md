## Summary

PR #69639 adds local sub-agent research/reading notes as top-level `.agents/*.md` files. Current repo conventions keep local `.agents` material out of the committed tree and use `docs/` for shared documentation; the submitted docs are also Korean-only while this repo maintains English docs as the source of truth. Current `main` also confirms review-comment accuracy blockers in the proposed guides, including the real `src/plugins/registry.ts` and `openclaw onboard` entrypoint.

## Best Possible Solution

Close this PR as a local-agent research artifact rather than a source-repo docs change. If maintainers want shared architecture onboarding, the right path is a new focused English docs PR under `docs/concepts/` or `docs/reference/`, with current-main file references corrected and public docs navigation updated as needed.
