## Summary

Keep open. Current `main` still ignores live `session.message` payloads while a chat run is active, so the UI half of this PR is not implemented on `main`, and the ACP half has an unresolved P1 review note about `git diff --stat HEAD` missing untracked or unborn-HEAD changes.
