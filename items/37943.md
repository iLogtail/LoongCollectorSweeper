## Summary

Keep open. The report is clear enough and not protected, but current main does not provide high-confidence evidence for any close reason: the node service still runs through `openclaw node run`, and the node host still reads the main config with `loadConfig()`. The only related PR (#38017) was a closed, unmerged draft that added issue-bootstrap metadata only.
