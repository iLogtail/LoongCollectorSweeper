## Summary

Keep open. Current main still has the reported broken export HTML placeholder shape: the template contains formatter-expanded MARKED_JS/HIGHLIGHT_JS/JS blocks, while generateHtml() still performs exact contiguous-token replacements. The linked PR #41861 remains relevant but is not merged, and the closed/unmerged markdown-export PR #51105 does not fix the /export-session HTML path.
