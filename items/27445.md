## Summary

Keep #27445 open. Current main has partial sub-agent completion improvements, but it still does not expose the requested `announceTarget` config default or `sessions_spawn` per-call override. Top-level requester completions still use external delivery when the requester is dormant, while only active requester sessions and nested subagents get parent/session-internal handling.

## Best Possible Solution

Keep #27445 open as a core subagent announce policy request. A complete maintainer path would add a public `announceTarget` or broader announce policy to `sessions_spawn`, `agents.defaults.subagents`, config schema/help/docs, and tests covering top-level parent-only routing, current channel delivery, active requester steering, nested subagent delivery, and interaction with #33478 callback/thread hardening.
