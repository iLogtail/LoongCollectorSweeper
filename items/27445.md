## Summary

Keep #27445 open. Current main still does not expose the requested `announceTarget` config default or native `sessions_spawn` per-call override. The code has adjacent improvements for active requester steering, nested subagent routing, ACP `streamTo:"parent"`, and delivery fallbacks, but dormant top-level native subagent completions still use external channel delivery by design.

## Best Possible Solution

Keep #27445 open as a core subagent announce policy request. The likely maintainer path is to add a public parent/channel announce policy to native `sessions_spawn` and `agents.defaults.subagents`, update config schema/help/generated docs and https://docs.openclaw.ai/tools/subagents, and cover top-level parent-only routing, current channel delivery, active requester steering, nested subagent delivery, and interactions with #33478 callback/thread hardening.
