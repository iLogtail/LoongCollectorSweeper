## Summary

Keep #28965 open. Current main has useful targeted model normalization and single-session override clearing, but it still does not provide the requested `openclaw models reconcile` command or bulk `openclaw sessions model-reset` workflow. The related #41736 is broader and open, but it does not clearly supersede the specific reconcile/reset proposal.

## Best Possible Solution

Keep this issue open until OpenClaw either implements first-class core CLI support for model config reconciliation plus bulk session model override reset, or a maintainer explicitly marks a canonical issue for the same concrete work. The likely implementation belongs in the core CLI and config/session-store layers, with docs updates for `docs/cli/models.md`, `docs/cli/sessions.md`, and the model-selection/failover concepts.
