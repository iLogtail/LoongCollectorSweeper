## Summary

Keep open. Current `main` has partial reasoning-tag stripping for assistant text extraction, but it does not implement this PR's reply-normalization and gateway streaming protections, so the change is not already on `main` and still targets a plausible gap.

This explains why part of the PR overlaps with newer mainline code, but only for extraction paths that use this helper.
