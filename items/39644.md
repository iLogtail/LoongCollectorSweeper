## Summary

Keep PR #39644 open. Current main still has the Windows PowerShell completion install bug and the time-format PowerShell resolver issue described by the PR. The PR discussion also surfaced a valid reload-hint fix, and the contributor added a follow-up commit for it; the useful path is to rebase or manually port the patch to the current `completion-runtime.ts` layout.

## Best Possible Solution

Keep this PR open and port the useful changes to current main: add a PowerShell branch in `src/cli/completion-runtime.ts`, emit PowerShell dot-source syntax and a correct reload hint, switch `src/agents/date-time.ts` to `resolvePowerShellPath()` with non-interactive flags, and add focused regression tests for PowerShell install/idempotence plus the Windows time-format command path.
