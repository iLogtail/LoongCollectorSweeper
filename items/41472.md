## Summary

Keep open. Current main does not show a high-confidence core fix for the reported Tailscale Serve WebSocket drop, nor does it implement the suggested dual loopback+tailnet bind. Main does have a BlueBubbles plugin-owned webhook auth path, which may address part of the workaround when the webhook includes the BlueBubbles password, but that does not fully close the reported iOS/Tailscale Serve compatibility issue.
