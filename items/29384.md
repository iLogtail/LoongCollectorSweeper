## Summary

Keep #29384 open. Current main still does not expose Docker sandbox GPU, IPC, raw Docker create args, or equivalent fields for `--gpus all` / `--ipc=host`. Related work is adjacent only: OpenShell has a separate `gpu` backend option, and #53108 is an open capAdd-only PR that would not satisfy Docker GPU/IPC support.

## Best Possible Solution

Keep the issue open until a maintainer-approved Docker sandbox config extension lands. The best path is a constrained explicit contract such as `agents.defaults.sandbox.docker.gpus` plus a carefully reviewed `ipc` policy or dangerous override, not broad raw-args passthrough by default. Update the config type, strict Zod schema, resolver/hash behavior, runtime validation/audit messaging, Docker create-arg builder, generated schema/help/docs, and focused regression tests.
