## Summary

Keep open. Current `main` does not implement this PR’s hook-context length cap: plugin hook prompt fields are still merged and injected without any `MAX_HOOK_CONTEXT_LENGTH`-style clamp in the hook runner, shared prompt-build helper, embedded runner, or CLI runner. The existing tests cover merge order and legacy compatibility, not oversized-context truncation.
