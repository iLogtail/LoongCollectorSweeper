Keep #16896 open. A complete fix should add an accessible message-level Reply action or context menu in the Control UI chat surface, store the selected reply target or quoted text in composer state, focus the input, and send a designed quote representation or additive `chat.send` reply-target field with protocol, rendering, docs, and UI/controller tests updated together.
