## Summary

PR #69385 adds a new optional bundled AJ Router plugin. Current OpenClaw scope and docs direct optional plugins to external ClawHub/npm distribution, and current main already exposes the plugin hooks, config, command, and ClawHub install surfaces needed to build this outside core. Existing PR review comments also identify correctness issues that should be fixed before publishing externally.

## Best Possible Solution

Close this core PR as ClawHub scope. The AJ Router should live as an externally maintained ClawHub/npm plugin using the existing hook, command, config, and install surfaces; if that implementation reveals a missing core API, open a focused SDK proposal for that seam instead of bundling the whole router in core.
