## Summary

Current `main` does not contain the proposed stable setup-surface export. `openclaw configure` is still interactive-only, there is no `configure surface` command or related export module on disk, and the provider auth contract on `main` still exposes wizard metadata but not the serializable `surface` metadata this PR adds.

## Evidence
