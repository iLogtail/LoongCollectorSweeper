## Summary

本 PR 修复了 metric_system_v2 插件中 fd_max 与 fd_allocated 指标因 float64 转换导致的大数精度丢失问题，改为字符串上报。该变更针对真实缺陷，当前尚未合入 main 分支，且基线较旧，需维护者进一步审查与合并。

## Best Possible Solution

维护者应审查本 PR 的代码变更与 CI 状态，确认下游消费端对字符串类型指标值的兼容性。若方案可行，请指导作者 rebase 至最新 main 分支以解决潜在冲突后合并；或在评审中讨论作者提出的替代方案（直接移除恒定的 fd_max 指标）以简化实现。
