## Summary

本 PR 修复了 metric_system_v2 插件中 fd_max 与 fd_allocated 指标因 float64 转换导致的大数精度丢失问题，提议改为字符串上报。该缺陷真实存在，但指标类型变更（float64 转 string）属于破坏性更新，需评估下游监控系统兼容性。此外 PR 基线较旧，需 Rebase 至最新 main。符合保守维护策略，建议保持开启等待维护者兼容性裁决与作者更新。

## Best Possible Solution

维护者需审查指标类型变更的下游兼容性。若确认字符串上报可接受，请指导作者 Rebase 至最新 main 分支并解决冲突后合并；若下游强依赖数值类型，可评估作者提出的替代方案（直接移除恒定的 fd_max 指标）或采用 int64 上报方案。需维护者介入决策后推进。
