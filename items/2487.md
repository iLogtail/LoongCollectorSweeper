## Summary

本 PR 修复了 metric_system_v2 插件中 fd_max 与 fd_allocated 指标因 float64 精度丢失导致大整数显示异常的问题，提议改为 string 类型上报。由于指标类型变更属于下游破坏性更新，且作者提出了改用 int64 或直接移除恒定 fd_max 等替代方案，需维护者进行产品兼容性裁决。此外 PR 基线较旧需 Rebase。建议保持开启等待维护者评估技术选型与作者同步最新代码。

## Best Possible Solution

建议维护者评估指标类型变更的下游兼容性影响。若下游强依赖数值类型，可指导作者改用 int64/uint64 上报以保留精度，或采纳移除恒定 fd_max 指标的建议；若确认改为 string 可接受，请作者 Rebase 至最新 main 分支，补充或更新相关单元测试与文档后合并。当前保持开启等待维护者技术选型裁决与作者 Rebase。
