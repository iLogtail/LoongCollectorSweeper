## Summary

本 PR 提议将 metric_system_v2 插件的 fd_allocated 与 fd_max 指标上报类型由 float64 改为 string，以修复大整数精度丢失问题。该变更涉及下游监控消费端的类型兼容性，且作者正文中已提出替代方案（改用 uint64 或直接移除恒定 fd_max）。需维护者进行技术裁决与兼容性评估，并同步 Rebase 至最新 main 分支后合并。

## Best Possible Solution

维护者需评估 float64 转 string 对下游的兼容性影响。若确认方向，请作者 Rebase 至最新 main 并补充或更新单元测试后合并；若倾向替代方案（如 uint64 或移除恒定 fd_max），请指导作者调整实现。当前保持开启等待技术裁决与代码同步。
