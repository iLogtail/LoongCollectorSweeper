## Summary

本 PR 修复了 metric_system_v2 插件中 fd 指标因 float64 转换导致的大整数精度丢失问题，改为使用字符串上报。经核对当前 main 分支，该修复尚未合入，且无其他替代实现或重复 PR，属于有效贡献，应保持开启供维护者审查。

## Best Possible Solution

建议保持开启并等待维护者审查。需重点验证下游指标消费端对字符串类型指标值的兼容性。若确认 helper.NewMetricLogStringVal 接口符合规范且无破坏性变更，可在 CI 通过后合并；或根据 PR 正文建议评估是否直接移除恒定的 fd_max 指标以简化逻辑。
