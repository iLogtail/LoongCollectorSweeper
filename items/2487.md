## Summary

本 PR 修复了 metric_system_v2 插件中 fd_max 与 fd_allocated 指标因 float64 转换导致的大整数精度丢失问题。经核对当前 main 分支，该修复尚未合入，PR 改动最小化且利用了已有的字符串指标接口，属于有效缺陷修复，应保持开启供维护者审查与合并。

## Best Possible Solution

建议保持开启并等待维护者审查。需重点验证下游指标消费端对字符串类型指标值的兼容性。若确认 helper.NewMetricLogStringVal 接口符合规范且无破坏性变更，可在 CI 通过后合并；或根据 PR 正文建议评估是否直接移除恒定的 fd_max 指标以简化逻辑。
