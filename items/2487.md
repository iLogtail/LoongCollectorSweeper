## Summary

本 PR 修复了 metric_system_v2 插件中 fd_max 和 fd_allocated 指标因转换为 float64 导致的精度丢失问题，改为使用字符串上报。改动合理且最小化，当前处于开放待审状态，需维护者评估下游消费端对字符串类型指标值的兼容性后决定是否合并。

## Best Possible Solution

建议维护者审查本 PR 的 CI 结果与下游消费端对字符串类型指标值的兼容性。若确认 helper.NewMetricLogStringVal 接口符合规范且无破坏性变更，可予以合并；或根据 PR 正文建议评估是否直接移除恒定的 fd_max 指标。等待 Maintainer Review 与测试验证。
