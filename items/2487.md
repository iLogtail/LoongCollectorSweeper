## Summary

本 PR 修复了 metric_system_v2 插件中 fd 指标因 float64 精度限制导致的大整数失真问题，方案为将指标类型改为 string。该变更属于指标数据类型的 Breaking Change，可能影响下游监控系统的数值解析与聚合逻辑。当前需维护者对技术方向（改为 string、改用 uint64 或直接移除恒定 fd_max）进行兼容性评估与裁决，且 PR 基线较旧需 Rebase 至最新 main。鉴于其为真实缺陷修复且具备明确实现路径，保守评估应保持开启等待维护者决策与作者同步。

## Best Possible Solution

建议维护者评估将 fd 指标类型改为 string 的下游兼容性影响。若方向确认，请作者 Rebase 至最新 main 分支解决潜在冲突，并补充或更新相关单元测试后合并。若维护者倾向于直接移除恒定的 fd_max 指标或改用 uint64 处理以维持数值类型，请指导作者相应调整实现。当前保持开启等待技术裁决与代码同步。
