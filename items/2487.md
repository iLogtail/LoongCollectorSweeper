## Summary

本 PR 修复了 metric_system_v2 插件中 fd_max 和 fd_allocated 指标因 float64 转换导致的精度丢失问题，改为字符串上报。PR 代码变更合理，针对真实缺陷，目前处于开放待审状态，应保持开启等待维护者审查与 CI 验证。

## Best Possible Solution

建议维护者审查本 PR 的实现。若确认字符串上报符合下游消费预期且 helper.NewMetricLogStringVal 接口可用，可合并；或根据 PR 正文建议评估是否直接移除恒定的 fd_max 指标。等待 CI 通过及 Maintainer Review。
