## Summary

本 PR 修复了 metric_system_v2 插件中 fd_max 与 fd_allocated 指标因 float64 转换导致的精度丢失问题，改为通过新增的 addMetricStringVal 方法以字符串形式上报。变更针对真实缺陷，逻辑清晰且改动最小化，当前处于开放待审状态，符合合并条件，建议保持开启等待维护者审查。

## Best Possible Solution

建议维护者审查本 PR 的实现细节与 CI 结果。若确认下游消费端兼容字符串类型的指标值，且 helper.NewMetricLogStringVal 接口符合规范，可予以合并；或根据 PR 正文建议评估是否直接移除恒定的 fd_max 指标。等待 Maintainer Review 与测试验证。
