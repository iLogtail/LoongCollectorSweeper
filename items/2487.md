## Summary

本 PR 修复了 metric_system_v2 插件中 fd_max 与 fd_allocated 指标因 float64 转换导致的大数精度丢失问题，改为字符串上报。当前 main 分支仍使用 float64 解析，该修复尚未合入，属于有效缺陷修复，需维护者审查并指导 rebase 后合并。

## Best Possible Solution

维护者应审查本 PR 变更，确认下游消费端对字符串类型指标值的兼容性。若方案可行，请指导作者 rebase 至最新 main 分支以解决基线过旧带来的潜在冲突后合并；或在评审中讨论作者提出的替代方案（直接移除恒定的 fd_max 指标）以简化实现。
