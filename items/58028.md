## Summary

Keep open. Current `main` does not implement a core `/resume` slash command for restoring prior OpenClaw conversation context. The built-in command registry and public slash-command docs list `/new`, `/reset`, and `/compact`, but no core `/resume`; the only documented `resume` command is plugin-scoped `/codex resume`. The cross-referenced PR #36481 proposed this feature, but it was closed unmerged due to inactivity, so it is evidence of prior interest rather than a shipped fix.
