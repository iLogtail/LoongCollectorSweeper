## Summary

Keep open. Current `main` does not contain this fix: pending-descendant blocks still return plain `false`, cleanup still treats that as the generic failed-announce path, and the retry/give-up logic on `main` still matches the race the PR describes. The PR discussion also shows unresolved follow-up concerns on the latest revision, so this is neither implemented nor safely obsolete.
