## Summary

Keep open. Current main partially preserves already-extracted tool media into final outbound payloads, but it still does not implement the `details.path`-missing fallback requested in #41744 and linked PR #50359. The current regression test explicitly expects image content with no `MEDIA:` directive and no `details.path` to produce no media paths.
