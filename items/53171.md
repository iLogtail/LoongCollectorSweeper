## Summary

Keep open. Current `main` does not implement this PR’s proposed package change, and the proposed fix is incomplete against today’s packaging contract: the published package still omits `scripts/ui.js` and the `ui/` source tree, while current code/docs treat package-manager installs as shipping prebuilt `dist/control-ui` rather than rebuilding UI assets in place.

## Evidence

- **Published package still excludes the UI helper script while exposing `ui:build`:** `package.json` still publishes only four specific `scripts/*.mjs` files; it does not include `scripts/ui.js` or `ui/`. At the same time, the package scripts still advertise `ui:build`, `ui:dev`, and `ui:install` as `node scripts/ui.js ...` commands.
