## Summary

Keep open. Current `main` at `a6aa151653ab478e9aa23e95c6446c41adc4fce1` still treats any gateway close as terminal for one-shot calls, and `GatewayClient` still flushes pending errors and emits `onClose` for pre-hello `(1000, "")` closes. The PR’s described fix is not implemented on current `main`.
