## Summary

Keep open. Current `main` still makes the GitHub skill depend on `gh` binary discovery, and local binary discovery is cached until `PATH`/`PATHEXT` changes. A fresh `gh` install into an already-listed PATH directory therefore will not become visible to the running gateway process until restart, while the current skill/docs text does not give restart guidance for this case.
