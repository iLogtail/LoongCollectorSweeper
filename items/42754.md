## Summary

Keep open. Current main still reconstructs Feishu card action events with only action.value and action.tag, and the fallback dispatch path only reads action.value. The linked PR #42908 tracks the exact missing support but remains open, while #45961 was closed unmerged, so the requested capability is still absent on main.
