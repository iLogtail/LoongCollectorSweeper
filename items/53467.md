## Summary

Current main still does not implement Slack `ignoreOtherMentions`. The field exists for Discord only; Slack config types/schema, Slack channel resolution, Slack docs, and Slack prepare-message gating do not expose or enforce it. PR #53467 remains useful and appears to be the active canonical implementation; the related PR #55570 was closed unmerged and does not supersede it.

## Best Possible Solution

Keep #53467 open as the canonical active Slack implementation. The maintainer path is to review and land a scoped patch that adds the Slack config type/schema/resolver field, runtime other-mention drop gate, generated config metadata/baselines, docs, and tests for channel drop, explicit bot mention passthrough, DM/no-mention passthrough, and degraded bot identity handling.
