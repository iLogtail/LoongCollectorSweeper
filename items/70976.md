## Summary

Keep open. Current `main` still exposes failover decisions and terminal before-reply collapse primarily through runner/logging paths, while the public plugin hook surface remains limited to coarse lifecycle hooks like `agent_end`. The request is recent, coherent, in core plugin-SDK scope, and the issue timeline already points to active follow-up work in PR #70990.
