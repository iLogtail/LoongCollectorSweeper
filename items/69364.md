## Summary

Keep #69364 open. Current main has useful pieces for per-agent files, tools, skills, channels, cron, and a flat Sessions table, but it does not implement the requested cross-agent graph/team view that relates delegation, dataflow, task sequence, and live session status. The related #52803 remains a broader Control UI orchestration request rather than a canonical duplicate for this specific config/symlink/TASK.json relationship view.

## Best Possible Solution

Keep the issue open as a focused first-party Control UI product request. Best path is to narrow the first implementation to a read-only relationship graph sourced from existing config/session/task/workspace data, reuse the existing per-agent Files/Tools/Skills editors where possible, and coordinate with the broader open #52803 without treating it as an exact duplicate.
