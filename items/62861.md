## Summary

Current main now carries Dreaming config and cron reconciliation through the gateway startup path, keeps the enabled Dreaming plugin in startup scope, and has regression tests for the exact silent-initialization/config-subset failure mode described here. The reported 2026.4.5 behavior is superseded by later shipped fixes, with the most complete matching fix landing by v2026.4.22.
