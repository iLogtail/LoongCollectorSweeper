## Summary

Keep open. Current main has some diagnostic/audit coverage for suspicious macOS config clobbers, but it does not restore the lost configuration, and the macOS Sparkle updater still defaults automatic checking/downloading to enabled. That leaves the reported user-visible problem unresolved rather than implemented, non-reproducible, or superseded.
