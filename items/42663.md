## Summary

Keep open. Current main does not implement the requested Discord `trustedWebhooks` allowlist or `mapToUser` behavior. Discord inbound preflight still drops non-PluralKit bot-authored messages when `allowBots` is off, and the only narrower existing mode is `allowBots: "mentions"`, which is mention-gated rather than webhook-ID allowlisted.

## Evidence

- **No trusted webhook config exists:** A repo search for `trustedWebhooks`, `trustedWebhook`, and `mapToUser` across `src`, `extensions`, and `docs` returned no matches.
  - command: `rg -n "trustedWebhooks|trustedWebhook|mapToUser" src extensions docs`
  - sha: [e640c0a95fcb](https://github.com/openclaw/openclaw/commit/e640c0a95fcbd06a1311f18fc635518d011d45da)
- **Discord guild config is strict and lacks trustedWebhooks:** The strict `DiscordGuildSchema` lists guild-level fields such as `requireMention`, `users`, `roles`, and `channels`, but has no `trustedWebhooks` field.
  - file: [src/config/zod-schema.providers-core.ts:476](https://github.com/openclaw/openclaw/blob/e640c0a95fcbd06a1311f18fc635518d011d45da/src/config/zod-schema.providers-core.ts#L476)
  - sha: [e640c0a95fcb](https://github.com/openclaw/openclaw/commit/e640c0a95fcbd06a1311f18fc635518d011d45da)
- **Discord account config only supports allowBots:** The Discord account schema exposes `allowBots` as `boolean | "mentions"`; it does not define a webhook allowlist or user-mapping shape.
  - file: [src/config/zod-schema.providers-core.ts:521](https://github.com/openclaw/openclaw/blob/e640c0a95fcbd06a1311f18fc635518d011d45da/src/config/zod-schema.providers-core.ts#L521)
  - sha: [e640c0a95fcb](https://github.com/openclaw/openclaw/commit/e640c0a95fcbd06a1311f18fc635518d011d45da)
- **Bot preflight still drops bot authors by default:** Preflight resolves `allowBotsMode` from `allowBots`, then drops `author.bot` messages when the mode is off and the sender is not PluralKit. The resolved webhook ID is not checked against any trusted-webhook policy.
  - file: [extensions/discord/src/monitor/message-handler.preflight.ts:392](https://github.com/openclaw/openclaw/blob/e640c0a95fcbd06a1311f18fc635518d011d45da/extensions/discord/src/monitor/message-handler.preflight.ts#L392)

  - file: [extensions/discord/src/monitor/message-handler.preflight.ts:1016](https://github.com/openclaw/openclaw/blob/e640c0a95fcbd06a1311f18fc635518d011d45da/extensions/discord/src/monitor/message-handler.preflight.ts#L1016)
