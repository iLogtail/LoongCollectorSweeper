## Summary

Keep open. Current `main` still lacks the issue-reported config-context threading and still renders optional memory counts unconditionally. The issue-linked fix PR #62895 was closed unmerged as stale/conflicting, and the broader related fix PR #63013 is still open.

## Evidence

That is the same surface #62895 proposed to change.
