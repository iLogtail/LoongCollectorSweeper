## Summary

Keep open. Current main appears to carry an already-stored token across same-gateway session selection, but the discussion identified a separate loopback token-scope bug, and current main still scopes gateway tokens by exact host. The linked fix PR #43254 is not present in this checkout, so a meaningful reported auth-loss path remains unresolved on main.
