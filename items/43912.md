## Summary

Keep open. Current `main` at `50e36983bb2d827adc617282c090c0a0359c6eee` does not implement this PR's behavior: scoped specs like `@openclaw/feishu` still bypass bundled pre-install resolution, and the existing tests explicitly lock that behavior in.

## Evidence

  - file: [src/cli/plugin-install-plan.test.ts:28](https://github.com/openclaw/openclaw/blob/50e36983bb2d827adc617282c090c0a0359c6eee/src/cli/plugin-install-plan.test.ts#L28)
