## Summary

Keep #8299 open. Current main still does not expose a documented config option or public `sessions_spawn` parameter that deterministically suppresses top-level sub-agent completion announcements; suppression remains token/model-driven, and the earlier implementation PRs were closed unmerged.

## Best Possible Solution

Keep #8299 open until OpenClaw adds a documented framework-level sub-agent announce suppression control, either as `agents.defaults.subagents` config and/or a public `sessions_spawn` parameter. A complete fix should update the TypeScript and zod config contracts, generated schema/help, tool schema/parser, registry and announce delivery contracts, cleanup bookkeeping, docs, and regression tests for deterministic skip behavior.
