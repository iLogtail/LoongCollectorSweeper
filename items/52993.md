## Summary

Keep open. Current `main` does not implement a Feishu message recall/delete tool, so this is not `implemented_on_main`, and the PR still has at least one concrete blocker from review: it introduces a `tools.message` toggle but the strict Feishu config schema on `main` has no `message` key. Current Feishu helpers/tests also show a newer multi-account registration and routing contract that the PR discussion flagged but does not show as resolved.
