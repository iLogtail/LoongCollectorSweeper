## Summary

Current main already implements configurable FTS tokenization for memory search, including a built-in `trigram` option documented for CJK support plus query-side fallback coverage for short CJK searches.

If you still hit misses on current main with `store.fts.tokenizer = "trigram"`, please open a fresh issue with a minimal repro corpus and exact config.
