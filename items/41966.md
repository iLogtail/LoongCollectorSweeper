## Summary

Keep open. Current main still explicitly skips MEDIA extraction for lines inside fenced code blocks and preserves those lines as text, which matches the reported failure mode. The existing regression test also asserts that fenced MEDIA lines are ignored. The related PR #41980 addressed the issue but was closed unmerged, so it does not supersede or implement the issue on main.
