## Summary

Keep open. Current `main` still lacks the PR's `"cli"` manager-purpose path in the moved memory-core code, and the remaining gaps called out in review discussion are still present: `memory status --index`, `memory index`, and `memory search` still use the watcher-enabled/default path, while the manager and QMD search-manager code only special-case `"status"`.
