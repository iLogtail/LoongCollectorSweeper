## Summary

Keep open. Current main does not implement the requested old_text/new_text aliases; the edit tool now validates a path plus edits[] payload and still rejects legacy/top-level alias parameters. The issue discussion and related unmerged PRs remain relevant, including the follow-up comment that Claude also emits failing oldString/newString variants.
