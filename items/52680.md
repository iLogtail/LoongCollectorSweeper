## Summary

Keep open. Current main does not contain the requested recovery path: the OpenClaw read wrapper still calls the upstream read tool directly for explicit limits and for paged reads without a catch/retry/diagnostic path, so an upstream out-of-range offset error would still propagate. The related PRs in the provided GitHub context are useful evidence, but they were closed unmerged and no current regression test is present on main.

## Evidence
