## Summary

Keep #70153 open. Current main still has the auto-reply CLI path call `runCliAgent` directly and return the result without persisting the successful turn transcript, while the direct CLI command path already persists CLI turns. The PR is a focused bug fix with tests and a follow-up commit that addresses the threadId review feedback from Greptile.

## Best Possible Solution
