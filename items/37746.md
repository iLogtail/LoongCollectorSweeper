## Summary

Keep open. Current main still treats heartbeat work as prompt text sent to the model, and the skill metadata/runtime surfaces do not include declared sandboxed function schemas or a core-enforced remote invocation path. The issue describes a real security/API gap, not a ClawHub-only cleanup item.
