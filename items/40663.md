## Summary

Keep open. Current `main` still renders the contradictory sandbox path guidance this PR is trying to fix, and the surrounding sandbox bridge code supports the PR's core claim that container paths are mapped automatically. The PR still looks useful, but it is not implemented on `main`, and its review thread also identified a missing test update that would need to be addressed before merge.
