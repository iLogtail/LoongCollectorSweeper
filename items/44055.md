## Summary

Keep open. Current `main` does not contain the `acp-remote` backend/plugin-sdk surface from PR #44055, and the PR’s ACP cwd-behavior fixes are also absent from the checked-out code.

## Evidence

- **No `acp-remote` plugin or `plugin-sdk/acp` entrypoint on current main:** A repo search only finds the existing `extensions/acpx/*` backend and `src/plugin-sdk/acp-runtime.ts`; there is no `extensions/acp-remote/` tree and no `src/plugin-sdk/acp.ts` file.
  - command: `rg --files extensions src/plugin-sdk | rg '(^extensions/acp-remote/|^src/plugin-sdk/acp\.ts$|^extensions/acpx/|^src/plugin-sdk/acp-runtime\.ts$)'`
  - sha: [50e36983bb2d](https://github.com/openclaw/openclaw/commit/50e36983bb2d827adc617282c090c0a0359c6eee)
- **Package exports still expose only `acp-runtime`-style ACP SDK subpaths:** `package.json` exports `./plugin-sdk/acp-runtime`, `./plugin-sdk/acp-binding-runtime`, and `./plugin-sdk/acp-binding-resolve-runtime`; there is no generic `./plugin-sdk/acp` export matching the PR.
  - file: [package.json:404](https://github.com/openclaw/openclaw/blob/50e36983bb2d827adc617282c090c0a0359c6eee/package.json#L404)
  - sha: [50e36983bb2d](https://github.com/openclaw/openclaw/commit/50e36983bb2d827adc617282c090c0a0359c6eee)
- **ACP backend messaging is still acpx-specific:** `requireAcpRuntimeBackend()` still throws `Install and enable the acpx runtime plugin.`, which shows current `main` has not adopted the PR’s backend-agnostic ACP runtime wording.
  - file: [src/acp/runtime/registry.ts:88](https://github.com/openclaw/openclaw/blob/50e36983bb2d827adc617282c090c0a0359c6eee/src/acp/runtime/registry.ts#L88)

That does not match a shipped remote ACP backend plugin.
  - file: [docs/cli/acp.md:32](https://github.com/openclaw/openclaw/blob/50e36983bb2d827adc617282c090c0a0359c6eee/docs/cli/acp.md#L32)

That is the opposite of the PR’s `preserve omitted cwd semantics on reopen/reload` work.
  - file: [src/acp/control-plane/runtime-options.ts:266](https://github.com/openclaw/openclaw/blob/50e36983bb2d827adc617282c090c0a0359c6eee/src/acp/control-plane/runtime-options.ts#L266)

  - file: [src/acp/session.ts:84](https://github.com/openclaw/openclaw/blob/50e36983bb2d827adc617282c090c0a0359c6eee/src/acp/session.ts#L84)
