## Summary

Current `main` does not contain this PR's fix. The Channels UI still derives card order directly from `snapshot.channelMeta` or `snapshot.channelOrder`, so a snapshot containing only custom channel ids would omit built-in cards. The gateway snapshot is still built from currently loaded channel plugins only, and loaded plugins are not guaranteed to include bundled built-ins.
