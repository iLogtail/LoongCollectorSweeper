## Summary

Keep open. Current main does not already implement the PR's session working-context capsule, and the linked umbrella issue #67511 remains open rather than superseded by a merged implementation. This PR is a recent, focused draft that adds a plausible core persistence primitive with tests; it is not a ClawHub/plugin-bound request because the requested behavior spans core session persistence, /status, compaction, and subagent spawn lifecycle.

## Best Possible Solution

Keep the PR open for maintainer design review as a focused MVP toward #67511. The best path is to either accept the storage primitive with its tests after deciding the on-disk shape/locking contract, or ask the author to add the first integration point and docs so the capsule becomes observable through `/status`, compaction, and spawn flows. This belongs in core session/runtime code rather than ClawHub because it affects OpenClaw's own session continuity contract.
