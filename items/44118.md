## Summary

Current `main` does not implement this Feishu slash-command mention-bypass. The group mention gate still returns before command detection, so `/reset`/`/help`-style commands in `requireMention` group chats are still dropped unless the bot is mentioned. The broadcast path also still derives `activeAgentId` only from `ctx.mentionedBot || !requireMention`, so a non-mentioned command would not activate any broadcast agent.
