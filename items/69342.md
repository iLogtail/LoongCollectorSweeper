## Summary

Keep open. Current `main` still does not implement per-model provider `extraParams`: the config type/schema reject the field, the runtime extra-param merge path has no model-level source, and the model-resolution paths do not preserve configured model `extraParams`. The PR remains the only implementation path in the provided context, so there is no strong basis for conservative auto-close.
