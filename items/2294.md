## Summary

本 PR 旨在为 e2e 测试引擎新增 CPU 与内存混沌注入步骤，属于有价值的测试基础设施增强。当前代码中部分函数（如 CPU100NProc、Oom、MemFrag）硬编码依赖了未随 PR 提交的外部脚本（/root/e2eshel/*），且存在维护者明确指出的未解决审查意见。因缺失关键测试资产导致 CI 无法独立验证，需等待作者补充脚本或重构实现，故保持开启。

## Best Possible Solution

建议作者响应维护者审查意见，将缺失的 /root/e2eshel/ 脚本提交至仓库测试资源目录，或统一重构为使用已集成的 chaosblade 命令及标准 Linux 工具以消除外部硬编码依赖。待依赖补齐、CI 通过且 Review 批准后合并。
