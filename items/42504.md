## Summary

Current main does not implement the requested Agents tab avatar upload path. It can display configured agent avatars and `agents.update` accepts an `avatar` string, but there is no `agents.avatar.upload` RPC, no protocol schema/result type for avatar uploads, and no Agents tab file picker that writes image bytes to `workspace/avatars/` and updates `IDENTITY.md`.
