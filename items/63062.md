## Summary

Keep open. Current `main` still routes OpenRouter Anthropic payloads through `applyAnthropicEphemeralCacheControlMarkers`, and that helper only marks `system`/`developer` content and strips assistant thinking markers. It does not apply cache markers to the trailing user turn, while the direct Anthropic path still does. The existing `main` tests also assert the old OpenRouter behavior, so this PR is not stale or already implemented.
