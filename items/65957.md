## Summary

Keep open. Current `main` still contains the exact routing/prompt behavior this PR changes, so it is not already implemented and there is not enough evidence to conservatively close it. The PR discussion also shows the author addressed review findings and that the remaining red merge-run checks were reported as unrelated merge-base breakage, not a rejection of the heartbeat fix.
