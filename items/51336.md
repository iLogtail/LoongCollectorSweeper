## Summary

Keep open. Current main still formats overload and rate-limit failures with provider-agnostic user-facing text, even though provider/model context is available in nearby paths. The linked implementation PR #55851 tracks this request but is still open, so the issue is not implemented or safely superseded by a landed fix yet.
