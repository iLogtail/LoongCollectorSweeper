Keep #40039 open. Current main still lacks the proposed limited-tools troubleshooting section, and the PR is salvageable docs work, but it should be revised before merge because current main supports the `minimal` profile and does not ship the `Global tools profile` status row referenced by the PR text.

## Best Possible Solution
