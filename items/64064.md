## Summary

Keep #64064 open. Current main does not yet implement Anthropic advisor tool support: there is no advisor wrapper/config path, no advisor beta injection tied to `params.advisor`, no server-side result display handling, and unknown Anthropic stop reasons still throw. The PR is a plausible focused implementation with review discussion and live validation evidence, while the canonical related tracker #63930 remains open.

## Best Possible Solution

Keep the PR open for maintainer review. Close only if the same behavior lands on main through another canonical PR or a maintainer explicitly decides the feature should not be part of the bundled Anthropic provider.
