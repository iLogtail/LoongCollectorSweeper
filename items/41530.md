## Summary

Keep open. Current main partially mitigates the reported default data-loss path by making ordinary run-mode subagents default to cleanup="keep", but meaningful requested work remains: the archive TTL is still 60 minutes, delete-mode runs still bake archiveAtMs at registration time, restored persisted runs keep their old archiveAtMs, and the sweeper still calls sessions.delete with deleteTranscript:true once that timestamp expires. The cross-reference to #68981 is related session-management UX work, but it does not supersede this regression report.
