## Summary

Keep open. Current `main` still matches the reported behavior: subagent auto-archive defaults to 60 minutes, the archive deadline is baked into `archiveAtMs` when the run is registered/replaced, and the sweep path deletes the session with `deleteTranscript: true`. I found docs for the feature, but not evidence that the regression was removed or migrated away.
