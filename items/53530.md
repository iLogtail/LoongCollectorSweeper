## Summary

Current `main` does not implement this PR's direct-session/DM tool-policy behavior. The core resolver still only derives tool policy from group/channel session contexts, Feishu DM config still lacks `tools`/`toolsBySender`, and the empty sender-override fallthrough fix is also absent, so there is not strong evidence to close this PR as already implemented or obsolete.
