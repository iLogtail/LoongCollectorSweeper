## Summary

Keep open. Current `main` only does Discord audio preflight to decide whether a mention-gated message should pass the mention check; unmentioned voice messages are still skipped and their pending history is stored as the normal message text placeholder, not as a transcript. There is also no Discord config field for a `preflightTranscribe`-style option on `main`.

## Evidence

- **Discord has no `preflightTranscribe` channel/guild setting:** The Discord channel and guild config types expose `requireMention`, `ignoreOtherMentions`, tool overrides, allowlists, threading, etc., but no `preflightTranscribe` or equivalent silent-transcription flag.
