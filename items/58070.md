## Summary

Current `main` does not implement this PR's macOS LaunchAgent changes. Generated launchd plists still omit `ProcessType=Interactive` entirely and only emit `WorkingDirectory` when an explicit directory is passed, so the reported behavior is still outstanding rather than implemented or obsolete.
