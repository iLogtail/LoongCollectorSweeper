## Summary

Keep open. Current `main` still lacks a TUI refresh path for external `sessions.reset`, so this is not implemented on `main`; the bug surface is narrower now because `/new` was changed to isolate TUI sessions, but the shared-session `/reset` case remains and the PR needs a rebase against newer TUI state handling rather than auto-closure.
