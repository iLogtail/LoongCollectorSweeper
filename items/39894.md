## Summary

Keep open. Current `main` still parses outbound reply directives from raw `payload.text` in the shared outbound normalization path, so the reported metadata-leak bug is not implemented away on `main`, and the specific metadata-plus-`MEDIA:` regression test from the PR is not present.

## Evidence
