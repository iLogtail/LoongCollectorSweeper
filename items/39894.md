## Summary

Current `main` still feeds outbound reply text directly into directive parsing without stripping echoed inbound metadata first, so the reported leak is not implemented on `main`. This draft PR still contains useful unmerged work for the bug, even though it is mixed with unrelated CI/macOS test-runner changes.
