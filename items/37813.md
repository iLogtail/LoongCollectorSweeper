## Summary

Current main appears to have partial fixes for this report, including xAI forward-compat model resolution, session-override fallback-chain handling, and fallback notices in status output. Keep open because the requested-model tool-permission inheritance still appears missing: fallback attempts run with the active fallback provider/model, and tool policy is resolved from that active model rather than the originally requested model.
