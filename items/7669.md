## Summary

Keep #7669 open. Current main still does not provide a supported npm dev dist-tag publish path; the release workflow, publish helper, release-plan types, tests, and release docs all model stable/latest and beta, while dev remains a git/main channel with npm dev described only as optional.

## Best Possible Solution

Keep #7669 open until maintainers either implement a first-class main-snapshot npm dev path, including workflow trigger/input support, release-plan validation, publish-helper preservation of `dev`, post-publish verification, and docs; or explicitly retire npm dev snapshots and update the public release-channel, updating, FAQ, and release docs to make dev git-only.
