## Summary

Keep PR #71062 open. Current main does not implement the central fixes: the client-tool adapter still returns a plain pending result without Pi's `terminate: true`, the `/v1/responses` SSE schema/streaming path still lacks `response.function_call_arguments.{delta,done}` events, and the upstream Responses parser still handles argument deltas but not `.done`-only final arguments. The PR is a recent, focused bug-fix attempt with active author follow-up to bot review comments, so no conservative auto-close reason applies.

No ClawSweeper close reason fits because current main still lacks the requested behavior and there is no canonical duplicate or superseding shipped fix in the provided context.
