## Summary

Current `main` does not implement this PR’s requested behavior for intermediate mutating tool failures. The embedded runner still intentionally surfaces mutating tool warnings even when there is a final assistant reply, and `edit` is classified as a mutating tool, so the reported `edit` retry warning case remains in scope. Main also documents the existing shipped behavior as hiding only non-mutating tool-failure warnings, not all intermediate failures.
