## Summary

Keep open. Current `main` still has a real docs/runtime/schema mismatch for Mattermost preview streaming: the docs advertise `channels.mattermost.streaming`, but the Mattermost plugin schema/types/doctor path do not support that key, and runtime always wires the draft-preview flow without a configurable preview mode. The reporter's separate 'many deleted posts' symptom is not clearly supported by current tests, but the core bug remains unresolved.

## Evidence
