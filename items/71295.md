## Summary

Keep open. The reported broken stable release still exists with zero macOS assets, and current `main` still shows a split release process where the public macOS workflow is validation-only and asset publication/appcast promotion happen later in private follow-up steps. I did not find a landed fix on `main` that gates stable promotion on successful macOS asset upload.
