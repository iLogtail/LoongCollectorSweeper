## Summary

Keep open. Current main still rejects `action: "send"` when non-empty poll creation fields are present, and the message tool still exposes shared poll creation fields. Main only appears to have the narrower zero-valued numeric default mitigation, not the requested action-aware stripping, split schema, or channel-scoped poll fields.
