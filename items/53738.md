## Summary

Keep open. Current `main` still bypasses the config-aware resolver in both the PDF and image media paths, so the reported `models.providers[provider].baseUrl` override gap remains plausible and unfixed. The PR discussion also shows substantive follow-up on fallback edge cases rather than an obsolete or incoherent change.
