## Summary

Keep #11492 open. Current main still does not expose Slack `channel-info` or `channel-list` through the message tool, even though the generic action names and Discord parity path exist. The related implementation PR #11491 is closed unmerged, and PR #53199 remains open and only partially overlaps the request.

## Best Possible Solution

Keep the issue open until Slack exposes both requested message-tool actions on main. The likely fix belongs in `extensions/slack`: add `channel-info` and `channel-list` to Slack action discovery behind `channelInfo`, dispatch them through `message-action-dispatch`, implement runtime helpers using Slack `conversations.info` and paginated `conversations.list`, and update Slack docs plus focused discovery/dispatch/runtime tests.
