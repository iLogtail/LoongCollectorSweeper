## Summary

Keep #11492 open. Current main still does not expose Slack `channel-info` or `channel-list` through the Slack message tool, even though the shared action names, Slack `channelInfo` config key, and Discord parity implementation exist. Related PR #11491 is closed unmerged, and PR #53199 is still open and only partially covers this issue because it proposes `channel-list` plus search, not the requested `channel-info` action.

## Best Possible Solution

Keep the issue open until the bundled Slack plugin exposes both `channel-info` and `channel-list` through the message-tool surface. The likely implementation belongs in `extensions/slack`: add discovery behind the existing `channelInfo` gate, dispatch generic actions to Slack runtime actions, implement `conversations.info` and paginated `conversations.list` helpers, and update Slack docs plus focused discovery, dispatch, and runtime tests.
