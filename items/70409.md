The PR is also not cleanly merge-ready: the provided review context flags P1 issues in the proposed apply/rollback and daily-signal ingestion paths, and the file list includes unrelated Bedrock Mantle/CI churn.

## Best Possible Solution

Keep #70409 open for maintainer review or ask the contributor to rescope it.
