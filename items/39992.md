## Summary

Keep open. Current `main` partially improves discoverability by synthesizing allowlisted models into catalog-backed `models.list` surfaces, but it still does not implement the requested general `openclaw doctor` warning for runtime synthetic models that are available yet missing from `agents.defaults.models`. The only related doctor check is a narrow `hooks.gmail.model` allowlist warning.
