## Summary

Keep open. Current `main` does not implement a doctor check for runtime-available models missing from `agents.defaults.models`. The issue body is partly outdated because model discoverability has improved on `main`, but the requested doctor warning is still absent.

The only model-related entry is `doctor:hooks-model`; there is no contribution that compares the runtime model catalog against `agents.defaults.models`.
