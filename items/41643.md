## Summary

Keep open. Current main only implements part of this PR: `validateAnthropicTurns()` already understands `toolCall`, but the narrower Anthropic-only recovery for empty assistant `stopReason: "error"` stubs is not present, and transport conversion still drops all errored assistant turns, including non-empty partial responses.
