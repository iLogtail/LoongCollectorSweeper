## Summary

Keep open. Current `main` already sanitizes normal final delivery through the shared assistant-visible text pipeline, but the Discord draft-preview paths still bypass that sanitizer for streamed partials and preview-final edits. The PR is directionally useful, but it targets old Discord paths and still has unresolved review concerns, so there is not strong evidence to close it as already implemented or obsolete.
