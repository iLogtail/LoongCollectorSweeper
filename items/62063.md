## Summary

Keep PR #62063 open. Current `main` still does not ship Swedish Control UI locale support: `sv` is absent from the locale type, lazy registry, browser-locale resolver, generated locale files, i18n metadata, docs, and registry/structure tests. The PR addresses a real missing capability, and the provided review discussion shows the initial missing navigator detection issue was addressed by the second PR commit. It should be rebased/refreshed against current `main`, which has since added Thai locale support.

## Best Possible Solution
