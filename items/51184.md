## Summary

Keep open. Current main still renders raw session keys in the CLI/TUI `/status` session list and in the TUI header/statusline path. The Control UI dropdown portion was addressed separately, and open PR #51257 tracks part of the remaining `/status` work, but it is unmerged and does not clearly cover the full statusline/session-picker request from this issue.
