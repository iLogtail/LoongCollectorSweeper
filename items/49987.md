## Summary

Keep open. Current `main` still does not write a restart sentinel from `openclaw gateway restart`, so the user-visible notification this PR targets is not implemented yet. The PR discussion also surfaced an unresolved correctness risk: the proposed sentinel write would default to the CLI shell environment instead of the daemon service environment, which can miss profile/custom state-dir installs.
