## Summary

Current main does not yet implement the requested doctor/status visibility fix. The bundled `memory-lancedb` plugin is still a `kind: "memory"` plugin that registers tools, CLI hooks, lifecycle hooks, and a service, but it does not register a memory runtime/capability on main. PR #65066 is open and directly tracks the fix, but it is not merged, so the issue should stay open until that lands or maintainers decide otherwise.
