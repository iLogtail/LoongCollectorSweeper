## Summary

Keep #52510 open. Current main still has the core cleanup-hint footgun: when any extra gateway-like service is detected, doctor prints cleanup commands for the canonical active gateway service instead of the detected service, and the darwin scanner still broadly matches plist contents containing "openclaw". The linked fix PR #52515 is still open, and the broader #54740 does not make this focused report obsolete.
