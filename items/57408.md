## Summary

Keep open. Current `main` and the latest release still intentionally ignore `OPENCLAW_CONFIG_PATH` and `OPENCLAW_STATE_DIR` from a project-local workspace `.env`, and the test suite explicitly locks that behavior in, so #57408 is not implemented on `main` and is not obsolete.
