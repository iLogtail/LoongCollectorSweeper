## Summary

Keep #43117 open. Current main already supports Gemini Embedding 2, `outputDimensionality`, and text-query search over image/audio files indexed from `memorySearch.extraPaths`, matching newcodebook’s helpful comment. But the original request still asks for multimodal conversation attachments plus broader Gemini modalities such as PDF/video, and current code/docs explicitly limit multimodal memory to image/audio files under `extraPaths` while session transcript indexing remains text-only.
