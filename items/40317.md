## Summary

Keep open. Current `main` still uses strict schema validation for config writes, gateway startup still treats invalid configs as invalid instead of stripping unknown keys, and the only unknown-key stripping on `main` is a doctor repair helper rather than an operational read-mode. The linked follow-up PR `#46213` also still exists as an open remote ref, so this request remains active rather than stale or already shipped.
