## Summary

Keep open. Current `main` still does not expose configurable concurrency for `CommandLane.Nested`, and cron/hook isolated agent work still routes onto the shared `nested` lane. The report’s `sessions_send` deadlock claim is partly outdated because `sessions_send` now uses per-session nested lanes, but the main feature request remains unimplemented.
