## Summary

本 Issue 请求为内置 Loki Flusher 增加对 Grafana Loki 结构化元数据（Structured Metadata）的支持。经核查当前 main 分支代码，该功能尚未实现，且需求提供了明确的 API 载荷规范与官方文档参考，属于合理且边界清晰的核心插件增强。无关联 PR 或替代工单，应保留以待排期或社区贡献。

## Best Possible Solution

在现有 Loki Flusher 插件中新增配置项（如 StructuredMetadataKeys），允许用户指定需提取为结构化元数据的日志字段。在序列化推送载荷时，将匹配字段组装为 JSON 对象并附加至 values 数组的第三项。实现需同步更新插件配置文档、增加单元测试覆盖高基数元数据场景，并在 Release Notes 中说明兼容性。建议维护者评估后分配里程碑或接受社区 PR。
