## Summary

Keep open. Current main still has the Docker VM runtime docs bug, and #71796 contains a useful wacli correction that the related #72154 does not fully cover. The PR is not mergeable as-is because its AGENTS.md gateway wording would document an invalid `openclaw gateway restart --deep` invocation, and the claimed Node baseline change is not actually in the PR diff.

## Best Possible Solution

Keep the PR open for maintainer/author revision or split it. The Docker URL fix should coordinate with #72154 and include an explicit wacli Linux availability decision; the AGENTS.md gateway line should be corrected to distinguish `gateway restart` from `gateway status --deep`; the Node baseline should either be patched to `22.14+`/Node 24 recommended or removed from the PR claims.
