## Summary

Keep PR #71796 open. Current main still has the Docker VM runtime URL problems and the Node baseline mismatch the PR targets, so the work is not obsolete or duplicated; however, the PR needs revision because its AGENTS.md gateway wording would still imply `openclaw gateway restart --deep`, while current CLI/docs only support `--deep` on `gateway status`.

## Best Possible Solution

Keep the PR open and request a small revision: retain the Docker VM runtime URL fixes after current asset verification, align the Node baseline with `package.json` engines, and change the AGENTS.md gateway guidance so restart remains `openclaw gateway restart` while deep diagnostics use `openclaw gateway status --deep`. After that, a docs-only changed gate is sufficient.
