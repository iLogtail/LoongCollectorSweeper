## Summary

Keep open. Current `main` still has the docs/schema mismatch: `docs/tools/exec.md`, `src/config/types.tools.ts`, and the generated base config schema all advertise `tools.exec.approvalRunningNoticeMs`, but runtime validation still parses top-level `tools.exec` through `ToolExecSchema`, which omits that key. The same omission is present in the latest release commit `a9797214338ba31c52c796adbb75afb16e0684a9`, so this is not implemented on shipped builds yet.
