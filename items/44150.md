## Summary

Keep open. Current `main` still does not pass `messages` into `before_compaction` hooks, so the feature is not implemented. The PR’s proposed `messages: session.messages` approach also appears unsafe because compaction mutates the live session array in place, so hooks could observe or cause transcript mutation unless this is changed to a snapshot and covered by stronger assertions.
