## Summary

Keep #60078 open. Current main still appears to route subagent announce follow-up calls through the gateway `agent` path without applying `channels.modelByChannel`, while normal reply/status code paths do resolve channel model overrides. The request is a real, recent core behavior bug rather than a cleanup close candidate.
