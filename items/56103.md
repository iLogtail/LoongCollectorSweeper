## Summary

Keep open. Current `main` does not implement `ui.seamColor` for the Control UI frontend, so this is not `implemented_on_main`, but the repo also documents and implements `seamColor` primarily for native/Talk Mode surfaces, so there is not strong evidence to close this PR as obsolete or invalid. The item still needs maintainer product judgment and a cleaner implementation than the submitted patch.
