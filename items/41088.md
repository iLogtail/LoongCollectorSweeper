## Summary

Keep open. Current `main` still launches Chrome with `--disable-features=Translate,MediaRouter` only, and headless mode only adds `--headless=new` plus `--disable-gpu`; the requested OptGuide/WasmTts download-disabling flags are not implemented by default. OpenClaw does expose a generic `browser.extraArgs` workaround, but not the requested automatic headless/server behavior or dedicated config knob.
