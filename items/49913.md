## Summary

Keep open. Current `main` still uses the Unix-only `git ls-files ... | xargs oxfmt` commands for both `format:docs` and `format:docs:check`, so the underlying Windows/PowerShell problem is not implemented on `main`. The PR also has unresolved review concerns: its proposed `spawn("oxfmt", ...)` approach would need Windows `.cmd` handling, and it only updates the check variant, not the write variant.
