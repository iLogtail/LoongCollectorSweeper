## Summary

Keep open. Current `main` still emits `model.usage` from the reply-runner path but not from the CLI/ingress `agentCommandInternal` path, so the gap described by the PR remains real. The proposed patch is stale against current `main` because the code moved from `src/commands/agent.ts` to `src/agents/agent-command.ts`, and current context-limit resolution now goes through `resolveContextTokensForModel`, matching the review feedback that the original patch shape would need adjustment rather than closure.
