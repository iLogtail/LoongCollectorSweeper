## Summary

Keep open. Current `main` still promotes assistant text from assistant messages that also contain tool calls into `latestAssistantText`, then prefers and freezes that text as the subagent completion result. I did not find a local merged commit for the linked PR `#55510` in this checkout.

## Evidence

- **Mixed assistant+tool-call turns are still accepted as latest assistant output:** `summarizeSubagentOutputHistory()` counts tool calls on assistant messages, but it still assigns any extracted assistant text from that same message to `latestAssistantText` and `assistantFragments` without checking whether the message also contained tool calls.
  - file: [src/agents/subagent-announce-output.ts:208](https://github.com/openclaw/openclaw/blob/34896839ba22f3a71a3d5a8fb4ca5f4c6989a534/src/agents/subagent-announce-output.ts#L208)
  - command: `nl -ba src/agents/subagent-announce-output.ts | sed -n '156,290p'`
  - sha: [34896839ba22](https://github.com/openclaw/openclaw/commit/34896839ba22f3a71a3d5a8fb4ca5f4c6989a534)
- **That mixed-turn text is still preferred as the completion reply:** `selectSubagentOutputText()` returns `latestAssistantText` before partial-progress formatting or raw tool output, and `readSubagentOutput()` uses that selection directly from `chat.history`.
  - file: [src/agents/subagent-announce-output.ts:266](https://github.com/openclaw/openclaw/blob/34896839ba22f3a71a3d5a8fb4ca5f4c6989a534/src/agents/subagent-announce-output.ts#L266)
  - command: `nl -ba src/agents/subagent-announce-output.ts | sed -n '156,290p'`
  - sha: [34896839ba22](https://github.com/openclaw/openclaw/commit/34896839ba22f3a71a3d5a8fb4ca5f4c6989a534)
- **Selected reply is still frozen into durable subagent state:** `freezeRunResultAtCompletion()` captures the completion reply and writes it straight into `entry.frozenResultText`, so any mixed-turn text selected upstream becomes the persisted child result.
  - file: [src/agents/subagent-registry-lifecycle.ts:203](https://github.com/openclaw/openclaw/blob/34896839ba22f3a71a3d5a8fb4ca5f4c6989a534/src/agents/subagent-registry-lifecycle.ts#L203)
  - command: `nl -ba src/agents/subagent-registry-lifecycle.ts | sed -n '188,215p'`
  - sha: [34896839ba22](https://github.com/openclaw/openclaw/commit/34896839ba22f3a71a3d5a8fb4ca5f4c6989a534)
- **Assistant text extraction does not reject messages with tool calls:** `extractAssistantText()` joins assistant text blocks from assistant messages and has no guard that excludes assistant messages whose `content` also includes tool-call blocks.
  - file: [src/agents/tools/chat-history-text.ts:41](https://github.com/openclaw/openclaw/blob/34896839ba22f3a71a3d5a8fb4ca5f4c6989a534/src/agents/tools/chat-history-text.ts#L41)
