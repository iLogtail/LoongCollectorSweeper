## Summary

Keep #61347 open. Current main has useful partial mitigations for this problem family, including global continuation skipping, a full disable switch, fixed character budgets, and heartbeat-specific lightweight/isolated modes. It still does not implement the central request: declarative per-file selective injection, lazy/TOC workspace context, or token-budget-aware summarization for normal workspace bootstrap files.

## Best Possible Solution

Keep the issue open, but narrow it around a first-class bootstrap context policy: declarative per-file inclusion conditions by run kind/session kind, an optional lazy or TOC mode for normal workspace files, and clear interaction with existing `contextInjection`, heartbeat `lightContext`, `bootstrapMaxChars`, `bootstrapTotalMaxChars`, `agent:bootstrap` hooks, `/context` reporting, and docs. The shipped heartbeat fixes should remain as partial mitigation, not the canonical resolution for this request.
