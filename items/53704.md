## Summary

Keep open. Current `main` still has the schema mismatch described by the PR: the Control UI `Set Default` action writes `agents.defaultId`, but the strict agents config schema only allows `defaults` and `list`, and runtime default resolution still comes from `agents.list[].default`. This is not implemented on `main` yet, so there is no conservative close path.
