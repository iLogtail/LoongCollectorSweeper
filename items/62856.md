## Summary

Current `main` no longer matches the reported circular-schema crash. Plugin config loading only AJV-validates each plugin manifest `configSchema`, and the current Google/MiniMax manifests are small acyclic object schemas. Separately, Gemini-family tool-schema normalization now guards repeated local `$ref` expansion with a ref stack, which prevents infinite recursion. The same guarded cleaner is already present in release `v2026.4.23`.

## Evidence
