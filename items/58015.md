## Summary

Keep open. Current `main` does not implement the PR’s claimed OpenRouter+Mistral `strict9` tool-call ID handling. The old `src/agents/provider-capabilities.ts` path from the PR no longer exists; replay policy ownership moved into provider plugins, and the OpenRouter plugin still only opts into Gemini thought-signature passthrough behavior, not `strict9` tool-call ID sanitization.
