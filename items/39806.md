## Summary

Keep PR #39806 open. Current main still has the Debug Snapshots UX gap from #39777: disconnected refresh returns silently before setting a loading/offline state, and the Snapshots card body still renders only Status, Health, and Last heartbeat blocks while the Refresh button text changes during loading. The PR targets a real open bug, but its patch still needs review because the provided review context flags a layout regression risk.

## Best Possible Solution
