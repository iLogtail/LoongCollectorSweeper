## Summary

Keep open. Current `main` still routes ACP parent relay updates only through local `onAgentEvent(...)` delivery, so the gateway-state recovery proposed in PR #45739 is not implemented on `main`, and the PR discussion still contains unresolved correctness concerns about the proposed fallback logic.
