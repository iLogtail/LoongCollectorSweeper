## Summary

Keep open. Current `main` still does not implement `openclaw agent --cwd`: the CLI option and docs are absent, the gateway `agent` protocol still has no public `workspaceDir` field, and the server continues to treat workspace overrides as internal spawned-session inheritance only. The PR discussion also shows the contributor addressed the main review concerns around authorization and path normalization, so this remains useful pending maintainer review rather than cleanup closure.
