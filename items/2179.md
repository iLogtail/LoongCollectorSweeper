## Summary

本 Issue 请求为 flusher_http 增加单字段提取并作为 HTTP Body 输出的便捷配置。经核查当前 main 分支，该功能尚未内置，且涉及 Processor 与 Flusher 的职责边界划分，需维护者进行产品裁决或提供标准 Processor 链路实践文档，故保持开启。

## Best Possible Solution

保持开启以待维护者产品裁决。建议维护者评估：1) 是否将此便捷提取逻辑纳入 flusher_http 核心配置以降低用户门槛；2) 或明确推荐通过标准 Processor 链路（如前置 processor_json 解析、processor_drop 剔除无关字段、processor_rename 提升目标字段）完成数据重组后再交由 flusher_http 发送。若决定不纳入核心，可补充标准 Processor 实践文档或示例配置后关闭本 Issue。
