## Summary

Keep open. Current `origin/main` still exact-compares normalized gateway entrypoint paths, still treats `dist/entry.js` vs `dist/index.js` in the same install root as a mismatch in the doctor tests, and still has the runtime/install split that makes the report plausible (`openclaw.mjs` boots `dist/entry.js` while install resolution prefers `dist/index.js`). This PR addresses a live bug and is not implemented on main.
