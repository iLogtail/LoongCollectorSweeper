## Summary

Keep #52213 open. Current main has WhatsApp audio preflight support, but the group mention gate still runs before that preflight path, so a requireMention group voice note whose raw body is `<media:audio>` can still be skipped before transcription. The related PR #62788 was useful context and current main appears to include downstream transcription pieces, but it does not prove the ordering requested by this issue is fixed.

## Evidence
