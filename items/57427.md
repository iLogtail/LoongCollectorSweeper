## Summary

Keep open. Current `main` still documents and implements sandbox exec without inheriting host or `.env` variables, and the OpenShell backend only forwards explicit sandbox/request env into the remote command path. That means the reported behavior is not implemented on `main` and remains a plausible current bug/feature gap.

## Evidence
