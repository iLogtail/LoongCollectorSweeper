## Summary

The linked bug #39392 remains plausible from the current render path: switching agents updates selected state, but the overview panel stays active, renders the same unkeyed Lit subtree, and contains native model/fallback form controls. This PR remains potentially useful work, so it is not a conservative cleanup candidate.

## Best Possible Solution
