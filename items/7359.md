## Summary

Keep open. Current main records outbound Slack/channel sends into their own target session and exposes manual cross-session recall tools, but it still defaults session visibility to `tree` (current session plus spawned children). That means a Slack DM session still cannot see sibling channel sessions by default, so the requested unified DM/channel self-visibility is not implemented on main.
