## Summary

Current main does not implement the requested MSYS2 install.sh support or fish-specific PATH guidance. The installer still classifies anything other than darwin, linux-gnu, or WSL as unsupported and points Windows users to install.ps1; PATH persistence/warnings in install.sh are still bash/zsh-only. The related Microsoft tracker PR #49126 was closed unmerged and is not a canonical superseding fix.
