## Summary

Keep open: current `main` still does not implement a separate WhatsApp outbound `allowTo` allowlist. Config, docs, outbound target resolution, reaction authorization, cron implicit delivery, and directory-backed target lookup are all still coupled to `allowFrom`, so the PR remains materially useful rather than obsolete.
