## Summary

Keep open. Current main still generates session-memory slugs through a temp session file and `runEmbeddedPiAgent`, so #65134 is not obsolete or implemented. The PR’s core `completeSimple` direction is supported by existing short-text generation patterns, but the branch also includes unrelated timeout-failover logging changes that should be split or dropped before maintainer review.

## Best Possible Solution

Keep #65134 open for maintainer review, but ask for a narrow slug-only revision: replace the embedded-agent slug call with the existing `completeSimple` model/auth/transport pattern, update `src/hooks/llm-slug-generator.test.ts`, and split the unrelated timeout-failover logging changes into a separate PR. Coordinate the model-selection choice with #33962, either by deliberately reusing `agents.defaults.heartbeat.model` or by adding a slug-specific config surface.
