## Summary

Keep open. Current main no longer has the reported 2-minute default: exec approvals now default to 30 minutes, and docs say pending approvals expire after 30 minutes. But the requested user-configurable approval timeout is still not implemented in openclaw.json/config; the current config surface only covers approval forwarding/native routing, and the related PR #51292 remains unmerged.
