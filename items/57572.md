## Summary

Keep open. Current `main` does not implement the proposed configurable health-monitor restart mode: the runtime still does unconditional stop/reset/start restarts, the public config/schema/docs only expose the existing interval/stale-threshold/restart-cap knobs, and there is no shipped `channelHealthRestartMode`/per-channel `restartMode` surface to point to.
