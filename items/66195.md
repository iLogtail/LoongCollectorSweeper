## Summary

Keep open. Current main and the latest shipped release expose `stringEnum` on scoped plugin SDK subpaths, but the deprecated root/compat surfaces implicated by the report still do not show that export. That means the exact `import { stringEnum } from "openclaw/plugin-sdk"` compatibility regression is not shown fixed in OpenClaw core with high confidence.
