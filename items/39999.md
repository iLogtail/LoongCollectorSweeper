## Summary

Current `main` has newer global-update verification around packaged installs, but it still does not implement this PR's automatic rollback-to-previous-version behavior after a post-update doctor failure. The PR also has an unresolved review finding on its rollback guard, so there is not strong evidence to auto-close it.
