## Summary

Keep open. Current `main` still leaves supervisor orphan reconciliation as a deliberate no-op, process/session visibility is still backed by in-memory registries, and the generic background exec PTY path still lacks the CLI runner's scope-replacement and no-output-timeout safeguards. The latest shipped release contains the same code, so this is not already fixed.
