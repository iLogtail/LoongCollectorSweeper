## Summary

Keep PR #65753 open. Current main still uses a simple boolean toggle for lines starting with triple backticks in `scripts/docs-link-audit.mjs`, so it does not support tilde fences or delimiter-length matching. The existing test file also lacks the proposed fence delimiter coverage. The PR is a focused scripts fix with recent author follow-up and no protected-label or maintainer-author reason to auto-close.

## Best Possible Solution

Keep the PR open and either merge a refreshed version or apply the same fix directly: make `scripts/docs-link-audit.mjs` parse fenced code delimiters by marker type and length, support both backticks and tildes, and add regression coverage for skipped links inside those fences.
