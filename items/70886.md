## Summary

Keep open. Current `main` still wires Z.AI image understanding straight through the shared provider config, while Z.AI onboarding can still set a Coding Plan base URL for the whole provider. The bundled Z.AI image model remains `glm-4.6v`, so the reported coding-endpoint/vision mismatch is still plausible on `main`. There is an open follow-up PR (#70915) with a targeted routing fix, but that fix is not present in this checkout.
