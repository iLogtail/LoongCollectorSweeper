## Summary

Keep #70886 open. Current main still does not implement a ZAI vision-model endpoint override for Coding Plan base URLs, and the exact fix attempt is still an open PR (#70915), not shipped on main.

## Best Possible Solution

Keep #70886 open as the user-facing bug until a fix lands. Review or adapt #70915 against current main, preferably in the ZAI provider/media-understanding path, and add regression coverage for `zai-coding-global`/`zai-coding-cn` image routing so `zai/glm-4.6v` uses the standard `.../paas/v4` endpoint while text/coding models keep the Coding Plan endpoint. Track the MiniMax comment separately unless the same tests prove the shared payload bug is fixed.
