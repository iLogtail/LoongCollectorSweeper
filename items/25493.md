## Summary

Keep #25493 open. Current main still does not provide editable system-prompt templates or section-level workspace overrides such as SAFETY.md; the requested Safety/system tips/CLI prompt customization remains missing, and related PR #25938 was closed unmerged.

## Best Possible Solution

Keep #25493 open as a core prompt-builder/product decision. A complete implementation should add a documented and tested section/template override path for selected OpenClaw-owned prompt sections, likely in src/agents/system-prompt.ts plus workspace/config loading, with explicit safety and prompt-cache semantics. If maintainers decide Safety or other core sections must not be section-editable, document that boundary and keep agents.defaults.systemPromptOverride as the supported full-replacement escape hatch.
