## Summary

Keep open. Current `main` does not implement schema-level tool security classification properties like `isDestructive`, `isReadOnly`, `isConcurrencySafe`, `interruptBehavior`, `isOpenWorld`, or `toAutoClassifierInput`; instead it still relies on ad-hoc tool metadata plus hard-coded approval/mutation heuristics. The request also fits OpenClaw core scope rather than a ClawHub/plugin-only path because it changes core tool contracts and safety behavior.
