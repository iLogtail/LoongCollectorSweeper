## Summary

Keep open. Current `main` does not show this fix as already implemented: xhigh support is now provider-owned, but `vercel-ai-gateway` still exposes upstream OpenAI refs without any thinking-profile/xhigh hook, so `sessions.patch` would still reject `thinkingLevel: "xhigh"` for `vercel-ai-gateway/openai/gpt-5.4`.
