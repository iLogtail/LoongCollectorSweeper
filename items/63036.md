## Summary

Keep open. Current `main` still supports `evaluate`; it was only gated behind `browser.evaluateEnabled`, and that flag defaults to enabled. The PR's stated goal to remove `evaluate` from the browser agent route, tool schema, CLI, and docs is not implemented on `main`.
