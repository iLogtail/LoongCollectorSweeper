## Summary

Keep #7424 open. Current main still has a global `agents.defaults.models` allowlist/catalog and per-agent primary/fallback defaults, but no ordinary `agents.list[]` model allowlist schema or agent-scoped `/models` filtering/enforcement. The provided context shows #9010 was closed as a duplicate of #7424, so this remains the canonical tracker.

## Best Possible Solution

Keep #7424 open as the canonical feature request. Add an agent-scoped model allowlist, likely under `agents.list[].model.allowlist` or `agents.list[].models`, and enforce it consistently across schema/types/docs/generated metadata, model-list resolution with agent context, session model changes, OpenAI-compatible overrides, Control UI model pickers, and tests.
