## Summary

Keep open. Current main has moved OpenClaw’s OpenAI paths toward OpenClaw-owned boundary transports and direct openai@6.34.0, but the relevant OpenAI SDK SSE parser still unconditionally parses empty sse.data, and @mariozechner/pi-ai still brings the reported openai@6.26.0 version transitively. I did not find a repo-side guard or regression test for event-only SSE frames.
