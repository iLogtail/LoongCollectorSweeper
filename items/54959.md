## Summary

Current `main` has a new general OpenShell backend doc, but it does not implement this PR’s sandbox deployment guide well enough to close the PR. The shipped page covers backend setup, workspace modes, config, and lifecycle; it does not cover the PR’s proxy-only/OpenShell-in-sandbox material such as launcher pattern, TLS/`SSL_CERT_FILE`, Slack Socket Mode policy, CoreDNS fixes, or troubleshooting. The PR also still needs maintainer review because its latest discussed fetch-guard workaround touches current SSRF guard behavior.
