## Summary

Keep open. Current `main` does not implement the requested left-side conversation management feature set. The chat UI uses a header session `<select>` grouped by agent/other sessions, not a sortable or folder/tag-based conversation list, and the only built-in rename path I found is the label field in the separate Sessions table.
