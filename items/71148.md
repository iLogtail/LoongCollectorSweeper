## Summary

Keep open. Current `main` still ships the optional Lobster plugin as an in-process wrapper around the external `@clawdbot/lobster/core` runtime, and the issue’s reported leak targets that embedded runtime path. I found no evidence in this checkout that the Ajv compile behavior has already been fixed or made obsolete on `main`, so there is not a high-confidence close reason.
