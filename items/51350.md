## Summary

Keep open. Current main has the Slack media pipeline and partial regression coverage, but this issue is an audit/documentation task for specific Slack handoff cases. I did not find a main-branch audit artifact or Slack docs that document direct image, thread-root image inheritance, multi-image, and forwarded/shared attachment behavior end to end; the related PR #51400 was provided as closed and unmerged, so it does not supersede the issue by itself.
