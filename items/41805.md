## Summary

Keep open. Current `main` still routes Discord component clicks back to the agent as plain inbound text rather than exposing structured interaction metadata. The Discord plugin does retain button label and source message id internally, and plugin interactive handlers receive limited structured fields, but the agent-facing inbound context only carries generic sender info plus the interaction id, not the source message/button identifiers requested in #41805.
