## Summary

Keep open. Current main has relevant memory-search mitigations, but I do not see high-confidence proof that the reported CJK keyword inconsistency is fully fixed for the default path. The CJK substring fallback is gated on the trigram FTS tokenizer, while the default remains unicode61, and the closest hybrid keyword-only integration tests are currently skipped.
