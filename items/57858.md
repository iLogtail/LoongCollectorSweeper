## Summary

Keep open. Current `main` still ships bundled `coding-agent` guidance for Claude Code without clearing the nesting-related env vars named in the report, and the Claude CLI env clearlist on `main` still omits `CLAUDECODE` and `CLAUDE_CODE_SESSION`.
