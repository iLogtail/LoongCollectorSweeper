## Summary

Current `main` already avoids this failure mode. Skill snapshots keep canonical absolute `resolvedSkills` paths for runtime/session consumers, while only the prompt-facing catalog compacts paths to `~/...`; the read-path helpers also expand `~` against the OS home directory, including Windows home resolution.
