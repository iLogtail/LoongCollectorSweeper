## Close Comment

Closing after Codex review. This PR is a docs-only discoverability addition for an external community plugin, but current `main` now treats ClawHub as the canonical discovery path for community plugins and explicitly says not to open docs-only PRs just to add a plugin listing (`docs/plugins/community.md:14-16`, `docs/plugins/community.md:147-153`). That matches the repo vision to keep optional capability in external plugins rather than growing core-maintained surfaces (`VISION.md:54-61`). Please publish or maintain the MAX plugin on ClawHub instead; if there is a source-doc gap beyond discoverability, open a focused docs PR for that specific documentation problem.
