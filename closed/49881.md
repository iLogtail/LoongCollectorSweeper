## Close Comment

Closing after Codex review. On current `main` (`4514691`), the two helpers are already available on supported public SDK subpaths: `promptAccountId` is exported from `src/plugin-sdk/setup.ts:65`, and `resolveMentionGatingWithBypass` is exported from `src/plugin-sdk/channel-mention-gating.ts:20` (also via `src/plugin-sdk/channel-inbound.ts:47`). The docs now explicitly treat `plugin-sdk/feishu` as a bundled-maintenance seam rather than a recommended third-party path (`docs/plugins/sdk-overview.md:45-48`, `docs/plugins/sdk-channel-plugins.md:207-209`). Because of that, the PR's core claim that these symbols are not reachable from any public surface no longer matches current `main`. If the external Feishu plugin still needs an update, the supported path is to migrate its imports to the generic SDK seams rather than expanding `openclaw/plugin-sdk/feishu`.
