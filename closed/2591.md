## Summary

本 PR 为构建脚本、打包脚本及 Dockerfile 添加了显式的 riscv64 架构支持，修正了非 x86_64 主机被错误归类为 arm64 或 sw64 的问题。变更范围清晰保守，仅修正架构探测与库路径映射，未触及核心数据通路，且作者已提供详细的本地验证步骤。属于有价值的可移植性增强，需维护者审查后合并。

## Best Possible Solution

保持开启。建议维护者审查架构检测逻辑及 Dockerfile 映射是否符合项目发版规范，确认现有 CI 流水线在 amd64/arm64 上未受破坏后合并。若后续需将 riscv64 加入正式 GitHub Actions 发版矩阵或扩展 CentOS 生产镜像路径，可作为独立特性另行跟进。
