## Summary

Current `main` and the latest release do not contain this PR's MCP Apps gateway surface. The repo still ships only the MCP loopback `tools/list`/`tools/call` path, with no `mcp.*` WebSocket methods, no resource registry/read surface, and no corresponding gateway protocol schemas/docs. The PR is therefore not implemented on `main`, and the review discussion still includes design choices that need maintainer judgment rather than auto-close cleanup.
